# 余额查询

AiDog 支持查询部分平台的账户余额和 Coding Plan 配额。

## 支持自动查询的平台

以下平台支持余额自动查询：

| 平台 | 查询内容 |
|------|---------|
| OpenAI | 账户余额 |
| DeepSeek | 账户余额 |
| StepFun (阶跃星辰) | 账户余额 |
| SiliconFlow | 账户余额 |
| OpenRouter | 账户余额 / 使用量 |
| Novita | 账户余额 |
| Kimi (月之暗面) | 账户余额 |
| GLM (智谱) | 账户余额 |
| MiniMax | 账户余额 |

## 查看余额

余额信息显示在：

1. **平台卡片** — 每个平台的实时余额
2. **分组卡片** — 组内平台的余额汇总
3. **系统托盘** — 显示余额状态

## 刷新余额

余额数据定时自动刷新。也可手动点击刷新按钮。

## Coding Plan

部分平台提供 Coding Plan 配额信息（如 OpenAI 的 Plus/Team/Enterprise 计划），AiDog 会一并查询和展示。

## 手动预算

对于不支持自动余额查询的平台，AiDog 提供手动预算功能：

### 配置方式

在平台配置中设置手动预算，支持以下类型：

| 类型 | 说明 |
|------|------|
| 总额（Total） | 设置一个总额度上限 |
| 滚动窗口（Rolling） | 在指定时间窗口内的消耗上限 |
| 固定周期（Fixed） | 按固定周期重置的消耗上限 |
| 每日（Daily） | 每日消耗上限 |

### 预算单位

- **美元（USD）** — 按费用计量
- **Token** — 按 Token 数量计量

### 窗口时长

滚动窗口和固定周期支持多种时间单位：分钟、小时、天、周、月。

预算消耗由 AiDog 根据代理请求自动累计，达到上限后平台会被标记为余额不足。

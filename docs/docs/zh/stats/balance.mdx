# 余额与配额

AiDog 帮你追踪各平台的账户余额和编程计划配额使用情况，让你对花费一目了然。

## 自动余额查询

AiDog 会自动查询以下平台的账户余额：

| 平台 | 查询内容 |
|------|---------|
| OpenAI | 账户余额 |
| DeepSeek | 账户余额 |
| 阶跃星辰 (StepFun) | 账户余额 |
| SiliconFlow | 账户余额 |
| OpenRouter | 账户余额 / 使用量 |
| Novita | 账户余额 |
| Kimi（月之暗面） | 账户余额 |
| GLM（智谱） | 账户余额 |
| MiniMax | 账户余额 |

余额数据定时自动刷新，也可手动点击刷新按钮。

## 编程计划配额

部分平台提供编程计划（Coding Plan），如 OpenAI 的 Plus / Team / Enterprise 计划、智谱的 Coding Plan、Kimi 的 Code Plan 等。

AiDog 会查询并展示编程计划的配额信息：

- 各层级配额名称
- 当前使用率
- 剩余比例

在选择平台类型时，带有"编程计划"标识的预设会自动配置对应的配额查询。

## NewAPI 余额查询

对于使用 NewAPI 系统的平台（如自建的 API 聚合服务），AiDog 支持独立的余额查询配置：

- **余额查询地址** — 独立于 API 地址的查询地址
- **余额查询 Token** — 从控制台获取的查询令牌（与 API Key 不同）
- **用户 ID** — 可选，用于查询指定用户的余额

在平台配置的「余额查询配置」部分填写这些信息即可启用。

## 手动预算

对于不支持自动余额查询的平台，AiDog 提供手动预算功能。

### 设置方式

在平台配置的「手动预算」部分，点击「添加限额」：

| 类型 | 说明 |
|------|------|
| 总额 | 设置一个总花费上限 |
| 滑动窗口 | 在滚动时间窗口内的花费上限 |
| 固定窗口 | 按固定周期重置的花费上限 |
| 每日 | 每日花费上限 |

### 预算单位

- **美元** — 按费用计量
- **Token** — 按 Token 数量计量

### 窗口时长

滑动窗口和固定窗口支持多种时间粒度：分钟、小时、天、周、月。

### 预算耗尽时

当任一预算限额耗尽时，AiDog 会停止向该平台转发请求（返回 402 错误），并自动触发故障转移到组内其他平台。窗口重置后自动恢复正常转发。

## 查看余额信息

余额信息在多个位置展示：

- **平台卡片** — 每个平台的实时余额和配额使用率
- **分组卡片** — 组内平台的余额汇总
- **系统托盘** — 可配置显示余额状态

## 统计页面

在 **「统计」** 页面，可以按平台、分组、模型维度查看：

- 今日请求数、成功率
- Token 用量（输入 / 输出 / 缓存）
- 预估费用
- 用量趋势图（按日 / 周 / 月）

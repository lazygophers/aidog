# 余额查询

AiDog 支持查询部分平台的账户余额和 Coding Plan 配额。

## 支持的平台

以下平台支持余额自动查询：

| 平台 | 查询内容 |
|------|---------|
| OpenAI | 账户余额 |
| DeepSeek | 账户余额 |
| StepFun (阶跃星辰) | 账户余额 |
| SiliconFlow | 账户余额 |
| OpenRouter | 账户余额 / 使用量 |
| Novita | 账户余额 |
| Kimi (月之暗面) | 账户余额 |
| GLM (智谱) | 账户余额 |
| MiniMax | 账户余额 |

## 查看余额

余额信息显示在：

1. **平台卡片** — 每个平台的实时余额
2. **分组卡片** — 组内平台的余额汇总
3. **系统托盘** — 显示余额状态

## 刷新余额

余额数据定时自动刷新。也可手动点击刷新按钮。

## Coding Plan

部分平台提供 Coding Plan 配额信息，AiDog 会一并查询和展示。

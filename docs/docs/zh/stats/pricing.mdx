# 定价与计费

AiDog 基于模型定价表自动估算每次请求的费用。在 **「设置」→「模型价格」** 标签页中管理所有定价数据。

## 费用估算

每次请求完成后，AiDog 会自动：

1. 从上游响应中获取 Token 用量（输入 / 输出 / 缓存）
2. 查询模型定价表获取单价
3. 计算预估费用

定价回退顺序：自定义定价 → 同步定价 → 兜底价格

:::tip{title="提示"}
预估费用仅供参考，实际费用以平台账单为准。
:::

## 定价管理页面

### 价格同步

AiDog 支持从定价源一键同步最新模型定价数据：

- 点击「立即同步」按钮获取最新价格
- 同步结果显示：新增数量、更新数量、失败数量
- 开启「自动同步」后按设定间隔自动拉取

自动同步支持以下间隔：1 小时 / 6 小时 / 12 小时 / 24 小时 / 7 天

### 兜底价格

对于定价表中未收录的模型，使用兜底价格计算费用：

- **输入兜底价** — 默认 $3.0 / 百万 Token
- **输出兜底价** — 默认 $3.0 / 百万 Token

可在「模型价格」页面的同步设置区域调整。

### 浏览与搜索

价格表显示以下列：

| 列 | 说明 |
|----|------|
| 模型 | 模型名称 |
| 来源 | LiteLLM（同步） / 手动 |
| 默认平台 | 该模型对应的默认平台 |
| 输入 $/M | 输入 Token 价格（每百万 Token） |
| 输出 $/M | 输出 Token 价格 |
| 缓存读取 $/M | 缓存 Token 读取价格 |

- 按模型名称搜索
- 按来源筛选（LiteLLM / 手动）
- 分页浏览，支持调整每页数量和跳转到指定页码

### 管理定价

- **删除** — 移除某条定价记录（点击行尾删除按钮）
- 定价数据修改后立即生效，后续请求使用新价格计算

## 查看费用

在 **「统计」** 页面中查看费用相关数据：

- 概览卡片的「预估成本」指标（含环比变化）
- 维度排行表的「预估成本」列
- 趋势图的成本详情（悬停 Tooltip）

# 定价与计费

AiDog 基于模型定价表自动估算每次请求的费用。

## 费用估算

每次请求完成后，AiDog 会：
1. 从上游响应中获取 Token 用量
2. 查询模型定价表获取单价
3. 计算 `费用 = Prompt Token × 输入价 + Completion Token × 输出价`

## 定价来源

AiDog 内置模型定价数据，并支持以下来源：
- **内置定价** — 覆盖主流模型的官方定价
- **自定义定价** — 手动设置特定模型的单价
- **定价同步** — 自动从平台获取最新定价

## 预估费用字段

每条日志的 `est_cost` 字段记录预估费用：
- 使用 `resolve_price` 回退链确保总有定价
- 回退顺序：自定义定价 → 内置定价 → 默认价
- 前端使用 `SUM(est_cost)` 聚合每日/月费用

:::tip{title="提示"}
预估费用仅供参考，实际费用以平台账单为准。
:::

## 查看费用

在 **「统计」** 页面的「定价」标签页中：
- 查看各模型的定价
- 编辑自定义定价
- 触发定价同步

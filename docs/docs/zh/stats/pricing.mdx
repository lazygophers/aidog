# 定价与计费

AiDog 基于模型定价表自动估算每次请求的费用。

## 费用估算

每次请求完成后，AiDog 会：
1. 从上游响应中获取 Token 用量
2. 查询模型定价表获取单价
3. 计算 `费用 = Prompt Token × 输入价 + Completion Token × 输出价`

:::tip{title="提示"}
预估费用仅供参考，实际费用以平台账单为准。
:::

## 定价来源

AiDog 支持多种定价数据来源：

- **内置定价** — 覆盖主流模型的官方定价
- **自定义定价** — 手动设置特定模型的单价，优先于内置定价
- **自动同步** — 从定价源自动拉取最新模型定价数据

定价回退顺序：自定义定价 → 内置定价 → 默认价

## 定价管理页面

在 **「设置」→「模型价格」** 标签页中管理所有模型定价：

### 浏览与搜索

- 按模型名称搜索
- 按定价来源筛选（内置 / 自定义 / 同步）
- 分页浏览所有定价记录

### 编辑自定义定价

- 为任意模型添加自定义单价
- 分别设置输入 Token 价格和输出 Token 价格
- 编辑或删除已有自定义定价

### 定价同步

- 配置定价同步源
- 一键同步最新定价数据
- 查看同步结果

## 查看费用

在 **「统计」** 页面中查看：
- 今日/本周/本月的预估费用
- 按平台、分组、模型的费用分布
- 费用趋势图

# 使用统计

**「统计」** 页面提供所有代理请求的用量概览、趋势分析和维度排行，帮助你全面了解 AI API 的使用情况。

## 时间范围

页面顶部提供三种时间预设：

| 预设 | 范围 |
|------|------|
| 今天 | 当日零点至今 |
| 近 7 天 | 过去 7 天 |
| 近 30 天 | 过去 30 天 |

选择后，所有指标和图表自动刷新。

## 概览卡片

页面顶部显示 8 个核心指标，每个指标旁显示与上一等长周期的环比变化：

| 指标 | 说明 |
|------|------|
| 总请求 | 当前周期内的请求总数 |
| 成功率 | 成功请求占比（按颜色编码） |
| 输入 Token | 所有请求的输入 Token 总量 |
| 输出 Token | 所有请求的输出 Token 总量 |
| 缓存 Token | 缓存命中的 Token 总量 |
| 缓存率 | 缓存 Token 占输入 Token 的比例 |
| 平均延迟 | 平均请求响应时间（毫秒） |
| 预估成本 | 基于定价表计算的预估费用 |

环比箭头：绿色向上表示正向变化（如成功率提升），红色向上表示负面变化（如成本上升）。

## 筛选器

在时间预设旁，可以通过以下条件精确筛选：

- **粒度**：按天 / 按小时
- **维度**：按平台 / 按模型 / 按分组
- **分组**：按分组名称筛选（支持搜索）
- **模型**：按模型名称筛选（支持搜索）
- **平台类型**：按平台类型筛选（支持搜索）

筛选条件变化时自动重新查询。

## 趋势图

概览卡片下方显示请求量趋势柱状图：

- 横轴按选定粒度（天/小时）分桶
- 柱体高度表示请求数量
- 颜色渐变反映错误比例（红色越多表示失败越多）
- 鼠标悬停显示详细 Tooltip：请求数、成功数、失败数、平均延迟、预估成本

## 维度排行表

趋势图下方显示按当前维度（平台/模型/分组）的排行表：

| 列 | 说明 |
|----|------|
| 名称 | 平台 / 模型 / 分组名称 |
| 请求 | 该维度的请求总数 |
| 成功 | 成功请求数（颜色编码表示健康度） |
| 输入 | 输入 Token 数 |
| 输出 | 输出 Token 数 |
| 缓存 | 缓存 Token 数 |
| 平均延迟 | 平均响应时间 |
| 预估成本 | 预估费用（颜色编码表示花费高低） |

### 排序与分页

- 点击任意列头按该列排序（再次点击切换升序/降序）
- 每页 50 条，底部提供翻页导航

## 实时刷新

统计数据自动随新请求刷新，无需手动操作。

:::tip{title="提示"}
预估费用基于模型定价表计算，仅供参考。实际费用以平台账单为准。
:::

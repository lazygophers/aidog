# 使用统计

AiDog 自动统计所有代理请求的用量数据，帮助你了解 AI API 的使用情况。

## 统计页面

进入 **「统计」** 页面，查看用量概览。

### 今日统计

- 总请求数
- 成功/失败率
- Token 总用量（Prompt + Completion）
- 预估费用

### 趋势图

按日/周/月查看：
- 请求数趋势
- Token 用量趋势
- 费用趋势

### 分组统计

Group 卡片的统计数据由关联平台聚合而来（前端求和）。

## 统计维度

可按以下维度查看统计：
- **按平台** — 每个平台的用量
- **按分组** — 每个分组的用量
- **按模型** — 每个模型的用量
- **按时间** — 按天/周/月汇总

## 数据来源

统计数据来自 `proxy_logs` 表的聚合查询：
- Token 用量：直接从上游 API 响应提取
- 预估费用：通过定价表 + 模型名计算（`resolve_price` 回退链）
- 请求计数：日志行数聚合

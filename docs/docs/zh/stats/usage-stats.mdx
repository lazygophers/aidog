# 使用统计

AiDog 自动统计所有代理请求的用量数据，帮助你了解 AI API 的使用情况。

## 统计页面

进入 **「统计」** 页面，查看用量概览。

### 今日统计

- 总请求数
- 成功/失败率
- Token 总用量（Prompt + Completion）
- 预估费用

### 趋势图

按日/周/月查看：
- 请求数趋势
- Token 用量趋势
- 费用趋势

### 分组统计

分组卡片的统计数据由其关联平台聚合而来。

## 统计维度

可按以下维度查看统计：
- **按平台** — 每个平台的用量
- **按分组** — 每个分组的用量
- **按模型** — 每个模型的用量
- **按时间** — 按天/周/月汇总

## 数据来源

- Token 用量：直接从上游 API 响应中提取
- 预估费用：根据模型定价表自动计算
- 请求计数：代理日志记录聚合

:::tip{title="提示"}
预估费用基于内置定价表计算，仅供参考。实际费用以平台账单为准。
:::

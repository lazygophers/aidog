# 模型价格

在 **「设置」→「模型价格」** 标签页中管理所有模型的定价数据，用于计算请求的预估费用。

## 价格同步

从定价源一键同步最新模型价格表：

- 点击「立即同步」按钮拉取最新价格
- 同步结果：新增数量、更新数量、失败数量
- 开启「自动同步」后按设定间隔自动拉取

自动同步间隔：1 小时 / 6 小时 / 12 小时 / 24 小时 / 7 天

## 兜底价格

对于定价表中未收录的模型，使用兜底价格计算：

- **输入兜底价** — 默认 $3.0 / 百万 Token
- **输出兜底价** — 默认 $3.0 / 百万 Token

可在同步设置区域调整。

## 价格表

价格表显示以下列：

| 列 | 说明 |
|----|------|
| 模型 | 模型名称 |
| 来源 | LiteLLM（同步） / 手动 |
| 默认平台 | 该模型对应的默认平台 |
| 输入 $/M | 输入 Token 价格（每百万 Token） |
| 输出 $/M | 输出 Token 价格 |
| 缓存读取 $/M | 缓存 Token 读取价格 |

支持按模型名称搜索、按来源筛选，可调整每页数量并跳转到指定页码。

## 管理定价

- 删除某条定价记录（行尾删除按钮）
- 定价数据修改后立即生效

## 费用估算原理

每次请求完成后，AiDog 自动：

1. 从上游响应获取 Token 用量（输入 / 输出 / 缓存）
2. 查询定价表获取单价
3. 计算预估费用

定价回退顺序：自定义定价 → 同步定价 → 兜底价格

:::tip{title="提示"}
预估费用仅供参考，实际费用以平台账单为准。
:::

详见 [定价与计费](/stats/pricing)。

# 应用设置

在 **「设置」** 页面配置 AiDog 的全局行为。

## 代理设置

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 代理端口 | `7860` | HTTP 代理监听端口 |
| 请求超时 | `300s` | 上游请求超时时间 |

## 日志设置

详见 [日志设置](/logs/log-settings)。

## 数据管理

### 导出数据

- 导出平台配置（JSON）
- 导出分组配置（JSON）
- 导出统计数据（CSV）

### 清理数据

- 清理过期日志
- 重置统计数据

## 界面设置

### 语言

支持 7 种界面语言：
- 简体中文 (zh-CN)
- English (en-US)
- العربية (ar-SA)
- Français (fr-FR)
- Deutsch (de-DE)
- Русский (ru-RU)
- 日本語 (ja-JP)

### 主题

详见 [主题定制](/settings/themes)。

## 数据存储

所有数据存储在本地：

| 数据 | 路径 |
|------|------|
| 数据库 | `~/.aidog/aidog.db` |
| 配置文件 | `~/.aidog/config.json` |

数据不离开本机，不上传任何服务器。

# 主题定制

AiDog 提供多种内置主题，每套主题都有 Light 和 Dark 两种模式。

## 内置主题

| 主题 | 风格 |
|------|------|
| Liquid Glass | 玻璃拟态，通透感 |
| Nord | 北极蓝调，冷静沉稳 |
| Dracula | 暗色系，流行配色 |
| Catppuccin | 温柔配色，护眼舒适 |
| Solarized | 经典科学配色 |

## 切换主题

在 **「设置」** 页面的主题选择器中切换。

### 亮色/暗色模式

每个主题都支持 Light/Dark 双模式，可跟随系统设置自动切换或手动选择。

## 主题特性

- **圆角体系** — 分层圆角设计（sm / md / lg / xl）
- **毛玻璃效果** — 半透明面板与模糊背景
- **平滑过渡** — 主题切换带缓动动画
- **高对比度** — 确保在亮色和暗色模式下都有良好的可读性

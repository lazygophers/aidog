# 主题定制

AiDog 提供多种内置主题，每套主题都有 Light 和 Dark 两种模式。

## 内置主题

| 主题 | 风格 |
|------|------|
| Liquid Glass | 玻璃拟态，通透感 |
| Nord | 北极蓝调，冷静沉稳 |
| Dracula | 暗色系，流行配色 |
| Catppuccin | 温柔配色，护眼舒适 |
| Solarized | 经典科学配色 |

## 切换主题

在 **「设置」** → 「主题」 中选择。

### 亮色/暗色模式

每个主题都支持 Light/Dark 双模式，跟随系统设置或手动切换。

## 主题架构

主题基于 CSS 变量系统实现：
- 每主题提供 light + dark 两组 CSS 变量
- 变量文件位于 `src/themes/` 目录
- 切换主题时替换 CSS 变量集

### 自定义主题

如果你想创建自定义主题：

1. 复制现有主题文件作为模板
2. 修改 CSS 变量值
3. 在 `src/themes/index.ts` 中注册新主题
4. 重新构建应用

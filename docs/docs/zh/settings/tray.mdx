# 托盘配置

AiDog 支持系统托盘常驻，在菜单栏/任务栏实时显示关键状态信息。在 **「设置」→「系统托盘」** 中配置显示内容和样式。

## 托盘行为

关闭主窗口后 AiDog 会继续在系统托盘运行，保持代理服务可用。右键点击托盘图标可显示/隐藏主窗口、启动/停止代理、退出应用。

## 实时预览

配置页面顶部提供模拟菜单栏的实时预览，所有改动即时反映到预览中。预览还显示当前行数占用（最多 2 行），超出会标黄提示。

## 展示项类型

点击「添加展示项」可添加以下类型的内容：

### 平台余额 / 配额

选择一个已启用的平台，显示其余额或编程计划配额：

| 显示模式 | 说明 |
|----------|------|
| 余额 | 显示账户余额（如 `$12.50`） |
| 配额 | 显示编程计划剩余比例（如 `85%`） |

### 今日统计

显示当日的汇总数据，支持 4 种指标：

| 指标 | 说明 |
|------|------|
| Tokens | 今日 Token 总消耗 |
| Cache% | 今日缓存命中率 |
| 花费$ | 今日预估费用 |
| 请求 | 今日请求总数 |

### 分隔符

在展示项之间插入分隔符，预设 6 种样式（`|`、`·`、`—`、`/`、`»`、空格），也支持自定义字符。

## 展示项自定义

展开任意展示项后，可调整以下属性：

| 属性 | 说明 |
|------|------|
| 展示内容 | 余额 / 配额（仅平台项） |
| 指标 | Tokens / Cache% / 花费$ / 请求（仅今日统计项） |
| 标签 | 自定义显示名称，留空使用默认 |
| 小数位 | 数值的小数位数 |
| 行模式 | 单行（标签和数值同行） / 两行（标签上行、数值下行） |
| 对齐 | 左对齐 / 居中 / 右对齐 |
| 值行对齐 | 两行模式下，第二行（数值）的独立对齐方式 |
| 颜色 | 跟随系统 / 预设色（红/绿/橙）/ 自定义色 |

### 颜色警告

选择自定义颜色时，如果颜色过暗或过亮，AiDog 会提示该颜色在部分菜单栏主题下可能不清晰。

## 拖拽排序

展示项支持拖拽排序，调整在托盘中的显示顺序。禁用的展示项不会显示在预览中。

## 开机自启动

在 **「设置」→「代理配置」** 中可配置 AiDog 随系统启动自动运行。

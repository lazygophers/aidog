# 托盘配置

AiDog 支持系统托盘常驻，在后台显示关键状态信息。

## 系统托盘

关闭主窗口后 AiDog 会继续在系统托盘运行，保持代理服务可用。

### 托盘信息

托盘图标可显示：
- 代理服务状态（运行/停止）
- 账户余额
- 今日用量概览

### 托盘菜单

右键点击托盘图标可：
- 显示/隐藏主窗口
- 启动/停止代理
- 退出应用

## 托盘自定义

在 **「设置」** → 「托盘」 中配置：

### 显示内容

选择要在托盘中显示的信息项：
- 余额信息
- 今日请求数
- 今日 Token 数
- 今日预估费用

### 状态栏配置

配置状态栏显示格式，支持自定义分隔符和显示顺序。

## 开机自启动

可配置 AiDog 随系统启动自动运行（最小化到托盘）。

# 创建与管理

分组是 AiDog 的路由核心 — 将多个平台按用途组织，配合规则控制请求路由。

## 创建分组

1. 进入 **「分组」** 页面
2. 点击 **「+ 添加分组」**
3. 配置：

| 字段 | 说明 |
|------|------|
| 名称 | 分组显示名称 |
| 描述 | 可选，分组说明 |
| 包含平台 | 选择要加入该分组的平台 |
| 路由策略 | 平台选择顺序 |

### 路由策略

- **顺序** — 按平台排列顺序依次尝试
- **随机** — 在可用平台中随机选择
- **加权** — 按权重分配请求比例

## 管理平台

分组内的平台支持：
- **拖拽排序** — 调整优先级（顺序策略下）
- **启用/禁用** — 临时跳过某平台
- **设置权重** — 加权策略下的请求比例
- **模型映射** — 分组级别的模型映射覆盖

## 编辑分组

点击分组卡片进入编辑模式，修改后点击「保存」。

## 删除分组

删除分组不会删除其中的平台，只是解除关联。

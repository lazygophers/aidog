# 支持的平台与接口

AiDog 支持丰富的 AI 平台和接口类型，并自动处理不同平台间的协议转换。

## 支持的平台

AiDog 内置 50+ 平台预设，涵盖主流 AI 服务：

| 平台类型 | 示例 |
|----------|------|
| 国际平台 | OpenAI、Anthropic、Google Gemini、Mistral |
| 国内平台 | DeepSeek、通义千问、智谱 GLM、Kimi、MiniMax |
| 聚合平台 | OpenRouter、SiliconFlow、Novita |

选择平台预设后，所有配置自动填充，无需手动选择接口类型。

## 接口自动识别

AiDog 会根据客户端请求自动识别接口类型，无需手动配置：

- 对话接口 — 最常用的 AI 对话请求
- 编程助手接口 — Codex、Claude Code 等编程工具使用的接口
- 文本补全接口 — 旧版文本补全请求

这意味着你可以用同一个代理地址同时服务不同类型的客户端。

## 协议自动转换

如果你的客户端使用一种格式发请求，但目标平台使用另一种格式，AiDog 会自动转换。例如：

- 客户端用 OpenAI 格式发请求 → 自动转换为 Anthropic 格式发到 Claude
- 编程助手用专用接口 → 自动转换为对应平台的格式

## 模拟测试平台

AiDog 提供模拟平台，不发送真实请求，用于开发测试。可自定义响应内容、状态码、延迟等参数。

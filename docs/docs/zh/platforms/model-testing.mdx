# 模型测试

AiDog 提供内置模型测试功能，无需借助外部工具即可验证平台连通性和模型可用性。

## 使用方式

在 **「平台」** 页面，点击平台卡片上的「测试」按钮，打开模型测试面板。

## 测试模式

| 模式 | 说明 |
|------|------|
| **快速测试** | 使用平台默认模型发送一条简短请求，快速验证连通性 |
| **单模型测试** | 选择一个指定模型进行测试 |
| **批量测试** | 选择多个模型依次测试，验证不同模型的可用性 |
| **随机测试** | 使用平台所有可用模型逐个测试 |
| **自定义测试** | 选择模型并输入自定义提示词和最大 Token 数 |

## 测试结果

每条测试结果展示：

- **模型名称** — 测试使用的模型
- **状态** — 成功或失败
- **响应时间** — 从发送请求到收到完整响应的耗时
- **Token 用量** — 输入和输出 Token 数
- **响应内容** — 模型返回的文本（可折叠查看）

## 测试场景

- **添加平台后** — 验证 API Key 和 Base URL 配置正确
- **切换端点后** — 确认新端点可以正常工作
- **模型映射后** — 确认映射的目标模型在平台上可用
- **排查故障** — 当代理请求失败时，先测试对应平台是否正常

## 注意事项

- 测试使用平台配置的 API Key 和端点，会消耗实际 Token
- Mock 平台的测试不会发送真实请求
- 批量和随机模式会依次测试多个模型，耗时较长

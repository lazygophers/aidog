# AiDog

AiDog 是一款跨平台桌面应用（macOS / Windows / Linux），作为 **AI API 网关代理**，帮你统一管理多个 AI 平台的 API Key、智能路由请求、追踪用量与费用。

## 核心功能

- **多平台聚合** — 统一管理 OpenAI、Anthropic、Gemini、DeepSeek、通义千问 等 50+ 平台的 API
- **智能路由** — 按分组、模型映射规则自动选择最优平台
- **用量统计** — 实时追踪 Token 用量、估算费用、查询余额
- **日志系统** — 三级记录粒度、可配置保留策略
- **Codex 集成** — 原生支持 OpenAI Codex / Claude Code 等编程助手
- **多协议支持** — OpenAI Chat/Completions/Responses、Anthropic、Gemini 原生协议
- **主题定制** — Liquid Glass、Nord、Dracula 等多种主题
- **系统托盘** — 常驻托盘显示余额与状态

## 三步上手

1. **添加平台** —— 填入某个 AI 平台的 API Key 与端点
2. **设置代理地址** —— 把你的客户端指向 AiDog 本地代理
3. **开始使用** —— 请求自动路由、计量、记录

→ [安装指南](/zh/getting-started/installation) · [快速开始](/zh/getting-started/quick-start)

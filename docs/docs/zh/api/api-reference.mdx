# API 接口

AiDog 提供本地 HTTP API 用于查询分组信息。所有端点仅限 **localhost** (`127.0.0.1`) 访问，无需外部网络。

## POST /api/group-info

查询分组的估算信息（余额、Coding Plan 使用率、请求统计等）。仅**单平台分组**返回可用数据；多平台或无平台分组返回 `applicable: false`。

### 鉴权

```
Authorization: Bearer <group_name>
```

`group_name` 是你在 AiDog 中配置的分组名称，与 Claude Code 使用的 `ANTHROPIC_AUTH_TOKEN` 环境变量值相同。

### 请求

- **方法**：POST
- **URL**：`http://127.0.0.1:<port>/api/group-info`
- **请求头**：`Authorization: Bearer <group_name>`

### 响应 (200 OK)

#### 单平台分组：

```json
{
  "applicable": true,
  "balance": 48.20,
  "spent": 1.23,
  "coding_plan": [
    {
      "name": "5h",
      "utilization": 23,
      "pace": "fast",
      "reset_at": null
    },
    {
      "name": "7d",
      "utilization": 41,
      "pace": "normal",
      "reset_at": null
    }
  ],
  "requests": 42,
  "success_rate": 95.2,
  "cache_rate": 13.3,
  "total_tokens": 123456,
  "currency": "",
  "balance_days_remaining": 5.2
}
```

#### 多平台 / 无平台分组：

```json
{
  "applicable": false,
  "balance": 0.0,
  "spent": 0.0,
  "coding_plan": [],
  "requests": 0,
  "success_rate": 0.0,
  "cache_rate": 0.0,
  "total_tokens": 0,
  "currency": "",
  "balance_days_remaining": null
}
```

### 响应字段

| 字段 | 类型 | 说明 |
|-------|------|-------------|
| `applicable` | boolean | 是否为有可用数据的单平台分组 |
| `balance` | number | 估算剩余余额 |
| `spent` | number | 累计估算消费 |
| `coding_plan` | array | Coding Plan 档位（Claude 等）；按余额计费的平台为空 |
| `coding_plan[].name` | string | 档位名（5h、7d 等） |
| `coding_plan[].utilization` | number | 使用率百分比 (0-100) |
| `coding_plan[].pace` | string | 消耗速率：`fast`、`normal` 或 `busy` |
| `coding_plan[].reset_at` | number\|null | 重置时间戳（Unix 秒） |
| `requests` | number | 总请求数 |
| `success_rate` | number | 成功率 (0-100) |
| `cache_rate` | number | 缓存命中率 (0-100) |
| `total_tokens` | number | 总消耗 token 数 |
| `currency` | string | 货币单位 |
| `balance_days_remaining` | number\|null | 余额预计剩余天数（无法计算时为 null） |

### 错误响应

| 状态码 | 条件 |
|--------|-----------|
| 401 | 缺失或无效的 `Authorization` 请求头 |
| 200 | 分组未找到 → `applicable: false` |

### 示例

```bash
curl -X POST \\
  -H "Authorization: Bearer my-group" \\
  http://127.0.0.1:9876/api/group-info
```

# API リファレンス

AiDog はグループ情報を照会するためのローカル HTTP API を提供します。すべてのエンドポイントは **localhost 専用**（`127.0.0.1`）で、外部ネットワークアクセスは不要です。

## POST /api/group-info

グループの推定情報（残高、Coding Plan 使用率、リクエスト統計など）を照会します。**単一プラットフォームグループ**のみ適用可能なデータを返します; マルチプラットフォームまたはプラットフォームなしのグループは `applicable: false` を返します。

### 認証

```
Authorization: Bearer <group_name>
```

`group_name` は AiDog で設定したグループ名です。Claude Code が使用する `ANTHROPIC_AUTH_TOKEN` 環境変数と同じ値です。

### リクエスト

- **メソッド**: POST
- **URL**: `http://127.0.0.1:<port>/api/group-info`
- **ヘッダー**: `Authorization: Bearer <group_name>`

### レスポンス (200 OK)

#### 単一プラットフォームグループ:

```json
{
  "applicable": true,
  "balance": 48.20,
  "spent": 1.23,
  "coding_plan": [
    {
      "name": "5h",
      "utilization": 23,
      "pace": "fast",
      "reset_at": null
    },
    {
      "name": "7d",
      "utilization": 41,
      "pace": "normal",
      "reset_at": null
    }
  ],
  "requests": 42,
  "success_rate": 95.2,
  "cache_rate": 13.3,
  "total_tokens": 123456,
  "currency": "",
  "balance_days_remaining": 5.2
}
```

#### マルチプラットフォーム / プラットフォームなしグループ:

```json
{
  "applicable": false,
  "balance": 0.0,
  "spent": 0.0,
  "coding_plan": [],
  "requests": 0,
  "success_rate": 0.0,
  "cache_rate": 0.0,
  "total_tokens": 0,
  "currency": "",
  "balance_days_remaining": null
}
```

### レスポンスフィールド

| フィールド | 型 | 説明 |
|-------|------|-------------|
| `applicable` | boolean | 利用可能なデータを持つ単一プラットフォームグループかどうか |
| `balance` | number | 推定残高 |
| `spent` | number | 累計推定消費 |
| `coding_plan` | array | Coding Plan ティア（Claude など）; 残高ベースのプラットフォームでは空 |
| `coding_plan[].name` | string | ティア名（5h、7d など） |
| `coding_plan[].utilization` | number | 使用率パーセンテージ (0-100) |
| `coding_plan[].pace` | string | 消費レート: `fast`、`normal`、`busy` |
| `coding_plan[].reset_at` | number\|null | リセットタイムスタンプ（Unix 秒） |
| `requests` | number | 総リクエスト数 |
| `success_rate` | number | 成功率 (0-100) |
| `cache_rate` | number | キャッシュヒット率 (0-100) |
| `total_tokens` | number | 総消費トークン数 |
| `currency` | string | 通貨単位 |
| `balance_days_remaining` | number\|null | 残高の推定残日数（算出不可の場合 null） |

### エラーレスポンス

| ステータス | 条件 |
|--------|-----------|
| 401 | `Authorization` ヘッダー欠落または無効 |
| 200 | グループ未検出 → `applicable: false` |

### 例

```bash
curl -X POST \\
  -H "Authorization: Bearer my-group" \\
  http://127.0.0.1:9876/api/group-info
```

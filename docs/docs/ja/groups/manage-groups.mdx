# 作成と管理

グループは AiDog のルーティングの中核です — 複数のプラットフォームを用途別に整理し、ルールでリクエストルーティングを制御します。

## グループの作成

1. **「グループ」** ページに移動
2. **「+ グループを追加」** をクリック
3. 設定：

| 項目 | 説明 |
|------|------|
| 名前 | グループの表示名 |
| 説明 | 任意のグループ説明 |
| プラットフォーム | グループに含めるプラットフォームを選択 |
| ルーティング戦略 | プラットフォームの選択順序 |

### ルーティング戦略

- **順次** — プラットフォームを並び順に試行
- **ランダム** — 利用可能なプラットフォームからランダムに選択
- **重み付き** — 重み比率でリクエストを分配

## プラットフォームの管理

グループ内のプラットフォームに対して以下が可能：
- **ドラッグで並び替え** — 優先順位を調整（順次戦略）
- **有効/無効** — プラットフォームを一時的にスキップ
- **重みの設定** — 重み付き戦略でのリクエスト比率
- **モデルマッピング** — グループレベルのモデルマッピングの上書き

## グループの編集

グループカードをクリックして編集モードに入ります。変更後「保存」をクリック。

## グループの削除

グループを削除しても、含まれるプラットフォームは削除されません — 関連付けのみが解除されます。

# モデル価格

すべてのモデル価格データを **「設定」→「モデル価格」** タブで管理します。価格データはリクエストコスト推定に使用されます。

## 価格同期

価格ソースから最新価格をワンクリック同期:

- 「今すぐ同期」をクリックして最新価格を取得
- 同期結果: 追加数、更新数、失敗数
- 「自動同期」を有効化すると設定間隔で自動取得

自動同期間隔: 1時間 / 6時間 / 12時間 / 24時間 / 7日

## フォールバック価格

価格表に載らないモデルにはフォールバック価格を使用:

- **入力フォールバック** — デフォルト $3.0 / 百万トークン
- **出力フォールバック** — デフォルト $3.0 / 百万トークン

同期セクションで調整可能。

## 価格表

表は以下の列を表示:

| 列 | 説明 |
|--------|-------------|
| モデル | モデル名 |
| ソース | LiteLLM (同期) / 手動 |
| デフォルトプラットフォーム | このモデルのデフォルトプラットフォーム |
| 入力 $/M | 入力トークン価格 (百万トークンあたり) |
| 出力 $/M | 出力トークン価格 |
| キャッシュ読取 $/M | キャッシュトークン読取価格 |

モデル名で検索、ソースで絞り込み、ページサイズ調整とページジャンプ付き。

## 価格管理

- 価格レコードを削除 (行末の削除ボタン)
- 価格変更は即座に反映

## コスト推定

各リクエスト後、AiDog は自動的に:

1. 上流レスポンスからトークン使用量 (入力 / 出力 / キャッシュ) を抽出
2. 価格表で単価を検索
3. 推定コストを計算

価格フォールバック順: カスタム価格 → 同期価格 → フォールバック価格

:::tip{title="注意"}
推定コストは参考用です。実際の請求はプラットフォーム請求に基づきます。
:::

[料金と請求](/ja/stats/pricing) 参照。

# テーマ

AiDog は複数のビルトインテーマを提供し、各テーマはライト・ダークモードに対応します。

## ビルトインテーマ

| テーマ | スタイル |
|-------|-------|
| Liquid Glass | グラスモーフィズム、半透明 |
| Nord | 北極の青色調、落ち着き |
| Dracula | ダークテーマ、人気パレット |
| Catppuccin | 暖色系、目に優しい |
| Solarized | 古典的な科学的パレット |

## テーマの切り替え

**「設定」** ページのテーマセレクターでテーマを選択します。

### ライト/ダークモード

各テーマはライト・ダーク両モードに対応。システム設定に自動追従、または手動切り替え。

## テーマ機能

- **角丸** — 階層化された border-radius システム (sm / md / lg / xl)
- **すりガラス** — ぼかし背景の半透明パネル
- **スムーズな切り替え** — イージングアニメーション付きテーマ切替
- **高コントラスト** — ライト・ダーク両モードで良好な可読性を確保

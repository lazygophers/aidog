# テーマ

AiDog は複数の内蔵テーマを提供しており、それぞれに Light と Dark モードがあります。

## 内蔵テーマ

| テーマ | スタイル |
|-------|--------|
| Liquid Glass | グラスモーフィズム、透明感 |
| Nord | 北極ブルーのトーン、穏やか |
| Dracula | ダークテーマ、人気のカラーパレット |
| Catppuccin | 温かみのある色合い、目に優しい |
| Solarized | クラシックな科学的カラーパレット |

## テーマの切り替え

**「設定」** → 「テーマ」で選択。

### Light/Dark モード

各テーマは Light と Dark の両モードに対応しており、システム設定に追従するか手動で切り替え可能です。

## テーマアーキテクチャ

テーマは CSS 変数システムで実装されています：
- 各テーマは light + dark の CSS 変数セットを提供
- 変数ファイルは `src/themes/` ディレクトリに配置
- テーマ切替時に CSS 変数セットを置換

### カスタムテーマ

カスタムテーマを作成するには：

1. 既存のテーマファイルをテンプレートとしてコピー
2. CSS 変数値を変更
3. `src/themes/index.ts` で新しいテーマを登録
4. アプリケーションを再ビルド

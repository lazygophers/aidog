# Themes

AiDog provides multiple built-in themes, each with Light and Dark modes.

## Built-in Themes

| Theme | Style |
|-------|-------|
| Liquid Glass | Glassmorphism, translucent |
| Nord | Arctic blue tones, calm |
| Dracula | Dark theme, popular palette |
| Catppuccin | Warm colors, easy on the eyes |
| Solarized | Classic scientific palette |

## Switching Themes

Select a theme in the **"Settings"** page theme selector.

### Light/Dark Mode

Each theme supports both Light and Dark modes. Follows system settings automatically or switch manually.

## Theme Features

- **Rounded corners** — Layered border-radius system (sm / md / lg / xl)
- **Frosted glass** — Semi-transparent panels with blurred backgrounds
- **Smooth transitions** — Theme switching with easing animations
- **High contrast** — Ensures good readability in both light and dark modes

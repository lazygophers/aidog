# アプリ設定

**「設定」** ページで AiDog のグローバル動作を設定します。

## プロキシ設定

| 設定項目 | デフォルト | 説明 |
|---------|---------|-------------|
| プロキシポート | `7860` | HTTP プロキシのリスニングポート |
| リクエストタイムアウト | `300s` | アップストリームリクエストのタイムアウト |

## ログ設定

詳細は[ログ設定](/logs/log-settings)を参照。

## データ管理

### データのエクスポート

- プラットフォーム設定のエクスポート（JSON）
- グループ設定のエクスポート（JSON）
- 統計データのエクスポート（CSV）

### データのクリーンアップ

- 期限切れログのクリーンアップ
- 統計データのリセット

## インターフェース設定

### 言語

7 種類のインターフェース言語に対応：
- 简体中文 (zh-CN)
- English (en-US)
- العربية (ar-SA)
- Français (fr-FR)
- Deutsch (de-DE)
- Русский (ru-RU)
- 日本語 (ja-JP)

### テーマ

詳細は[テーマ](/settings/themes)を参照。

## データストレージ

すべてのデータはローカルに保存されます：

| データ | パス |
|------|------|
| データベース | `~/.aidog/aidog.db` |
| 設定ファイル | `~/.aidog/config.json` |

データはお使いのコンピュータから外部に送信されることはありません。

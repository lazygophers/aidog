# アプリ設定

**「設定」** ページで AiDog のグローバル挙動を設定します。5つのタブ: プロキシ、Claude Code、Codex、モデル価格、システムトレイ。

![AiDog — 設定](/screenshots/settings.png)

## プロキシ

### プロキシ設定

| 設定 | デフォルト | 説明 |
|---------|---------|-------------|
| プロキシポート | `9876` | HTTPプロキシリッスンポート |
| リクエストタイムアウト | `300s` | 上流リクエストタイムアウト |
| 自動起動 | オン | AiDog 起動時にプロキシを自動開始 |

### ログ設定

ログの粒度と保持を制御。[ログ設定](/ja/logs/log-settings) 参照。

### データ管理

- プラットフォーム設定をエクスポート
- グループ設定をエクスポート
- 統計をエクスポート
- 期限切れログをクリーンアップ
- 統計をリセット

## Claude Code

モデル、権限、サンドボックス、プラグインなどを含む Claude Code 設定を編集。

- **Claude Code からインポート** — 既存設定を AiDog に読み込み
- **Claude Code へエクスポート** — プロキシアドレスを Claude Code 設定ファイルに書き込み
- **グループ設定を同期** — 各グループの独立設定ファイルを生成

[Codex 統合](/ja/proxy/codex-integration) 参照。

## Codex

Codex 設定を AiDog 内で直接編集、設定ファイルを手動で探す必要なし。

[Codex 統合](/ja/proxy/codex-integration) 参照。

## モデル価格

すべてのモデル価格データを閲覧・検索・管理。[料金と請求](/ja/stats/pricing) 参照。

## システムトレイ

トレイ表示内容と書式を設定。[トレイ設定](/ja/settings/tray) 参照。

## インターフェース

### 言語

7つのインターフェース言語対応: 简体中文, English, العربية, Français, Deutsch, Русский, 日本語

### テーマ

[テーマ](/ja/settings/themes) 参照。

## データセキュリティ

すべてのデータはローカルに保存されます。デバイスから出ることなく、サーバーへアップロードされません。

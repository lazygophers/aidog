# トレイ設定

AiDog はシステムトレイで動作し、メニューバー/タスクバーに主要なステータス情報を表示します。**「設定」→「システムトレイ」**で表示項目とスタイルを設定します。

## トレイ挙動

メインウィンドウを閉じても、AiDog はシステムトレイで動作を続け、プロキシサービスを利用可能に保ちます。トレイアイコンを右クリックしてメインウィンドウの表示/非表示、プロキシの開始/停止、アプリ終了。

## ライブプレビュー

設定ページ上部にシミュレートされたメニューバープレビューを表示。すべての変更が即座に反映されます。プレビューは現在の行使用量 (最大2行) も表示し、超過時に警告します。

## 表示項目タイプ

「表示項目を追加」をクリックして以下のコンテンツタイプを追加:

### プラットフォーム残高/クォータ

有効なプラットフォームを選択して残高または Coding Plan クォータを表示:

| 表示モード | 説明 |
|--------------|-------------|
| 残高 | アカウント残高 (例: `$12.50`) |
| クォータ | Coding Plan 残りパーセンテージ (例: `85%`) |

### 今日の統計

今日の集計データを表示、4つのメトリクスが利用可能:

| メトリクス | 説明 |
|--------|-------------|
| トークン | 今日の総トークン使用量 |
| キャッシュ率 | 今日のキャッシュヒット率 |
| コスト | 今日の推定コスト |
| リクエスト | 今日の総リクエスト数 |

### セパレータ

表示項目間にセパレータを挿入。6つのプリセットスタイル (`|`、`·`、`—`、`/`、`»`、スペース) とカスタム文字。

## 項目カスタマイズ

任意の表示項目を展開して以下のプロパティを調整:

| プロパティ | 説明 |
|----------|-------------|
| 表示 | 残高 / クォータ (プラットフォーム項目のみ) |
| メトリクス | トークン / キャッシュ率 / コスト / リクエスト (今日の統計項目のみ) |
| ラベル | カスタム表示名 (空欄でデフォルト) |
| 小数点 | 値の小数点以下桁数 |
| 行モード | 単一行 (ラベル + 値が同じ行) / 二行 (ラベル上、値下) |
| 配置 | 左 / 中央 / 右 |
| 値行配置 | 二行モードの2行目の独立配置 |
| 色 | システムに従う / プリセット (赤/緑/オレンジ) / カスタム |

### 色の警告

カスタム色選択時、AiDog は一部のメニューバーテーマで色が明確に見えない可能性がある場合に警告します。

## ドラッグで並べ替え

表示項目はドラッグ&ドロップでの並べ替えをサポート。無効化された項目はプレビューに表示されません。

## 起動時自動開始

**「設定」→「プロキシ設定」**でシステムとともに AiDog を自動起動するよう設定。

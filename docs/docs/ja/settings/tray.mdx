# トレイ設定

AiDog はシステムトレイでの常駐をサポートし、バックグラウンドで主要なステータス情報を表示します。

## システムトレイ

メインウィンドウを閉じても、AiDog はシステムトレイで動作を継続し、プロキシサービスを維持します。

### トレイ情報

トレイアイコンに以下を表示可能：
- プロキシサービスのステータス（稼働/停止）
- アカウント残高
- 本日の使用量の概要

### トレイメニュー

トレイアイコンを右クリックで：
- メインウィンドウの表示/非表示
- プロキシの開始/停止
- アプリケーションの終了

## トレイのカスタマイズ

**「設定」** → 「トレイ」で設定：

### 表示内容

トレイに表示する情報を選択：
- 残高情報
- 本日のリクエスト数
- 本日のトークン数
- 本日の推定費用

### ステータスライン設定

ステータスラインの表示形式を設定。カスタムセパレータと表示順序に対応。

## 自動起動

システム起動時に AiDog を自動起動するよう設定可能（トレイに最小化）。

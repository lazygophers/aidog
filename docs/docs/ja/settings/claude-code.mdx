# Claude Code

**「設定」→「Claude Code」** タブで AiDog と Claude Code の統合を設定します。

## 設定編集

すべての Claude Code 設定を AiDog 内で直接編集:

- **モデル** — Claude Code が使用するモデルを指定
- **権限** — ファイル読み書き、ネットワークアクセス制御
- **サンドボックス** — コード実行サンドボックス設定
- **プラグイン** — Claude Code プラグイン管理
- **Hooks** — ライフサイクルフック (PreToolUse / PostToolUse など)
- **StatusLine** — ステータスバー表示設定

すべての変更は `~/.claude.json` に即座に保存されます。

## Claude Code からインポート

「Claude Code からインポート」をクリックして既存の Claude Code 設定ファイルを読み込み、エディタにロードします。初期設定時に既存設定を移行するのに最適。

## Claude Code へエクスポート

「Claude Code へエクスポート」をクリックしてプロキシアドレスを Claude Code 設定ファイルに書き込み:

```json
{
  "apiBaseUrl": "http://127.0.0.1:9876"
}
```

エクスポート後、すべての Claude Code リクエストが AiDog 経由でプロキシされます。

## グループ設定を同期

「グループ設定を同期」をクリックして各グループの独立した Claude Code 設定ファイルを生成:

- `~/.aidog/settings.<group-name>.json` に保存
- 各設定ファイルは対応グループのプロキシパスを使用
- グループカードの「Claude」コピーボタンと併用

### 使用方法

1. グループカードの「Claude」ボタンをクリックして起動コマンドをコピー
2. ターミナルに貼り付け:

```bash
claude --brief --dangerously-skip-permissions \
  --settings ~/.aidog/settings.<group-name>.json
```

## 手動設定

または、環境変数で設定:

```bash
export ANTHROPIC_BASE_URL="http://127.0.0.1:9876"
export ANTHROPIC_API_KEY="any-value"
```

## 注意

- Claude Code はネイティブ Anthropic プロトコルを使用
- 対象プラットフォームが Anthropic プロトコルをサポートしているか、プロトコル変換が設定済みか確認
- AiDog は Claude Code 専用パスを提供
- 必要に応じてプラットフォームまたはグループ設定でモデルマッピングを追加

## 始めるための3ステップ

1. **プラットフォーム追加** —— AI プラットフォームの API キーとエンドポイントを入力
2. **プロキシアドレス設定** —— クライアントを AiDog ローカルプロキシに向ける
3. **使用開始** —— リクエストが自動的にルーティング、計量、ログ記録

→ [インストールガイド](/ja/getting-started/installation) · [クイックスタート](/ja/getting-started/quick-start)

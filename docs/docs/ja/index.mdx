# AiDog

AiDog はクロスプラットフォームのデスクトップアプリケーション（macOS / Windows / Linux）で、**AI API ゲートウェイプロキシ**として、複数の AI プラットフォームの API キーを一元管理、リクエストをインテリジェントにルーティング、使用量とコストを追跡します。

## 主な機能

- **マルチプラットフォーム統合** — OpenAI、Anthropic、Gemini、DeepSeek、Qwen など 50 以上のプラットフォーム API を一元管理
- **スマートルーティング** — グループとモデルマッピングルールによる自動プラットフォーム選択
- **使用量分析** — リアルタイムのトークン追跡、コスト推定、残高照会
- **ログシステム** — 3 段階の記録粒度、設定可能な保持ポリシー
- **Codex 連携** — OpenAI Codex / Claude Code などのコーディングアシスタントにネイティブ対応
- **マルチプロトコル** — OpenAI Chat/Completions/Responses、Anthropic、Gemini ネイティブプロトコル
- **テーマカスタマイズ** — Liquid Glass、Nord、Dracula など
- **システムトレイ** — 残高とステータスを表示する常駐トレイ

## 3 ステップで始める

1. **プラットフォームを追加** —— ある AI プラットフォームの API Key とエンドポイントを入力
2. **プロキシアドレスを設定** —— クライアントを AiDog のローカルプロキシに向ける
3. **利用開始** —— リクエストが自動でルーティング・計測・記録される

→ [インストールガイド](/ja/getting-started/installation) · [クイックスタート](/ja/getting-started/quick-start)

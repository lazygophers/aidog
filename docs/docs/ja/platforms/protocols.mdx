# 対応プラットフォームとインターフェース

AiDog は幅広い AI プラットフォームとインターフェースタイプをサポートし、異なるプラットフォーム間のプロトコル変換を自動処理します。

## 対応プラットフォーム

AiDog は主要 AI サービスをカバーする 50以上の内蔵プラットフォームプリセットを含みます:

| カテゴリ | 例 |
|----------|----------|
| 国際 | OpenAI、Anthropic、Google Gemini、Mistral |
| 中国 | DeepSeek、Qwen、Zhipu GLM、Kimi、MiniMax |
| アグリゲータ | OpenRouter、SiliconFlow、Novita |

プラットフォームプリセットを選択すると全設定が自動入力 — インターフェースタイプを手動選択する必要なし。

## インターフェース自動検出

AiDog はクライアントリクエストからインターフェースタイプを自動検出 — 手動設定不要:

- **チャットインターフェース** — 最も一般的な AI チャットリクエスト
- **コーディングアシスタントインターフェース** — Codex、Claude Code などのツールが使用
- **テキスト補完インターフェース** — レガシーテキスト補完リクエスト

つまり、同じプロキシアドレス経由で異なるタイプのクライアントにサービス提供可能。

## プロトコル自動変換

クライアントが一方のフォーマットでリクエストを送信し、対象プラットフォームが別のフォーマットを使用する場合、AiDog が自動変換。例:

- クライアントが OpenAI フォーマット送信 → Claude 用に Anthropic フォーマットへ自動変換
- コーディングアシスタントが独自インターフェース使用 → プラットフォームのフォーマットへ自動変換

## モックプラットフォーム

AiDog は実際のリクエストを送信せずに AI レスポンスをシミュレートするモックプラットフォームを提供。レスポンス内容、ステータスコード、遅延などをカスタマイズ可能。

# Supported Platforms & Interfaces

AiDog supports a wide range of AI platforms and interface types, automatically handling protocol conversion between different platforms.

## Supported Platforms

AiDog includes 50+ built-in platform presets covering major AI services:

| Category | Examples |
|----------|----------|
| International | OpenAI, Anthropic, Google Gemini, Mistral |
| Chinese | DeepSeek, Qwen, Zhipu GLM, Kimi, MiniMax |
| Aggregators | OpenRouter, SiliconFlow, Novita |

Select a platform preset and all configuration is auto-filled — no need to manually choose an interface type.

## Automatic Interface Detection

AiDog automatically detects the interface type from client requests — no manual configuration needed:

- **Chat interface** — The most common AI chat requests
- **Coding assistant interface** — Used by tools like Codex, Claude Code
- **Text completion interface** — Legacy text completion requests

This means you can serve different types of clients through the same proxy address.

## Automatic Protocol Conversion

If your client sends requests in one format but the target platform uses another, AiDog converts automatically. For example:

- Client sends OpenAI format → Automatically converted to Anthropic format for Claude
- Coding assistant uses its own interface → Automatically converted to the platform's format

## Mock Platform

AiDog provides a mock platform that simulates AI responses without sending real requests. You can customize response content, status codes, delays, and more.

# プラットフォーム追加

プラットフォームは AiDog の中核概念です — 各々が AI API サービスエンドポイントを表します。

![AiDog — プラットフォーム追加](/screenshots/add-platform.png)

## 追加手順

1. **「プラットフォーム」** ページへ移動
2. 右上の **「+」** ボタンをクリック
3. プリセットリストから選択 (または手動入力)
4. API Key を入力し保存をクリック

:::tip{title="自動グループ"}
プラットフォーム保存後、AiDog は自動的にグループを作成するので、手動設定なしで即使用可能。
:::

## 基本設定

| フィールド | 説明 | 例 |
|-------|-------------|---------|
| 名前 | 表示名 | `My OpenAI` |
| ベース URL | API ベース URL (バージョンプレフィックス付き) | `https://api.openai.com/v1` |
| API Key | プラットフォーム API キー | `sk-...` |

### プラットフォームプリセット

AiDog は 50以上の内蔵プラットフォームプリセット (OpenAI、Anthropic、Gemini、DeepSeek、Qwen など) を含みます。プリセット選択で全設定フィールドを自動入力 — API Key を入力するだけ。

一部プリセットは「Coding Plan」マーク付き — クォータ照会を自動設定。

## パススルーモード

一部プラットフォームタイプは「パススルー」モードをサポート: AiDog はプロトコル変換や認証注入なしでクライアントリクエストヘッダーとボディをそのまま転送。

- **上流 URL** — プラットフォームルート URL (例: `https://api.anthropic.com`) を入力、バージョンプレフィックスなし
- **API Key** — パススルーモードでは任意; クライアントが独自認証を持ち運び可能

クライアント側認証 (例: OAuth サブスクリプション) を保持する必要があるシナリオに最適。

## 複数エンドポイント

各プラットフォームは複数エンドポイントを持て、各々に独自アドレスとインターフェースタイプ。AiDog はクライアントリクエストに基づきマッチするエンドポイントを自動選択。

例えば同じプラットフォームがチャットインターフェースとコーディングアシスタントインターフェースの両方をサポート可能。

## モデル設定

### デフォルトモデル

プラットフォーム設定で、異なるリクエストタイプにデフォルトで使用するモデルを指定。AiDog は要求モデル名に基づき最適モデルを自動マッチ。

### モデルリスト取得

「モデル取得」ボタンをクリックしてプラットフォームから利用可能モデルを自動取得。

### モデルマッピング

要求モデル名をプラットフォームがサポートする実際のモデル名にマップ。[モデルマッピング](/ja/platforms/model-mapping) 参照。

## 残高照会設定

NewAPI システムを使用するプラットフォーム向けに別個の残高照会を設定:

- **照会 URL** — API URL とは別
- **照会トークン** — ダッシュボードから取得 (API Key とは別)
- **User ID** — 任意

[残高とクォータ](/ja/stats/balance) 参照。

## 手動予算

自動残高照会なしのプラットフォーム向けに手動予算リミットを設定:

- **合計** — 全体的な支出上限
- **ローリングウィンドウ** — ローリング時間ウィンドウ内の支出上限
- **固定ウィンドウ** — 固定スケジュールでリセットされる支出上限
- **日次** — 日次支出上限

枯渇時、AiDog は該当プラットフォームへの転送を停止し自動フェイルオーバー。[残高とクォータ](/ja/stats/balance) 参照。

## モデルテスト

プラットフォーム保存後、接続性をテスト:

1. プラットフォームカードの「テスト」ボタンをクリック
2. テストモードを選択: クイック / バッチ / カスタム
3. 結果を確認: 応答時間、トークン数、成功/失敗

[モデルテスト](/ja/platforms/model-testing) 参照。

## プラットフォーム一覧

プラットフォームページは追加された全プラットフォームカードを表示、各々:

- プラットフォーム名とロゴ
- ヘルスステータスインジケータ (緑/黄/赤)
- 残高またはクォータ使用率
- 本日のリクエスト数と成功率

プラットフォームを素早く特定するピンイン検索をサポート。

## その他の操作

- **有効化/無効化** — 削除せずにプラットフォームを一時停止
- **編集** — プラットフォームカードをクリックして詳細を表示し設定を変更
- **削除** — 削除ボタンをクリック。関連ログと統計は保持
- **ドラッグで並べ替え** — リスト内のカードをドラッグして表示順序を調整

## FAQ

### ベース URL の入力方法?

ベース URL はバージョンプレフィックスを含む完全なベースパス:

| プラットフォーム | ベース URL |
|----------|----------|
| OpenAI | `https://api.openai.com/v1` |
| Anthropic | `https://api.anthropic.com/v1` |
| DeepSeek | `https://api.deepseek.com/v1` |
| Qwen | `https://dashscope.aliyuncs.com/compatible-mode/v1` |

:::caution{title="注意"}
ベース URL に `/chat/completions` を追加しないでください。AiDog が自動処理します。
:::

### API Key セキュリティ

API キーはローカルに保存され、サーバーへアップロードされません。

# クイックスタート

5 分で AiDog を始めましょう。

![AiDog — Proxy Dashboard](/screenshots/dashboard.png)


## ステップ 1：プラットフォームを追加

1. 左サイドバーの **「プラットフォーム」** をクリック
2. **「+ 追加」** ボタンをクリック
3. 以下を入力：
   - **名前** — 例：`My OpenAI`
   - **Base URL** — API ベースアドレス（例：`https://api.openai.com/v1`）
   - **API Key** — プラットフォームの API キー
   - **プロトコル** — 対応するプロトコルを選択（デフォルト：`openai_chat`）
4. 保存をクリック

:::tip{title="ヒント"}
Base URL にはバージョンプレフィックス（例：`/v1`）が含まれています。AiDog が自動的に `/chat/completions` を追加します。
:::

## ステップ 2：プロキシを設定

AI クライアントで API エンドポイントを AiDog のプロキシアドレスに設定：

```
http://127.0.0.1:7860/v1
```

API キーは任意の値を設定できます（AiDog が設定済みの実際のキーを使用します）。

## ステップ 3：利用開始

`http://127.0.0.1:7860` に送信されるすべてのリクエストが AiDog 経由でルーティングされます：

- リクエストが対応するプラットフォームにルーティング
- 使用量が自動記録
- 費用が自動推定
- ログは「ログ」ページで確認可能

## 動作確認

curl でテスト：

```bash
curl http://127.0.0.1:7860/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-value" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

正常な AI レスポンスが返ってくれば、プロキシは正常に動作しています！

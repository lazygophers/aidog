# 使用統計

AiDog はプロキシ経由の全リクエストの使用量データを自動的に集計します。

## 統計ページ

**「統計」** ページで使用量の概要を確認できます。

### 今日の統計

- 総リクエスト数
- 成功率/失敗率
- 総トークン使用量（Prompt + Completion）
- 推定費用

### トレンド

日/週/月ごとに表示：
- リクエスト数のトレンド
- トークン使用量のトレンド
- 費用のトレンド

### グループ統計

グループカードの統計データは、関連付けられたプラットフォームの統計から集計されます（フロントエンドで合算）。

## 分析軸

以下の軸で統計を確認できます：
- **プラットフォーム別** — 各プラットフォームの使用量
- **グループ別** — 各グループの使用量
- **モデル別** — 各モデルの使用量
- **時間別** — 日/週/月ごとの集計

## データソース

統計は `proxy_logs` テーブルの集計クエリから取得：
- トークン使用量：アップストリーム API レスポンスから抽出
- 推定費用：価格テーブル + モデル名（`resolve_price` フォールバックチェーン）で計算
- リクエスト数：ログ行数の集計

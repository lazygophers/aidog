# 使用統計

**「統計」** ページはすべてのプロキシリクエストの使用概況、トレンド分析、次元別ランキングを提供します。

## 時間範囲

上部に3つの時間プリセットがあります:

| プリセット | 範囲 |
|--------|-------|
| 今日 | 今日の午前0時から |
| 過去7日 | 過去7日間 |
| 過去30日 | 過去30日間 |

プリセット選択時にすべてのメトリクスとグラフが自動更新されます。

## 概要カード

上部に8つのコアメトリクスを表示、各々に期間比較付き:

| メトリクス | 説明 |
|--------|-------------|
| 総リクエスト数 | 期間中の総リクエスト数 |
| 成功率 | 成功リクエストの割合 (色分け) |
| 入力トークン | 総入力トークン |
| 出力トークン | 総出力トークン |
| キャッシュトークン | 総キャッシュヒットトークン |
| キャッシュ率 | 入力トークンに対するキャッシュトークンの割合 |
| 平均レイテンシ | 平均応答時間 (ミリ秒) |
| 推定コスト | 価格表に基づく推定コスト |

比較矢印: 緑上向き = ポジティブ変化 (例: 成功率向上)、赤上向き = ネガティブ変化 (例: コスト増加)。

## フィルタ

時間プリセットの隣で、以下のフィルタでデータを絞り込み:

- **粒度**: 日次 / 時間別
- **グループ化**: プラットフォーム / モデル / グループ
- **グループ**: グループ名で絞り込み (検索可)
- **モデル**: モデル名で絞り込み (検索可)
- **プラットフォームタイプ**: プラットフォームタイプで絞り込み (検索可)

フィルタ変更時にデータが自動リロードされます。

## トレンドグラフ

概要カードの下の棒グラフがリクエスト量トレンドを表示:

- X軸は選択粒度 (日/時間) でバケット化
- 棒の高さがリクエスト数を表す
- 色グラデーションがエラー率を反映 (赤いほど失敗多い)
- ホバーでツールチップ: リクエスト数、成功、エラー、平均レイテンシ、推定コスト

## 次元ランキング表

グラフの下に、現在の次元 (プラットフォーム/モデル/グループ) でランキングする表:

| 列 | 説明 |
|--------|-------------|
| 名前 | プラットフォーム / モデル / グループ名 |
| リクエスト | この次元の総リクエスト数 |
| 成功 | 成功リクエスト数 (健全性で色分け) |
| 入力 | 入力トークン数 |
| 出力 | 出力トークン数 |
| キャッシュ | キャッシュトークン数 |
| 平均レイテンシ | 平均応答時間 |
| 推定コスト | 推定コスト (支出レベルで色分け) |

### ソートとページネーション

- 列ヘッダーをクリックでソート (再度クリックで昇順/降順切り替え)
- ページあたり50行、下部にページネーションコントロール

## リアルタイム更新

新規リクエスト到着時に統計が自動更新 — 手動操作不要。

:::tip{title="注意"}
推定コストは価格表から算出され、参考用です。実際の請求はプラットフォーム請求に基づきます。
:::

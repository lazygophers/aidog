# Pricing & Billing

AiDog automatically estimates request costs based on model pricing tables.

## Cost Estimation

After each request, AiDog:
1. Extracts token usage from the upstream response
2. Looks up the model's unit price in the pricing table
3. Calculates `Cost = Prompt Tokens × Input Price + Completion Tokens × Output Price`

:::tip{title="Note"}
Estimated costs are for reference only. Actual costs are based on platform billing.
:::

## Pricing Sources

AiDog supports multiple pricing data sources:

- **Built-in pricing** — Official pricing for mainstream models
- **Custom pricing** — Manually set prices for specific models, takes priority over built-in pricing
- **Auto-sync** — Automatically fetch the latest model pricing from pricing sources

Pricing fallback order: Custom pricing → Built-in pricing → Default price

## Pricing Management Page

Manage all model pricing in **"Settings" → "Model Pricing"**:

### Browse & Search

- Search by model name
- Filter by pricing source (Built-in / Custom / Synced)
- Paginated browsing of all pricing records

### Edit Custom Pricing

- Add custom unit prices for any model
- Set input token price and output token price separately
- Edit or delete existing custom pricing

### Price Sync

- Configure pricing sync sources
- One-click sync of latest pricing data
- View sync results

## Viewing Costs

In the **"Statistics"** page:
- Today/this week/this month estimated costs
- Cost distribution by platform, group, and model
- Cost trend charts

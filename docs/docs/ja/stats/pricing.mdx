# 料金と課金

AiDog はモデル価格テーブルに基づいて各リクエストの費用を自動推定します。

## 費用推定

各リクエスト完了後、AiDog は以下を実行：
1. アップストリームレスポンスからトークン使用量を取得
2. 価格テーブルでモデルの単価を検索
3. `費用 = Prompt トークン × 入力単価 + Completion トークン × 出力単価` を計算

## 価格ソース

AiDog は内蔵価格データを含み、以下をサポート：
- **内蔵価格** — 主要モデルの公式価格
- **カスタム価格** — 特定モデルの単価を手動設定
- **価格同期** — プラットフォームから最新価格を自動取得

## 推定費用フィールド

各ログエントリの `est_cost` フィールドに推定費用を記録：
- `resolve_price` フォールバックチェーンを使用し、常に価格が利用可能
- フォールバック順：カスタム価格 → 内蔵価格 → デフォルト価格
- フロントエンドは `SUM(est_cost)` で日次/月次費用を集計

:::tip{title="注意"}
推定費用は参考値です。実際の費用はプラットフォームの請求に基づいてください。
:::

## 費用の確認

**「統計」** ページの「料金」タブで：
- 各モデルの価格を確認
- カスタム価格を編集
- 価格同期を実行

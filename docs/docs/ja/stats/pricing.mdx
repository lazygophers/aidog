# 料金と請求

AiDog はモデル価格表に基づき各リクエストのコストを自動推定します。すべての価格データは **「設定」→「モデル価格」** タブで管理します。

## コスト推定

各リクエスト後、AiDog は自動的に:

1. 上流レスポンスからトークン使用量 (入力 / 出力 / キャッシュ) を抽出
2. 価格表で単価を検索
3. 推定コストを計算

価格フォールバック順: カスタム価格 → 同期価格 → フォールバック価格

:::tip{title="注意"}
推定コストは参考用です。実際の請求はプラットフォーム請求に基づきます。
:::

## 価格管理ページ

### 価格同期

AiDog は最新モデル価格データのワンクリック同期をサポートします:

- 「今すぐ同期」をクリックして最新価格を取得
- 同期結果に表示: 追加数、更新数、失敗数
- 「自動同期」を有効化すると設定間隔で自動取得

自動同期間隔: 1時間 / 6時間 / 12時間 / 24時間 / 7日

### フォールバック価格

価格表に載らないモデルにはフォールバック価格を使用:

- **入力フォールバック** — デフォルト $3.0 / 百万トークン
- **出力フォールバック** — デフォルト $3.0 / 百万トークン

「モデル価格」ページの同期セクションで調整可能。

### 閲覧と検索

価格表は以下の列を表示:

| 列 | 説明 |
|--------|-------------|
| モデル | モデル名 |
| ソース | LiteLLM (同期) / 手動 |
| デフォルトプラットフォーム | このモデルのデフォルトプラットフォーム |
| 入力 $/M | 入力トークン価格 (百万トークンあたり) |
| 出力 $/M | 出力トークン価格 |
| キャッシュ読取 $/M | キャッシュトークン読取価格 |

- モデル名で検索
- ソースで絞り込み (LiteLLM / 手動)
- ページサイズ調整とページジャンプ付きページネーション

### 価格管理

- **削除** — 価格レコードを削除 (行末の削除ボタン)
- 価格変更は以降のリクエストに即座に反映

## コスト確認

**「統計」** ページでコスト関連データを確認:

- 概要カードの「推定コスト」 (期間比較付き)
- 次元ランキング表の「推定コスト」列
- トレンドグラフのツールチップ内コスト詳細

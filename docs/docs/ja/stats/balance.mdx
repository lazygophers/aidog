# 残高照会

AiDog は対応プラットフォームのアカウント残高と Coding Plan クォータの照会に対応しています。

## 対応プラットフォーム

| プラットフォーム | 照会内容 |
|----------|---------------|
| OpenAI | アカウント残高 |
| DeepSeek | アカウント残高 |
| StepFun | アカウント残高 |
| SiliconFlow | アカウント残高 |
| OpenRouter | アカウント残高 / 使用量 |
| Novita | アカウント残高 |
| Kimi | アカウント残高 |
| GLM (智譜) | アカウント残高 |
| MiniMax | アカウント残高 |

## 残高の確認

残高情報は以下に表示されます：

1. **プラットフォームカード** — 各プラットフォームのリアルタイム残高
2. **グループカード** — グループ内プラットフォームの残高集計
3. **システムトレイ** — 残高ステータスの表示

## 更新

残高データはスケジュールに従って自動更新されます。手動で更新ボタンをクリックすることも可能です。

## Coding Plan

一部のプラットフォームは Coding Plan クォータ情報を提供しており、AiDog は残高データと併せて照会・表示します。

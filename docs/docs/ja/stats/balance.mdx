# Balance & Quotas

AiDog helps you track account balances and coding plan quotas across platforms, giving you full visibility into your spending.

## Automatic Balance Query

AiDog automatically queries account balances for these platforms:

| Platform | Query Content |
|----------|---------------|
| OpenAI | Account balance |
| DeepSeek | Account balance |
| StepFun | Account balance |
| SiliconFlow | Account balance |
| OpenRouter | Account balance / usage |
| Novita | Account balance |
| Kimi (Moonshot) | Account balance |
| GLM (Zhipu) | Account balance |
| MiniMax | Account balance |

Balance data refreshes automatically on a schedule. You can also manually click the refresh button.

## Coding Plan Quotas

Some platforms offer coding plans, such as OpenAI Plus/Team/Enterprise, Zhipu Coding Plan, Kimi Code Plan, etc.

AiDog queries and displays coding plan quota information:

- Tier names
- Current utilization rate
- Remaining percentage

Platform presets marked with "Coding Plan" automatically configure quota queries.

## NewAPI Balance Query

For platforms using the NewAPI system (e.g. self-hosted API aggregators), AiDog supports separate balance query configuration:

- **Query URL** — Separate from the API URL
- **Query Token** — Obtained from the dashboard (different from the API Key)
- **User ID** — Optional

Fill in these details in the platform's "Balance Query Configuration" section.

## Manual Budget

For platforms that don't support automatic balance queries, AiDog provides manual budget limits.

### Setup

In the platform's "Manual Budget" section, click "Add Limit":

| Type | Description |
|------|-------------|
| Total | Overall spending limit |
| Rolling Window | Spending limit within a rolling time window |
| Fixed Window | Spending limit that resets on a fixed schedule |
| Daily | Daily spending limit |

### Budget Units

- **USD** — Measured by cost
- **Tokens** — Measured by token count

### Window Duration

Rolling window and fixed window support multiple time granularities: minute, hour, day, week, month.

### When Budget is Depleted

When any budget limit is depleted, AiDog stops forwarding requests to that platform (returns 402 error) and automatically triggers failover to other platforms in the group. Forwarding resumes when the window resets.

## Where to See Balances

Balance information is displayed in multiple locations:

- **Platform cards** — Real-time balance and quota utilization per platform
- **Group cards** — Aggregated balance across group platforms
- **System tray** — Configurable balance status display

## Statistics Page

On the **"Statistics"** page, view data by platform, group, or model:

- Today's request count and success rate
- Token usage (input / output / cache)
- Estimated cost
- Usage trend charts (daily / weekly / monthly)

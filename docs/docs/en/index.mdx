## Three steps to get started

1. **Add a platform** —— enter the API key and endpoint of an AI platform
2. **Set the proxy address** —— point your client to the AiDog local proxy
3. **Start using it** —— requests are automatically routed, metered, and logged

→ [Installation Guide](/en/getting-started/installation) · [Quick Start](/en/getting-started/quick-start)

# AiDog

AiDog is a cross-platform desktop app (macOS / Windows / Linux) that acts as an **AI API gateway proxy**, helping you manage multiple AI platform API keys, intelligently route requests, and track usage & costs.

## Core Features

- **Multi-platform aggregation** — Manage APIs for OpenAI, Anthropic, Gemini, DeepSeek, Qwen, and 50+ platforms
- **Smart routing** — Automatic platform selection via groups, model mapping rules
- **Usage analytics** — Real-time token tracking, cost estimation, balance queries
- **Logging system** — 3-level recording granularity, configurable retention policies
- **Codex integration** — Native support for OpenAI Codex / Claude Code and other coding assistants
- **Multi-protocol** — OpenAI Chat/Completions/Responses, Anthropic, Gemini native protocols
- **Theme customization** — Liquid Glass, Nord, Dracula, and more
- **System tray** — Persistent tray display with balance & status

## Quick Start

Download → Launch → Add first platform → Set proxy address → Start using

# Balance Query

AiDog can query account balances and Coding Plan quotas for supported platforms.

## Supported Platforms

| Platform | Query Content |
|----------|---------------|
| OpenAI | Account balance |
| DeepSeek | Account balance |
| StepFun | Account balance |
| SiliconFlow | Account balance |
| OpenRouter | Account balance / usage |
| Novita | Account balance |
| Kimi | Account balance |
| GLM (Zhipu) | Account balance |
| MiniMax | Account balance |

## Viewing Balances

Balance information is displayed in:

1. **Platform cards** — Real-time balance per platform
2. **Group cards** — Aggregated balance across group platforms
3. **System tray** — Balance status display

## Refreshing

Balance data refreshes automatically on a schedule. You can also manually click the refresh button.

## Coding Plan

Some platforms provide Coding Plan quota information, which AiDog queries and displays alongside balance data.

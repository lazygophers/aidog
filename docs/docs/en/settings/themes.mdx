# Themes

AiDog provides multiple built-in themes, each with Light and Dark modes.

## Built-in Themes

| Theme | Style |
|-------|-------|
| Liquid Glass | Glassmorphism, translucent |
| Nord | Arctic blue tones, calm |
| Dracula | Dark theme, popular palette |
| Catppuccin | Warm colors, easy on the eyes |
| Solarized | Classic scientific palette |

## Switching Themes

Select a theme in **"Settings"** → "Themes".

### Light/Dark Mode

Each theme supports both Light and Dark modes, following system settings or manual toggle.

## Theme Architecture

Themes are implemented via CSS variables:
- Each theme provides light + dark CSS variable sets
- Variable files are in the `src/themes/` directory
- Switching themes replaces the CSS variable set

### Custom Themes

To create a custom theme:

1. Copy an existing theme file as a template
2. Modify the CSS variable values
3. Register the new theme in `src/themes/index.ts`
4. Rebuild the application

# App Settings

Configure AiDog's global behavior in the **"Settings"** page.

## Proxy Settings

| Setting | Default | Description |
|---------|---------|-------------|
| Proxy port | `7860` | HTTP proxy listen port |
| Request timeout | `300s` | Upstream request timeout |

## Log Settings

See [Log Settings](/logs/log-settings) for details.

## Data Management

### Export Data

- Export platform config (JSON)
- Export group config (JSON)
- Export statistics (CSV)

### Cleanup Data

- Clean expired logs
- Reset statistics

## Interface Settings

### Language

7 interface languages supported:
- 简体中文 (zh-CN)
- English (en-US)
- العربية (ar-SA)
- Français (fr-FR)
- Deutsch (de-DE)
- Русский (ru-RU)
- 日本語 (ja-JP)

### Themes

See [Themes](/settings/themes) for details.

## Data Storage

All data is stored locally:

| Data | Path |
|------|------|
| Database | `~/.aidog/aidog.db` |
| Config | `~/.aidog/config.json` |

Data never leaves your machine and is never uploaded to any server.

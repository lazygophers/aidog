# Быстрый старт

Начните работу с AiDog за 5 минут.

![AiDog — Панель прокси](/screenshots/dashboard.png)


## Шаг 1: Добавьте платформу

1. Нажмите **«Платформы»** в левой боковой панели
2. Нажмите **«+ Добавить платформу»**
3. Заполните данные:
   - **Имя** — например `My OpenAI`
   - **Base URL** — базовый URL API, например `https://api.openai.com/v1`
   - **API Key** — ваш ключ API платформы
   - **Протокол** — выберите подходящий протокол (по умолчанию: `openai_chat`)
4. Нажмите «Сохранить»

:::tip{title="Подсказка"}
Base URL уже включает префикс версии (например `/v1`). AiDog автоматически добавляет `/chat/completions`.
:::

## Шаг 2: Настройте прокси

В вашем AI-клиенте укажите эндпоинт API как адрес прокси AiDog:

```
http://127.0.0.1:9876/proxy/v1
```

API Key может быть любым (AiDog использует ваш настроенный реальный ключ).

## Шаг 3: Начните использовать

Все запросы к `http://127.0.0.1:9876` теперь маршрутизируются через AiDog:

- Запросы направляются на соответствующую платформу
- Использование автоматически записывается
- Стоимость автоматически оценивается
- Журналы доступны на странице «Журналы»

## Проверка работы

Проверьте через curl:

```bash
curl http://127.0.0.1:9876/proxy/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-value" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

Если вы получили нормальный AI-ответ — прокси работает!

# Создание и управление группами

Группы — ядро маршрутизации AiDog: объединяйте платформы по назначению и управляйте маршрутизацией запросов правилами.

## Создание группы

1. Перейдите на страницу **«Группы»**
2. Нажмите **«+ Добавить группу»**
3. Введите имя и путь
4. Выберите платформы
5. Выберите режим маршрутизации

:::tip{title="Автогруппа"}
При добавлении платформы AiDog автоматически создаёт соответствующую группу. Можно также создавать группы вручную.
:::

## Режимы маршрутизации

### Failover

Режим по умолчанию. Платформы используются по порядку. При сбое текущей автоматически пробуется следующая. Идеально для основных/резервных схем.

```
Запрос → Платформа A (основная) → Успех → Ответ
                ↓ (сбой)
         Платформа B (резервная) → Успех → Ответ
```

### Load Balance

Запросы распределяются по платформам по весу. Идеально для разделения трафика между платформами.

| Платформа | Вес | Ожидаемый трафик |
|----------|--------|-----------------|
| OpenAI | 70 | ~70% |
| DeepSeek | 30 | ~30% |

## Настройки тайм-аута

Каждая группа может независимо задавать тайм-ауты, переопределяя глобальные:

- **Тайм-аут запроса** — максимальное время ожидания ответа upstream
- **Тайм-аут подключения** — максимальное время установки соединения

Оставьте пустым для использования глобальных настроек.

## Управление платформами

В группе можно:
- **Перетаскивать** — менять приоритет (определяет порядок основной/резервная в failover)
- **Включать/Отключать** — временно пропускать платформу
- **Задавать веса** — долю запросов в режиме load balance
- **Маппинг моделей** — маппинг уровня группы, приоритет над платформой

## Команды запуска в один клик

Каждая карточка группы содержит две кнопки копирования:

- **Claude** — копирует команду запуска Claude Code для этой группы, включая путь файла настроек
- **Codex** — копирует команду запуска Codex для этой группы, включая профиль

Вставьте команду в терминал для немедленного использования маршрутизации группы.

## Статус здоровья

Карточки групп показывают индикаторы здоровья связанных платформ:

- 🟢 Все успешны — зелёный
- 🟡 Часть с ошибками — жёлтый
- 🔴 Все с ошибками — красный

## Редактирование и удаление

- Щёлкните карточку группы для входа в режим редактирования. Нажмите «Сохранить» после изменений
- Удаление группы не удаляет её платформы — только убирает связь
- Перетаскивайте карточки для изменения порядка

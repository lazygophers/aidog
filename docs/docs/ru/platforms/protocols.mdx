# Поддерживаемые платформы и интерфейсы

AiDog поддерживает широкий спектр AI-платформ и типов интерфейсов, автоматически обрабатывая преобразование протоколов между разными платформами.

## Поддерживаемые платформы

AiDog включает 50+ встроенных пресетов платформ, охватывающих основные AI-сервисы:

| Категория | Примеры |
|----------|----------|
| Международные | OpenAI, Anthropic, Google Gemini, Mistral |
| Китайские | DeepSeek, Qwen, Zhipu GLM, Kimi, MiniMax |
| Агрегаторы | OpenRouter, SiliconFlow, Novita |

Выберите пресет платформы — вся конфигурация заполнится автоматически, не нужно вручную выбирать тип интерфейса.

## Автоопределение интерфейса

AiDog автоматически определяет тип интерфейса из клиентских запросов — ручная настройка не требуется:

- **Чат-интерфейс** — наиболее частые AI chat-запросы
- **Интерфейс ассистента кода** — используется инструментами вроде Codex, Claude Code
- **Интерфейс text completion** — устаревшие запросы text completion

Это значит, что вы можете обслуживать разные типы клиентов через один адрес прокси.

## Автоматическое преобразование протоколов

Если клиент отправляет запросы в одном формате, а целевая платформа использует другой, AiDog преобразует автоматически. Например:

- Клиент шлёт формат OpenAI → автоматически преобразуется в формат Anthropic для Claude
- Ассистент кода использует свой интерфейс → автоматически преобразуется в формат платформы

## Платформа-заглушка (Mock)

AiDog предоставляет mock-платформу, симулирующую AI-ответы без реальных запросов. Можно настроить содержимое ответа, коды состояния, задержки и другое.

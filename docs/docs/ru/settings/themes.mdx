# Темы

AiDog предоставляет несколько встроенных тем, каждая со светлым и тёмным режимом.

## Встроенные темы

| Тема | Стиль |
|-------|-------|
| Liquid Glass | Glassmorphism, полупрозрачность |
| Nord | Арктические синие тона, спокойная |
| Dracula | Тёмная, популярная палитра |
| Catppuccin | Тёплые цвета, мягкая для глаз |
| Solarized | Классическая научная палитра |

## Переключение тем

Выберите тему в селекторе страницы **«Настройки»**.

### Светлый/Тёмный режим

Каждая тема поддерживает оба режима. Следует системным настройкам или переключается вручную.

## Возможности тем

- **Скруглённые углы** — многоуровневая система border-radius (sm / md / lg / xl)
- **Матовое стекло** — полупрозрачные панели с размытым фоном
- **Плавные переходы** — переключение с анимацией
- **Высокий контраст** — читаемость в обоих режимах

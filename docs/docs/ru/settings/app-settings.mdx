# Настройки приложения

Глобальное поведение AiDog настраивается на странице **«Настройки»**. Содержит подразделы: Системные, Claude Code, Codex, Middleware, Планирование, Уведомления, Цены моделей, Трей, Popover.

![AiDog — Настройки](/screenshots/settings.png)

## Системные настройки

| Настройка | По умолчанию | Описание |
|---------|---------|-------------|
| Порт прокси | `9876` | Порт прослушивания HTTP-прокси |
| Тайм-аут запроса | `300s` | Тайм-аут upstream-запроса |
| Автозапуск | Вкл | Автозапуск прокси при старте AiDog |

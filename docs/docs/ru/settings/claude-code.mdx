# Claude Code

Интеграция AiDog с Claude Code во вкладке **«Настройки» → «Claude Code»**.

## Редактирование конфигурации

Редактируйте все настройки Claude Code прямо в AiDog:

- **Модель** — модель Claude Code
- **Разрешения** — чтение/запись файлов, сетевой доступ
- **Sandbox** — конфигурация песочницы выполнения
- **Плагины** — управление плагинами Claude Code
- **Hooks** — hook'и жизненного цикла (PreToolUse / PostToolUse и др.)
- **StatusLine** — конфигурация строки состояния

Все изменения сразу сохраняются в `~/.claude.json`.

## Импорт из Claude Code

«Импорт из Claude Code» читает существующий файл конфигурации и загружает в редактор.

## Экспорт в Claude Code

«Экспорт в Claude Code» записывает адрес прокси в конфигурацию:

```json
{
  "apiBaseUrl": "http://127.0.0.1:9876"
}
```

После экспорта все запросы Claude Code идут через AiDog.

## Синхронизация настроек группы

«Синхронизация настроек группы» генерирует независимые файлы конфигурации Claude Code для каждой группы:

- Сохраняются в `~/.aidog/settings.<group-name>.json`
- Каждый файл использует путь прокси соответствующей группы
- Используется с кнопкой копирования «Claude» на карточках групп

### Использование

1. Нажмите «Claude» на карточке группы для копирования команды
2. Вставьте в терминал:

```bash
claude --brief --dangerously-skip-permissions \
  --settings ~/.aidog/settings.<group-name>.json
```

## Ручная конфигурация

Через переменные окружения:

```bash
export ANTHROPIC_BASE_URL="http://127.0.0.1:9876"
export ANTHROPIC_API_KEY="any-value"
```

## Примечания

- Claude Code использует нативный протокол Anthropic
- Убедитесь, что целевая платформа поддерживает Anthropic или имеет преобразование протокола
- AiDog предоставляет специальный путь для Claude Code
- Добавьте маппинг моделей при необходимости

# Конфигурация трея

AiDog работает в системном трее, показывая ключевую информацию в строке меню/панели задач. Настраивайте элементы во вкладке **«Настройки» → «Системный трей»**.

## Поведение трея

После закрытия главного окна AiDog продолжает работать в трее, сохраняя прокси доступным. Щёлкните правой кнопкой по иконке трея для показа/скрытия окна, старта/остановки прокси или выхода.

## Живой предпросмотр

Сверху — симуляция строки меню. Все изменения отражаются мгновенно. Предпросмотр показывает текущее использование строк (макс 2) и предупреждает при превышении.

## Типы элементов отображения

«Добавить элемент» для добавления:

### Баланс / Квота платформы

Выберите включённую платформу:

| Режим | Описание |
|--------------|-------------|
| Баланс | Баланс аккаунта (например `$12.50`) |
| Квота | Оставшийся процент Coding Plan (например `85%`) |

### Статистика за сегодня

4 метрики:

| Метрика | Описание |
|--------|-------------|
| Tokens | Сегодняшние токены |
| Cache% | Сегодняшний процент попадания в кэш |
| Cost$ | Сегодняшняя оценочная стоимость |
| Requests | Сегодняшние запросы |

### Разделители

Шесть пресетов (`|`, `·`, `—`, `/`, `»`, пробел) плюс пользовательские символы.

# Цены моделей

Управление ценами моделей во вкладке **«Настройки» → «Цены моделей»**. Данные цен используются для оценки стоимости запросов.

## Синхронизация цен

Синхронизация актуальных цен в один клик:

- «Синхронизировать» для получения последних цен
- Результат: добавлено / обновлено / неудачно
- «Автосинхронизация» для автоматического получения по интервалу

Интервалы: 1 час / 6 часов / 12 часов / 24 часа / 7 дней

## Запасная цена

Для моделей не из таблицы:

- **Входная** — по умолчанию $3.0 / млн токенов
- **Выходная** — по умолчанию $3.0 / млн токенов

Настраивается в разделе синхронизации.

## Таблица цен

| Столбец | Описание |
|--------|-------------|
| Модель | Имя модели |
| Источник | LiteLLM (синхр.) / Вручную |
| Платформа по умолчанию | Платформа по умолчанию |
| Входные $/М | Цена входных (за млн) |
| Выходные $/М | Цена выходных |
| Чтение кэша $/М | Цена чтения кэша |

Поиск по имени, фильтр по источнику, пагинация.

## Управление ценами

- Удаление записи (кнопка в конце строки)
- Изменения вступают в силу сразу

## Оценка стоимости

После каждого запроса AiDog:

1. Извлекает использование токенов из ответа upstream
2. Ищет цену в таблице
3. Рассчитывает стоимость

Порядок отката: Пользовательская → Синхронизированная → Запасная

:::tip{title="Примечание"}
Оценочная стоимость носит справочный характер.
:::

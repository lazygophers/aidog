# Входящие

Входящие хранят исторические уведомления (как popup, так и InboxOnly сохраняются) для последующего просмотра и управления. Показываются на странице Уведомления (`Notifications`).

## Структура уведомления (`Notification`)

| Поле | Описание |
| --- | --- |
| `id` | ID уведомления |
| `notif_type` | Литерал типа (task_complete / waiting_input / error / custom) |
| `title` | Заголовок |
| `body` | Тело |
| `read` | Флаг прочитанности |
| `created_at` | Время создания |

## Операции

| Операция | Команда | Описание |
| --- | --- | --- |
| Список | `notification_inbox_list` | Получить недавние (с лимитом) |
| Число непрочитанных | `notification_inbox_unread` | Число непрочитанных (для бейджа) |
| Отметить прочитанным | `notification_mark_read` | Одно или все |
| Очистить | `notification_clear` | Очистить входящие |

## Фронтенд

- **Страница Уведомления** (боковая панель): список + бейдж непрочитанных + отметить прочитанным + очистить
- **Страница настроек**: переключатели и конфигурация по типам (см. [Конфигурация](/ru/notifications/configuration))

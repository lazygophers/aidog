# AiDog

> Прокси-шлюз API ИИ — мультиплатформенная агрегация, умная маршрутизация, аналитика использования.

AiDog — локальный прокси-шлюз для AI API: агрегирует несколько платформ, обеспечивает интеллектуальную маршрутизацию, отработку отказов, учёт использования и уведомления.

## Основные возможности

- 🌐 **Мультиплатформенная агрегация** — единый локальный эндпоинт проксирует запросы к нескольким AI-провайдерам.
- 🔀 **Умная маршрутизация** — отказоустойчивость, балансировка нагрузки, привязка сессий, маршрутизация с учётом здоровья.
- 🛡️ **Промежуточное ПО** — фильтрация запросов, маскирование чувствительных данных, инъекция, переопределение ответов.
- 📊 **Статистика использования** — запросы, токены, стоимость по платформам, группам и моделям.
- 📋 **Журналирование** — полная запись запросов/ответов с настраиваемым сроком хранения.
- 🔔 **Уведомления** — TTS-озвучивание, всплывающие окна, входящие, hook'и для Claude Code / Codex.
- 🐕 **Системный трей** — управление из трея, настраиваемый popover.

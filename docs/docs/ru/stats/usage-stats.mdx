# Статистика использования

Страница **«Статистика»** предоставляет обзор использования, анализ трендов и рейтинги по измерениям для всех прокси-запросов.

## Временной диапазон

Три пресета сверху:

| Пресет | Диапазон |
|--------|-------|
| Сегодня | С полуночи сегодня |
| Последние 7 дней |Past 7 days |
| Последние 30 дней | За последние 30 дней |

Все метрики и графики обновляются автоматически при выборе пресета.

## Карточки обзора

Вверху 8 ключевых метрик, каждая со сравнением с прошлым периодом:

| Метрика | Описание |
|--------|-------------|
| Всего запросов | Общее число запросов за период |
| Успешность | Процент успешных (цветовая кодировка) |
| Входные токены | Всего входных |
| Выходные токены | Всего выходных |
| Токены кэша | Всего токенов попадания в кэш |
| Доля кэша | Токены кэша как процент от входных |
| Средняя задержка | Среднее время ответа (мс) |
| Оценочная стоимость | Стоимость по таблице цен |

Стрелки сравнения: зелёная вверх — положительное изменение (например выше успешность), красная вверх — отрицательное (выше стоимость).

## Фильтры

Рядом с пресетами времени:

- **Детализация**: По дням / По часам
- **Группировка**: Платформа / Модель / Группа
- **Группа**: фильтр по имени (с поиском)
- **Модель**: фильтр по имени (с поиском)
- **Тип платформы**: фильтр (с поиском)

Данные перезагружаются при изменении фильтров.

## График тренда

Столбчатая диаграмма показывает тренды объёма запросов:

- Ось X — выбранная детализация
- Высота столбца — число запросов
- Цветовой градиент — доля ошибок (краснее — больше сбоев)
- Наведение — всплывающая подсказка: запросы, успехи, ошибки, средняя задержка, стоимость

## Таблица рейтинга по измерению

Под графиком — таблица рейтинга по текущему измерению (платформа/модель/группа):

| Столбец | Описание |
|--------|-------------|
| Имя | Имя платформы / модели / группы |
| Запросы | Всего запросов этого измерения |
| Успех | Число успешных (цветовая кодировка здоровья) |
| Входные | Число входных токенов |
| Выходные | Число выходных токенов |
| Кэш | Число токенов кэша |
| Средняя задержка | Среднее время ответа |
| Оценочная стоимость | Стоимость (цветовая кодировка уровня расходов) |

### Сортировка и пагинация

- Щёлкните заголовок столбца для сортировки (повтор — смена направления)
- 50 строк на страницу с пагинацией

## Обновление в реальном времени

Статистика автообновляется по мере поступления запросов.

:::tip{title="Примечание"}
Оценочная стоимость рассчитывается по таблице цен и носит справочный характер. Фактические списания — по биллингу платформы.
:::

# Баланс и квоты

AiDog помогает отслеживать балансы аккаунтов и квоты coding-планов между платформами.

## Автоматический запрос баланса

AiDog автоматически запрашивает балансы для:

| Платформа | Содержимое запроса |
|----------|---------------|
| OpenAI | Баланс аккаунта |
| DeepSeek | Баланс аккаунта |
| StepFun | Баланс аккаунта |
| SiliconFlow | Баланс аккаунта |
| OpenRouter | Баланс / использование |
| Novita | Баланс аккаунта |
| Kimi (Moonshot) | Баланс аккаунта |
| GLM (Zhipu) | Баланс аккаунта |
| MiniMax | Баланс аккаунта |

Данные баланса обновляются автоматически по расписанию. Можно нажать кнопку обновления вручную.

## Квоты Coding Plan

Некоторые платформы предлагают coding-планы (OpenAI Plus/Team/Enterprise, Zhipu Coding Plan, Kimi Code Plan и др.).

AiDog запрашивает и отображает квоты coding-плана:

- Названия уровней
- Текущая утилизация
- Оставшийся процент

Пресеты платформ с пометкой «Coding Plan» автоматически настраивают запросы квот.

## Запрос баланса NewAPI

Для платформ системы NewAPI (например self-hosted агрегаторы) поддерживается отдельная конфигурация:

- **URL запроса** — отдельно от URL API
- **Токен запроса** — из панели (отличается от API Key)
- **User ID** — опционально

Заполняется в разделе «Конфигурация запроса баланса» платформы.

## Ручной бюджет

Для платформ без авто-запроса баланса — ручные лимиты бюджета.

### Настройка

В разделе «Ручной бюджет» платформы нажмите «Добавить лимит»:

| Тип | Описание |
|------|-------------|
| Total | Общий лимит расходов |
| Rolling Window | Лимит в скользящем окне |
| Fixed Window | Лимит, сбрасываемый по расписанию |
| Daily | Дневной лимит |

### Единицы бюджета

- **USD** — по стоимости
- **Tokens** — по числу токенов

### Длительность окна

Скользящее и фиксированное окно поддерживают: минута, час, день, неделя, месяц.

### При исчерпании бюджета

При исчерпании любого лимита AiDog прекращает перенаправление на эту платформу (возвращает 402) и автоматически запускает отработку отказа на другие платформы группы. Перенаправление возобновляется при сбросе окна.

## Где смотреть балансы

Информация о балансе отображается в нескольких местах:

- **Карточки платформ** — баланс и использование квоты в реальном времени
- **Карточки групп** — агрегированный баланс платформ группы
- **Системный трей** — настраиваемый показ статуса баланса

## Страница статистики

На странице **«Статистика»** — данные по платформам, группам, моделям:

- Запросы и успешность за сегодня
- Использование токенов (входные / выходные / кэш)
- Оценочная стоимость
- Графики трендов (день / неделя / месяц)

# Тарифы и биллинг

AiDog автоматически оценивает стоимость каждого запроса по таблице цен моделей. Управление — во вкладке **«Настройки» → «Цены моделей»**.

## Оценка стоимости

После каждого запроса AiDog автоматически:

1. Извлекает использование токенов (входные / выходные / кэш) из ответа upstream
2. Ищет цену в таблице
3. Рассчитывает оценочную стоимость

Порядок отката цен: Пользовательская → Синхронизированная → Запасная

:::tip{title="Примечание"}
Оценочная стоимость носит справочный характер. Фактические списания — по биллингу платформы.
:::

## Страница управления ценами

### Синхронизация цен

AiDog поддерживает синхронизацию актуальных цен в один клик:

- Нажмите «Синхронизировать» для получения последних цен
- Результат: добавлено / обновлено / неудачно
- Включите «Автосинхронизацию» для автоматического получения по интервалу

Интервалы: 1 час / 6 часов / 12 часов / 24 часа / 7 дней

### Запасная цена

Для моделей не из таблицы используется запасная цена:

- **Входная** — по умолчанию $3.0 / млн токенов
- **Выходная** — по умолчанию $3.0 / млн токенов

Настраивается в разделе синхронизации страницы «Цены моделей».

### Просмотр и поиск

Таблица цен:

| Столбец | Описание |
|--------|-------------|
| Модель | Имя модели |
| Источник | LiteLLM (синхр.) / Вручную |
| Платформа по умолчанию | Платформа по умолчанию для модели |
| Входные $/М | Цена входных токенов (за млн) |
| Выходные $/М | Цена выходных |
| Чтение кэша $/М | Цена чтения токенов кэша |

- Поиск по имени модели
- Фильтр по источнику (LiteLLM / Вручную)
- Пагинация с настраиваемым размером страницы

### Управление ценами

- **Удалить** — убрать запись (кнопка в конце строки)
- Изменения цен вступают в силу сразу для последующих запросов

## Просмотр стоимости

На странице **«Статистика»**:

- «Оценочная стоимость» в карточках обзора (со сравнением периодов)
- Столбец «Оценочная стоимость» в таблице рейтинга
- Детали стоимости во всплывающих подсказках графика трендов

# Справочник API

AiDog предоставляет локальный HTTP API для запроса информации о группе. Все эндпоинты **только localhost** (`127.0.0.1`) и не требуют внешнего сетевого доступа.

## POST /api/group-info

Запрос оценочной информации о группе (баланс, утилизация coding-плана, статистика запросов и т.д.). Только **группы с одной платформой** возвращают применимые данные; многоплатформенные группы или группы без платформ возвращают `applicable: false`.

### Аутентификация

```
Authorization: Bearer <group_name>
```

`group_name` — имя группы, настроенной в AiDog. То же значение, что переменная окружения `ANTHROPIC_AUTH_TOKEN` для Claude Code.

### Запрос

- **Метод**: POST
- **URL**: `http://127.0.0.1:<port>/api/group-info`
- **Заголовки**: `Authorization: Bearer <group_name>`

### Ответ (200 OK)

#### Группа с одной платформой:

```json
{
  "applicable": true,
  "balance": 48.20,
  "spent": 1.23,
  "coding_plan": [
    {
      "name": "5h",
      "utilization": 23,
      "pace": "fast",
      "reset_at": null
    },
    {
      "name": "7d",
      "utilization": 41,
      "pace": "normal",
      "reset_at": null
    }
  ],
  "requests": 42,
  "success_rate": 95.2,
  "cache_rate": 13.3,
  "total_tokens": 123456,
  "currency": "",
  "balance_days_remaining": 5.2
}
```

#### Многоплатформенная / без платформ:

```json
{
  "applicable": false,
  "balance": 0.0,
  "spent": 0.0,
  "coding_plan": [],
  "requests": 0,
  "success_rate": 0.0,
  "cache_rate": 0.0,
  "total_tokens": 0,
  "currency": "",
  "balance_days_remaining": null
}
```

### Поля ответа

| Поле | Тип | Описание |
|-------|------|-------------|
| `applicable` | boolean | Является ли группа одноплатформенной с доступными данными |
| `balance` | number | Оценочный остаток баланса |
| `spent` | number | Накопленные оценочные расходы |
| `coding_plan` | array | Уровни Coding Plan (Claude и др.); пусто для платформ на основе баланса |
| `coding_plan[].name` | string | Имя уровня (5h, 7d и т.д.) |
| `coding_plan[].utilization` | number | Утилизация в процентах (0-100) |
| `coding_plan[].pace` | string | Темп расхода: `fast`, `normal` или `busy` |
| `coding_plan[].reset_at` | number\|null | Timestamp сброса (Unix секунды) |
| `requests` | number | Общее число запросов |
| `success_rate` | number | Успешность (0-100) |
| `cache_rate` | number | Доля попадания в кэш (0-100) |
| `total_tokens` | number | Всего потрачено токенов |
| `currency` | string | Единица валюты |
| `balance_days_remaining` | number\|null | Оценка дней до исчерпания баланса (null если невычислимо) |

### Ошибки

| Статус | Условие |
|--------|-----------|
| 401 | Отсутствует или недействителен заголовок `Authorization` |
| 200 | Группа не найдена → `applicable: false` |

### Пример

```bash
curl -X POST \
  -H "Authorization: Bearer my-group" \
  http://127.0.0.1:9876/api/group-info
```

# المنصات والواجهات المدعومة

يدعم AiDog طيفًا واسعًا من منصات AI وأنواع الواجهات، ويتولى تلقائيًا تحويل البروتوكول بين المنصات المختلفة.

## المنصات المدعومة

يضم AiDog أكثر من 50 مُعدّ منصة مدمج يغطي خدمات AI الرئيسية:

| الفئة | أمثلة |
|----------|----------|
| دولية | OpenAI، Anthropic، Google Gemini، Mistral |
| صينية | DeepSeek، Qwen، Zhipu GLM، Kimi، MiniMax |
| مجمّعات | OpenRouter، SiliconFlow، Novita |

اختر مُعدّ منصة ويُملأ كل الإعداد تلقائيًا — لا حاجة لاختيار نوع الواجهة يدويًا.

## الكشف التلقائي عن الواجهة

يكتشف AiDog نوع الواجهة تلقائيًا من طلبات العميل — لا إعداد يدوي مطلوب:

- **واجهة الدردشة** — طلبات دردشة AI الأكثر شيوعًا
- **واجهة مساعد البرمجة** — تُستخدم من أدوات مثل Codex، Claude Code
- **واجهة إكمال النص** — طلبات إكمال النص القديمة

هذا يعني أنه يمكنك خدمة أنواع مختلفة من العملاء عبر عنوان البروكسي نفسه.

## التحويل التلقائي للبروتوكول

إذا أرسل عميلك الطلبات بتنسيق بينما تستخدم المنصة المستهدفة تنسيقًا آخر، يحوّل AiDog تلقائيًا. مثلًا:

- العميل يرسل تنسيق OpenAI ← يُحوَّل تلقائيًا إلى تنسيق Anthropic لـ Claude
- مساعد البرمجة يستخدم واجهته الخاصة ← يُحوَّل تلقائيًا إلى تنسيق المنصة

## منصة المحاكاة

يوفر AiDog منصة محاكاة تُحاكي استجابات AI دون إرسال طلبات فعلية. يمكنك تخصيص محتوى الاستجابة، رموز الحالة، التأخيرات، والمزيد.

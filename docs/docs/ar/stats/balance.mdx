## عرض الأرصدة

معلومات الرصيد معروضة في:

1. **بطاقات المنصات** — رصيد فوري لكل منصة
2. **بطاقات المجموعات** — رصيد مجمع لمنصات المجموعة
3. **شريط النظام** — عرض حالة الرصيد

## التحديث

يتم تحديث بيانات الرصيد تلقائياً حسب جدول زمني. يمكنك أيضاً النقر يدوياً على زر التحديث.

## Coding Plan

بعض المنصات توفر معلومات حصة Coding Plan، يستعلمها AiDog ويعرضها جنباً إلى جنب مع بيانات الرصيد.

# التسعير والفوترة

يقدر AiDog تلقائياً تكاليف الطلبات بناءً على جداول تسعير النماذج.

## تقدير التكاليف

بعد كل طلب، يقوم AiDog بـ:
1. استخراج استخدام الرموز من استجابة upstream
2. البحث عن سعر وحدة النموذج في جدول التسعير
3. حساب `التكلفة = رموز Prompt × سعر الإدخال + رموز Completion × سعر الإخراج`

## مصادر التسعير

يتضمن AiDog بيانات تسعير مدمجة ويدعم:
- **التسعير المدمج** — الأسعار الرسمية للنماذج الرئيسية
- **التسعير المخصص** — تعيين الأسعار يدوياً لنماذج محددة
- **مزامنة الأسعار** — جلب أحدث الأسعار تلقائياً من المنصات

## حقل التكلفة المقدرة

يسجل حقل `est_cost` في كل سجل التكلفة المقدرة:
- يستخدم سلسلة الاحتياط `resolve_price` لضمان توفر سعر دائماً
- ترتيب الاحتياط: تسعير مخصص ← تسعير مدمج ← سعر افتراضي
- الواجهة الأمامية تستخدم `SUM(est_cost)` لتجميع التكاليف اليومية/الشهرية

:::tip{title="ملاحظة"}
التكاليف المقدرة للإشارة فقط. التكاليف الفعلية تعتمد على فوترة المنصة.
:::

## عرض التكاليف

في صفحة **"الإحصائيات"**، تبويب "التسعير":
- عرض تسعير كل نموذج
- تعديل التسعير المخصص
- تشغيل مزامنة الأسعار

# إحصاءات الاستخدام

توفر صفحة **«الإحصاءات»** نظرة عامة على الاستخدام وتحليل الاتجاه وترتيب الأبعاد لكل طلبات البروكسي.

## النطاق الزمني

ثلاثة إعدادات مسبقة زمنية في الأعلى:

| الإعداد المسبق | النطاق |
|--------|-------|
| اليوم | منذ منتصف الليل اليوم |
| آخر 7 أيام | الـ 7 أيام الماضية |
| آخر 30 يومًا | الـ 30 يومًا الماضية |

تتحدّث كل المقاييس والمخططات تلقائيًا عند اختيار إعداد مسبق.

## بطاقات النظرة العامة

تُعرض ثمانية مقاييس أساسية في الأعلى، كلٌ بمقارنة فترة:

| المقياس | الوصف |
|--------|-------------|
| إجمالي الطلبات | العدد الكلي للطلبات في الفترة |
| معدل النجاح | نسبة الطلبات الناجحة (مُرمّز لونيًا) |
| توكنز الإدخال | إجمالي توكنز الإدخال |
| توكنز الإخراج | إجمالي توكنز الإخراج |
| توكنز الكاش | إجمالي توكنز إصابة الكاش |
| معدل الكاش | توكنز الكاش كنسبة من توكنز الإدخال |
| متوسط الكمون | متوسط زمن الاستجابة بالمللي ثانية |
| التكلفة التقديرية | التكلفة التقديرية وفق جدول الأسعار |

أسهم المقارنة: أخضر لأعلى = تغيّر إيجابي (مثل معدل نجاح أعلى)، أحمر لأعلى = تغيّر سلبي (مثل تكلفة أعلى).

## المرشحات

بجانب الإعدادات الزمنية المسبقة، حدّد البيانات بهذه المرشحات:

- **الحدة**: يومي / بالساعة
- **التجميع حسب**: المنصة / النموذج / المجموعة
- **المجموعة**: تصفية حسب اسم المجموعة (قابل للبحث)
- **النموذج**: تصفية حسب اسم النموذج (قابل للبحث)
- **نوع المنصة**: تصفية حسب نوع المنصة (قابل للبحث)

تُعاد تحميل البيانات تلقائيًا عند تغيير المرشحات.

## مخطط الاتجاه

مخطط أعمدة أسفل بطاقات النظرة العامة يُظهر اتجاهات حجم الطلبات:

- المحور السيني مجمّع حسب الحدة المختارة (يوم/ساعة)
- ارتفاع العمود يمثل عدد الطلبات
- تدرّج اللون يعكس نسبة الأخطاء (أكثر حمرة = مزيد من الفشل)
- مرّر الفأرة للتلميح: الطلبات، النجاحات، الأخطاء، متوسط الكمون، التكلفة التقديرية

## جدول ترتيب الأبعاد

أسفل المخطط، يُظهر جدول الترتيب حسب البُعد الحالي (المنصة/النموذج/المجموعة):

| العمود | الوصف |
|--------|-------------|
| الاسم | اسم المنصة / النموذج / المجموعة |
| الطلبات | إجمالي الطلبات لهذا البُعد |
| النجاح | عدد الطلبات الناجحة (مُرمّز لونيًا للصحة) |
| الإدخال | عدد توكنز الإدخال |
| الإخراج | عدد توكنز الإخراج |
| الكاش | عدد توكنز الكاش |
| متوسط الكمون | متوسط زمن الاستجابة |
| التكلفة التقديرية | التكلفة التقديرية (مُرمّزة لونيًا لمستوى الإنفاق) |

### الترتيب والترقيم

- انقر رأس أي عمود للترتيب (انقر مجددًا للتبديل تصاعدي/تنازلي)
- 50 صفًا لكل صفحة مع عناصر الترقيم في الأسفل

## التحديث اللحظي

تتحدّث الإحصاءات تلقائيًا عند وصول طلبات جديدة — لا حاجة لفعل يدوي.

:::tip{title="ملاحظة"}
تُحسب التكاليف التقديرية من جدول الأسعار وهي للاسترشاد فقط. الفوترة الفعلية تعتمد على فوترة المنصة.
:::

# إحصائيات الاستخدام

يتتبع AiDog تلقائياً بيانات الاستخدام لجميع طلبات الوكيل.

## صفحة الإحصائيات

انتقل إلى صفحة **"الإحصائيات"** لعرض ملخص الاستخدام.

### إحصائيات اليوم

- إجمالي الطلبات
- معدل النجاح/الفشل
- إجمالي استخدام الرموز (Prompt + Completion)
- التكلفة المقدرة

### الاتجاهات

عرض باليوم/الأسبوع/الشهر:
- اتجاه عدد الطلبات
- اتجاه استخدام الرموز
- اتجاه التكاليف

### إحصائيات المجموعات

إحصائيات بطاقات المجموعات يتم تجميعها من المنصات المرتبطة (الجمع في الواجهة الأمامية).

## الأبعاد

عرض الإحصائيات حسب:
- **حسب المنصة** — استخدام كل منصة
- **حسب المجموعة** — استخدام كل مجموعة
- **حسب النموذج** — استخدام كل نموذج
- **حسب الفترة** — تجميع يومي/أسبوعي/شهري

## مصدر البيانات

الإحصائيات مجمعة من جدول `proxy_logs`:
- استخدام الرموز: مستخرج من استجابات API upstream
- التكلفة المقدرة: محسوبة عبر جدول التسعير + اسم النموذج (سلسلة الاحتياط `resolve_price`)
- عدد الطلبات: تجميع صفوف السجلات

# صندوق الوارد

يحفظ صندوق الوارد الإشعارات المؤرشفة (المنبثقة و InboxOnly كلاهما محفوظ)، قابلة للمراجعة والإدارة. يُعرض في صفحة الإشعارات.

## البنية (`Notification`)

| الحقل | الوصف |
| --- | --- |
| `id` | مُعرّف الإشعار |
| `notif_type` | حرفي النوع (task_complete / waiting_input / error / custom) |
| `title` | العنوان |
| `body` | النص |
| `read` | علم القِراءة |
| `created_at` | وقت الإنشاء |

## العمليات

| العملية | الأمر | الوصف |
| --- | --- | --- |
| القائمة | `notification_inbox_list` | أحدث الإشعارات (حد ممكن) |
| غير المقروء | `notification_inbox_unread` | عدّ غير المقروء (شارة) |
| وضع مقروء | `notification_mark_read` | واحدة أو الكل |
| تفريغ | `notification_clear` | تفريغ الصندوق |

## الواجهة الأمامية

- **صفحة الإشعارات** (الشريط الجانبي): قائمة + شارة غير المقروء + وضع مقروء + تفريغ
- **صفحة الإعدادات**: المبدّئات وتكوين الأنواع (راجع [التكوين](/ar/notifications/configuration))

→ رجوع: [نظرة على الإشعارات](/ar/notifications)

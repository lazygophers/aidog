# مرجع API

يوفر AiDog واجهة HTTP محلية للاستعلام عن معلومات المجموعة. جميع نقاط النهاية **محلية فقط** (`127.0.0.1`) ولا تتطلب وصولاً خارجيًا للشبكة.

## POST /api/group-info

استعلم عن المعلومات التقديرية للمجموعة (الرصيد، استخدام Coding Plan، إحصاءات الطلبات، إلخ). فقط **مجموعات المنصة الواحدة** تُرجع بيانات قابلة للتطبيق؛ المجموعات متعددة المنصات أو بلا منصة تُرجع `applicable: false`.

### المصادقة

```
Authorization: Bearer <group_name>
```

`group_name` هو اسم المجموعة المُعدّ في AiDog. وهو نفس قيمة متغير البيئة `ANTHROPIC_AUTH_TOKEN` الذي يستخدمه Claude Code.

### الطلب

- **الطريقة**: POST
- **URL**: `http://127.0.0.1:<port>/api/group-info`
- **الترويسات**: `Authorization: Bearer <group_name>`

### الاستجابة (200 OK)

#### مجموعة منصة واحدة:

```json
{
  "applicable": true,
  "balance": 48.20,
  "spent": 1.23,
  "coding_plan": [
    {
      "name": "5h",
      "utilization": 23,
      "pace": "fast",
      "reset_at": null
    },
    {
      "name": "7d",
      "utilization": 41,
      "pace": "normal",
      "reset_at": null
    }
  ],
  "requests": 42,
  "success_rate": 95.2,
  "cache_rate": 13.3,
  "total_tokens": 123456,
  "currency": "",
  "balance_days_remaining": 5.2
}
```

#### مجموعة متعددة المنصات / بلا منصة:

```json
{
  "applicable": false,
  "balance": 0.0,
  "spent": 0.0,
  "coding_plan": [],
  "requests": 0,
  "success_rate": 0.0,
  "cache_rate": 0.0,
  "total_tokens": 0,
  "currency": "",
  "balance_days_remaining": null
}
```

### حقول الاستجابة

| الحقل | النوع | الوصف |
|-------|------|-------------|
| `applicable` | boolean | ما إذا كانت مجموعة منصة واحدة ذات بيانات متاحة |
| `balance` | number | الرصيد المتبقي التقديري |
| `spent` | number | الإنفاق التقديري التراكمي |
| `coding_plan` | array | مستويات Coding Plan (Claude وغيره)؛ فارغ للمنصات القائمة على الرصيد |
| `coding_plan[].name` | string | اسم المستوى (5h، 7d، إلخ) |
| `coding_plan[].utilization` | number | نسبة الاستخدام (0-100) |
| `coding_plan[].pace` | string | معدل الاستهلاك: `fast` أو `normal` أو `busy` |
| `coding_plan[].reset_at` | number\|null | طابع زمني لإعادة التعيين (ثواني Unix) |
| `requests` | number | إجمالي عدد الطلبات |
| `success_rate` | number | معدل النجاح (0-100) |
| `cache_rate` | number | معدل إصابة الكاش (0-100) |
| `total_tokens` | number | إجمالي التوكنز المستهلكة |
| `currency` | string | وحدة العملة |
| `balance_days_remaining` | number\|null | الأيام التقديرية المتبقية للرصيد (null إذا لا يمكن حسابها) |

### استجابات الأخطاء

| الحالة | الشرط |
|--------|-----------|
| 401 | ترويسة `Authorization` مفقودة أو غير صالحة |
| 200 | المجموعة غير موجودة → `applicable: false` |

### مثال

```bash
curl -X POST \\
  -H "Authorization: Bearer my-group" \\
  http://127.0.0.1:9876/api/group-info
```

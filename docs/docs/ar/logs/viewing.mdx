# عرض السجلّات

تسجّل صفحة **السجلّات** كل طلب بروكسي يمر عبر AiDog، مما يسهّل تشخيص مشاكل التوجيه وتحويل البروتوكول واستجابات المنبع.

## قائمة السجلّات

يُظهر كل سجل:

| الحقل | الوصف |
|------|------|
| المجموعة | اسم المجموعة المطابقة |
| النموذج | النموذج الأصلي المطلوب من العميل |
| النموذج الفعلي | النموذج المرسل للمنبع بعد التحويل |
| البروتوكول | البروتوكول الداخل → بروتوكول المنبع |
| المنصة | المنصة التي عالجت الطلب فعليًا |
| الحالة | رمز حالة HTTP (نجاح / فشل) |
| المدة | إجمالي مدة الطلب (مللي ثانية) |
| التوكنز | عدد توكنز الإدخال / الإخراج / الكاش |
| الوقت | وقت وقوع الطلب |

تُظهر القائمة 50 إدخالًا لكل صفحة مع ترقيم صفحات.

## التصفية

استخدم المرشحات في الأعلى لتحديد موقع السجلّات بسرعة:

- **المنصة**: تصفية حسب منصة محددة
- **المجموعة**: تصفية حسب المجموعة
- **الحالة**: نجاح فقط / فشل فقط
- **النطاق الزمني**: الكل / آخر ساعة / 6 ساعات / 24 ساعة / 7 أيام / 30 يومًا
- **النموذج**: بحث باسم النموذج الأصلي أو الفعلي

## تفاصيل الطلب

انقر أي إدخال لتوسيع تفاصيله ورؤية سلسلة الطلب الكاملة:

- **طلب المستخدم**: رابط الطلب الأصلي، الترويسات، ونص الطلب
- **طلب المنبع**: الرابط المحوّل، الترويسات، ونص الطلب المرسل للمنبع
- **استجابة المنبع**: رمز الحالة، الترويسات، ونص الاستجابة من المنبع
- **استجابة المستخدم**: الترويسات والنص النهائي المُعاد للعميل

تدعم التفاصيل النسخ بنقرة واحدة، للصق في تذكرة دعم أو أداة تصحيح.

:::tip
يتحكم إعداد السجلّات في محتوى «طلب المستخدم» و«طلب المنبع». افتراضيًا لا يُسجَّل نص الطلب. راجع [إعدادات السجلّات](/ar/logs/log-settings).
:::

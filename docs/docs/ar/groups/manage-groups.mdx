# إنشاء وإدارة المجموعات

المجموعات هي جوهر التوجيه في AiDog — نظم المنصات حسب الغرض وتحكم في توجيه الطلبات بالقواعد.

## إنشاء مجموعة

1. انتقل إلى صفحة **"المجموعات"**
2. انقر على **"+ إضافة مجموعة"**
3. التكوين:

| الحقل | الوصف |
|-------|-------------|
| الاسم | اسم عرض المجموعة |
| الوصف | وصف اختياري للمجموعة |
| المنصات | اختر المنصات لتضمينها |
| استراتيجية التوجيه | ترتيب اختيار المنصات |

### استراتيجيات التوجيه

- **متسلسل** — تجربة المنصات بالترتيب
- **عشوائي** — اختيار عشوائي من المنصات المتاحة
- **مرجح** — توزيع الطلبات حسب نسبة الأوزان

## إدارة المنصات

ضمن مجموعة، يمكنك:
- **السحب لإعادة الترتيب** — تعديل الأولوية (الاستراتيجية المتسلسلة)
- **تفعيل/تعطيل** — تخطي منصة مؤقتاً
- **تعيين الأوزان** — نسبة الطلبات للاستراتيجية المرجحة
- **تعيين النماذج** — تجاوز تعيين النماذج على مستوى المجموعة

## تعديل المجموعة

انقر على بطاقة المجموعة للدخول إلى وضع التعديل. انقر على "حفظ" بعد التعديلات.

## حذف المجموعة

حذف مجموعة لا يحذف منصاتها — تتم إزالة الارتباط فقط.

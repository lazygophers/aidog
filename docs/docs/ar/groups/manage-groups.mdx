# إنشاء وإدارة المجموعات

المجموعات هي جوهر توجيه AiDog — نظّم المنصات حسب الغرض وتحكّم في التوجيه بقواعد.

## إنشاء مجموعة

1. اذهب إلى صفحة **«المجموعات»**
2. انقر **«+ إضافة مجموعة»**
3. أدخل اسمًا ومسارًا
4. اختر المنصات لتضمينها
5. اختر وضع التوجيه

:::tip{title="مجموعة تلقائية"}
عند إضافة منصة، ينشئ AiDog مجموعة مقابلة تلقائيًا. يمكنك أيضًا إنشاء مجموعات يدويًا.
:::

## أوضاع التوجيه

### التبديل عند الفشل

الوضع الافتراضي. تُستخدم المنصات بالترتيب. عند فشل الحالية، تُجرّب التالية تلقائيًا. مثالي لإعداد رئيسي/احتياطي.

```
طلب → المنصة A (رئيسية) → نجاح → إرجاع
                ↓ (فشل)
         المنصة B (احتياطية) → نجاح → إرجاع
```

### موازنة الحمل

تُوزَّع الطلبات بين المنصات حسب الوزن. مثالية لتقسيم حركة المنصات المتعددة.

| المنصة | الوزن | الحركة المتوقعة |
|----------|--------|-----------------|
| OpenAI | 70 | ~70% |
| DeepSeek | 30 | ~30% |

## إعدادات المهلة

كل مجموعة يمكنها ضبط مهلاتها بشكل مستقل، متجاوزة الإعدادات العامة:

- **مهلة الطلب** — أقصى وقت لانتظار استجابة المنبع
- **مهلة الاتصال** — أقصى وقت لإقامة الاتصال

اتركه فارغًا لاستخدام المهلات العامة.

## إدارة المنصات

داخل مجموعة:

- **اسحب لإعادة الترتيب** — اضبط الأولوية (يحدد ترتيب رئيسي/احتياطي في وضع الفشل)
- **تفعيل/تعطيل** — تخطَّ منصة مؤقتًا
- **ضبط الأوزان** — نسبة الطلبات لموازنة الحمل
- **ربط النماذج** — ربط مستوى المجموعة، له الأولوية على المنصة

## أوامر تشغيل بنقرة واحدة

كل بطاقة مجموعة توفر زرّي نسخ:

- **Claude** — ينسخ أمر تشغيل Claude Code لهذه المجموعة مع مسار ملف الإعدادات
- **Codex** — ينسخ أمر تشغيل Codex مع الملف الشخصي

الصق الأمر في طرفيتك لاستخدام توجيه المجموعة فورًا.

## حالة الصحة

بطاقات المجموعة تعرض مؤشرات حالة لمنصاتها المرتبطة:

- 🟢 كلها ناجحة — أخضر
- 🟡 بعض الفشل — أصفر
- 🔴 كلها فاشلة — أحمر

## التحرير والحذف

- انقر بطاقة مجموعة للدخول لوضع التحرير. انقر «حفظ» بعد التغييرات
- حذف مجموعة لا يحذف منصاتها — يزيل الارتباط فقط
- اسحب بطاقات المجموعة في القائمة لإعادة الترتيب

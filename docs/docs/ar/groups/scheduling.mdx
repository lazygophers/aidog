# الجدولة وقاطع الدائرة

يوفر AiDog **جدولة ذكية** و**قاطع دائرة على مستوى المنصة** لمنصات متعددة ضمن مجموعة، مُرَكِّبًا الوعي بالصحة والحماية التلقائية فوق توجيه تجاوز الفشل/موازنة الحمل.

## الجدولة الذكية

يحافظ المجدول (`SchedulerState`) على مقاييس صحة لحظية لكل منصة:

| المقياس | الوصف |
| --- | --- |
| زمن EMA | متوسط متحرك أسي (α = 0.3، ~6 عينات أخيرة) |
| قيد التنفيذ | الطلبات غير المكتملة حاليًا (inflight) |
| الإخفاقات المتتالية | عداد إخفاقات القاطع |
| حالة القاطع | الحالة الحالية لآلة ثلاثية الحالات |

عند الاختيار، تُدمج المقاييس — يُفضَّل زمن منخفض، تنفيذ قليل، سليم.

## آلة قاطع الدائرة ثلاثية الحالات

كل منصة تحافظ مستقلة على حالة (`BreakerState`):

```
Closed ──إخفاقات تبلغ الحد──▶ Open ──مهلة──▶ HalfOpen ──نجاح التحقيق──▶ Closed
                                              └─فشل التحقيق──▶ Open
```

| الحالة | الحقل | السلوك |
| --- | --- | --- |
| **Closed** | `fails` | مرور عادي؛ يسجل الإخفاقات المتتالية |
| **Open** | `until_ms` | قاطع مفتوح؛ يُستبعد من المرشحين (`Admission = Reject`) |
| **HalfOpen** | `probes` | تحقيق نصف مفتوح؛ مرور محدود (`Admission = Probe`) |

### إعداد الحدود (`BreakerThresholds`)

| المعامل | الوصف |
| --- | --- |
| `failure_threshold` | إخفاقات لتشغيل القاطع |
| `open_secs` | مدة الفتح (ثوانٍ) |
| `half_open_max` | حد تحقيقات نصف المفتوح |

## القبول (Admission)

يُرجع المجدول نتيجة ثلاثية:

- **Allow** — مرور عادي
- **Reject** — قاطع المنصة Open؛ مستبعد
- **Probe** — مرور نصف مفتوح (محدود)

## مفكوك عن auto_disabled

القاطع (`BreakerState`) والتعطيل التلقائي (`auto_disabled`، فشل المصادقة) يُرشِّحان كـ**اتحاد**:

- Open → مستبعد مؤقتًا (قابل للاسترداد)
- auto_disabled → معطّل باستمرار (استرداد يدوي/مصادقة)
- أحدهما → غير موجَّه

مفكوكان: القاطع حماية مؤقتة واعية بالصحة؛ auto_disabled حالة فشل مصادقة مستمرة.

## جلسة لاصقة (Sticky)

جدول جلسة لاصقة مدمج (`StickyTable`):

- نفس مفتاح الجلسة → نفس المنصة (اتساق السياق)
- LRU في الذاكرة + إزالة TTL
- TTL: 30 دقيقة خمول
- سعة: 4096 مدخل

## نقاط التهيئة

- **مستوى المنصة**: حدود القاطع في تحرير المنصة
- **مستوى المجموعة**: استراتيجية في تحرير المجموعة
- **الافتراضي العام**: حدود في الإعدادات

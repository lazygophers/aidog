# استخدام البروكسي

يشغّل AiDog خادم بروكسي HTTP محلي يستقبل ويحوّل طلبات AI API.

## عنوان البروكسي

العنوان الافتراضي:

```
http://127.0.0.1:9876
```

يمكن تغيير المنفذ في **«الإعدادات → البروكسي»**.

## البدء التلقائي

فعّل «بدء البروكسي تلقائيًا». عند التفعيل، تبدأ خدمة البروكسي تلقائيًا عند إطلاق AiDog.

## الاستخدام الأساسي

وجّه أي عميل AI يدعم عناوين API مخصصة إلى AiDog:

| نوع العميل | الإعداد |
|-------------|---------------|
| OpenAI SDK | اضبط `base_url` لعنوان البروكسي |
| curl | اطلب عنوان البروكسي مباشرة |
| تطبيقات الويب | اضبط endpoint API للبروكسي |
| مساعدات البرمجة | اضبط عنوان API الأساسي |

### Python (OpenAI SDK)

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:9876/proxy/v1",
    api_key="any-value"  # يستخدم AiDog مفتاحك الحقيقي المضبوط
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

### Node.js

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'http://127.0.0.1:9876/proxy/v1',
  apiKey: 'any-value',
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello!' }],
});
```

### curl

```bash
curl http://127.0.0.1:9876/proxy/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-value" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

## البث

يدعم AiDog بث SSE (Server-Sent Events) بالكامل. تُحوَّل طلبات بث العميل بشفافية إلى المنصة المنبعثة.

## المهلة

لطلبات البروكسي حماية مهلة:

- المهلة الافتراضية: 300 ثانية
- قابلة للإعداد عموميًا أو لكل مجموعة
- تُرجع 504 Gateway Timeout عند الانتهاء

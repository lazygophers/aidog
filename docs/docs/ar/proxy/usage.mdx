# Proxy Usage

AiDog runs a local HTTP proxy server that receives and forwards AI API requests.

## Proxy Address

Default proxy address:

```
http://127.0.0.1:9876
```

The port can be changed in **"Settings" → "Proxy"**.

## Auto-Start

Enable "Auto-start proxy" in settings. When enabled, the proxy service starts automatically when AiDog launches.

## Basic Usage

Point any AI client that supports custom API addresses to AiDog:

| Client Type | Configuration |
|-------------|---------------|
| OpenAI SDK | Set `base_url` to proxy address |
| curl | Request proxy address directly |
| Web apps | Set API endpoint to proxy |
| Coding assistants | Set API base URL |

### Python (OpenAI SDK)

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:9876/proxy/v1",
    api_key="any-value"  # AiDog uses your configured real key
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

### Node.js

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'http://127.0.0.1:9876/proxy/v1',
  apiKey: 'any-value',
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello!' }],
});
```

### curl

```bash
curl http://127.0.0.1:9876/proxy/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-value" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

## Streaming

AiDog fully supports SSE (Server-Sent Events) streaming. Client streaming requests are transparently forwarded to the upstream platform.

## Timeout

Proxy requests have timeout protection:
- Default timeout: 300 seconds
- Configurable globally or per-group
- Returns 504 Gateway Timeout on expiry

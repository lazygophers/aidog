# AiDog

AiDog هو تطبيق سطح مكتب متعدد المنصات (macOS / Windows / Linux) يعمل كـ **بوابة وكيل API للذكاء الاصطناعي**، يساعدك في إدارة مفاتيح API لمنصات ذكاء اصطناعي متعددة، وتوجيه الطلبات بذكاء، وتتبع الاستخدام والتكاليف.

## الميزات الأساسية

- **تجميع متعدد المنصات** — إدارة واجهات برمجة تطبيقات OpenAI و Anthropic و Gemini و DeepSeek و Qwen وأكثر من 50 منصة
- **التوجيه الذكي** — اختيار تلقائي للمنصة عبر المجموعات وقواعد تعيين النماذج
- **تحليلات الاستخدام** — تتبع الرموز في الوقت الفعلي، تقدير التكاليف، استعلام الأرصدة
- **نظام السجلات** — 3 مستويات من دقة التسجيل، سياسات احتفاظ قابلة للتكوين
- **تكامل Codex** — دعم أصلي لـ OpenAI Codex / Claude Code ومساعدي البرمجة الآخرين
- **متعدد البروتوكولات** — بروتوكولات OpenAI Chat/Completions/Responses و Anthropic و Gemini الأصلية
- **تخصيص السمات** — Liquid Glass و Nord و Dracula والمزيد
- **شريط النظام** — عرض مستمر للرصيد والحالة

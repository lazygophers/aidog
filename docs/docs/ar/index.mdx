## ثلاث خطوات للبدء

1. **إضافة منصة** —— أدخل مفتاح API ونقطة نهاية منصة AI
2. **ضبط عنوان البروكسي** —— وجّه عميلك إلى بروكسي AiDog المحلي
3. **ابدأ الاستخدام** —— تُوجَّه الطلبات وتُقاس وتُسجَّل تلقائيًا

→ [دليل التثبيت](/ar/getting-started/installation) · [البدء السريع](/ar/getting-started/quick-start)

## ثلاث خطوات للبدء

1. **إضافة منصة** —— أدخِل مفتاح API ونقطة النهاية لإحدى منصات الذكاء الاصطناعي
2. **ضبط عنوان الوكيل** —— وجِّه عميلك إلى وكيل AiDog المحلي
3. **ابدأ الاستخدام** —— يتم توجيه الطلبات وقياسها وتسجيلها تلقائيًا

# البدء السريع

ابدأ مع AiDog في 5 دقائق.

![AiDog — Proxy Dashboard](/screenshots/dashboard.png)


## الخطوة 1: إضافة منصة

1. انقر على **"المنصات"** في الشريط الجانبي
2. انقر على **"+ إضافة"**
3. املأ التفاصيل:
   - **الاسم** — مثلاً `OpenAI الخاص بي`
   - **Base URL** — عنوان API الأساسي، مثل `https://api.openai.com/v1`
   - **API Key** — مفتاح API الخاص بالمنصة
   - **البروتوكول** — اختر البروتوكول المناسب (الافتراضي: `openai_chat`)
4. انقر على حفظ

:::tip{title="نصيحة"}
Base URL يتضمن بالفعل بادئة الإصدار (مثل `/v1`). يضيف AiDog تلقائياً `/chat/completions`.
:::

## الخطوة 2: تكوين الوكيل

في عميل الذكاء الاصطناعي، عيّن نقطة نهاية API على عنوان وكيل AiDog:

```
http://127.0.0.1:7860/v1
```

يمكن أن يكون مفتاح API أي قيمة (يستخدم AiDog مفتاحك الحقيقي المكون).

## الخطوة 3: بدء الاستخدام

جميع الطلبات المرسلة إلى `http://127.0.0.1:7860` يتم توجيهها الآن عبر AiDog:

- يتم توجيه الطلبات إلى المنصة المناسبة
- يتم تسجيل الاستخدام تلقائياً
- يتم تقدير التكاليف تلقائياً
- يمكن عرض السجلات في صفحة "السجلات"

## التحقق من العمل

اختبر باستخدام curl:

```bash
curl http://127.0.0.1:7860/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-value" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "مرحبا!"}]
  }'
```

إذا حصلت على رد ذكاء اصطناعي طبيعي، فالوكيل يعمل!

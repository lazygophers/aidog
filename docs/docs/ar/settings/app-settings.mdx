# إعدادات التطبيق

تكوين السلوك العام لـ AiDog في صفحة **"الإعدادات"**.

![AiDog — Settings](/screenshots/settings.png)


## إعدادات الوكيل

| الإعداد | الافتراضي | الوصف |
|---------|---------|-------------|
| منفذ الوكيل | `7860` | منفذ استماع وكيل HTTP |
| مهلة الطلب | `300s` | مهلة طلب upstream |

## إعدادات السجلات

راجع [إعدادات السجلات](/logs/log-settings) للتفاصيل.

## إدارة البيانات

### تصدير البيانات

- تصدير تكوين المنصات (JSON)
- تصدير تكوين المجموعات (JSON)
- تصدير الإحصائيات (CSV)

### تنظيف البيانات

- تنظيف السجلات منتهية الصلاحية
- إعادة تعيين الإحصائيات

## إعدادات الواجهة

### اللغة

7 لغات واجهة مدعومة:
- 简体中文 (zh-CN)
- English (en-US)
- العربية (ar-SA)
- Français (fr-FR)
- Deutsch (de-DE)
- Русский (ru-RU)
- 日本語 (ja-JP)

### السمات

راجع [السمات](/settings/themes) للتفاصيل.

## تخزين البيانات

جميع البيانات مخزنة محلياً:

| البيانات | المسار |
|------|------|
| قاعدة البيانات | `~/.aidog/aidog.db` |
| التكوين | `~/.aidog/config.json` |

البيانات لا تغادر جهازك ولا يتم رفعها إلى أي خادم.

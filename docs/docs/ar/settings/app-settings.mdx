# إعدادات التطبيق

اضبط السلوك العام لـ AiDog في صفحة **«الإعدادات»**. تحتوي على 5 تبويبات: البروكسي، Claude Code، Codex، أسعار النماذج، وشريط النظام.

![AiDog — الإعدادات](/screenshots/settings.png)

## البروكسي

### إعدادات البروكسي

| الإعداد | الافتراضي | الوصف |
|---------|---------|-------------|
| منفذ البروكسي | `9876` | منفذ استماع البروكسي HTTP |
| مهلة الطلب | `300s` | مهلة طلب المنبع |
| البدء التلقائي | تشغيل | بدء البروكسي تلقائيًا عند إطلاق AiDog |

### إعدادات السجلّات

تحكم في دقة تسجيل السجلّات وفترة الاحتفاظ. راجع [إعدادات السجلّات](/ar/logs/log-settings).

### إدارة البيانات

- تصدير إعداد المنصة
- تصدير إعداد المجموعة
- تصدير الإحصاءات
- تنظيف السجلّات المنتهية
- إعادة تعيين الإحصاءات

## Claude Code

حرّر إعدادات Claude Code بما فيها النموذج، الصلاحيات، sandbox، الإضافات، والمزيد.

- **استيراد من Claude Code** — قراءة الإعداد القائم إلى AiDog
- **تصدير إلى Claude Code** — كتابة عنوان البروكسي في ملف إعداد Claude Code
- **مزامنة إعدادات المجموعة** — توليد ملفات إعداد مستقلة لكل مجموعة

راجع [تكامل Codex](/ar/proxy/codex-integration).

## Codex

حرّر إعدادات Codex مباشرة في AiDog دون البحث يدويًا عن ملفات الإعداد.

راجع [تكامل Codex](/ar/proxy/codex-integration).

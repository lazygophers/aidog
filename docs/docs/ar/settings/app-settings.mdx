# App Settings

Configure AiDog's global behavior in the **"Settings"** page. It contains 5 tabs: Proxy, Claude Code, Codex, Model Pricing, and System Tray.

![AiDog — Settings](/screenshots/settings.png)

## Proxy

### Proxy Settings

| Setting | Default | Description |
|---------|---------|-------------|
| Proxy port | `9876` | HTTP proxy listen port |
| Request timeout | `300s` | Upstream request timeout |
| Auto-start | On | Automatically start proxy when AiDog launches |

### Log Settings

Control logging granularity and retention. See [Log Settings](/logs/log-settings).

### Data Management

- Export platform config
- Export group config
- Export statistics
- Clean expired logs
- Reset statistics

## Claude Code

Edit Claude Code settings including model, permissions, sandbox, plugins, and more.

- **Import from Claude Code** — Read existing configuration into AiDog
- **Export to Claude Code** — Write proxy address to Claude Code config file
- **Sync Group Settings** — Generate independent config files for each group

See [Codex Integration](/proxy/codex-integration).

## Codex

Edit Codex settings directly in AiDog, without manually locating config files.

See [Codex Integration](/proxy/codex-integration).

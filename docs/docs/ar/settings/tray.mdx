# Tray Configuration

AiDog runs in the system tray, displaying key status information in the menu bar/taskbar. Configure display items and styles in **"Settings" → "System Tray"**.

## Tray Behavior

After closing the main window, AiDog continues running in the system tray to keep the proxy service available. Right-click the tray icon to show/hide the main window, start/stop the proxy, or quit the app.

## Live Preview

The top of the configuration page shows a simulated menu bar preview. All changes are reflected instantly. The preview also shows current line usage (max 2 lines) and warns if exceeded.

## Display Item Types

Click "Add Display Item" to add any of these content types:

### Platform Balance / Quota

Select an enabled platform to show its balance or Coding Plan quota:

| Display Mode | Description |
|--------------|-------------|
| Balance | Account balance (e.g. `$12.50`) |
| Quota | Coding Plan remaining percentage (e.g. `85%`) |

### Today's Statistics

Show today's aggregated data, with 4 available metrics:

| Metric | Description |
|--------|-------------|
| Tokens | Today's total token usage |
| Cache% | Today's cache hit rate |
| Cost$ | Today's estimated cost |
| Requests | Today's total request count |

### Separators

Insert separators between display items. Six preset styles (`|`, `·`, `—`, `/`, `»`, space) plus custom characters.

## Item Customization

Expand any display item to adjust these properties:

| Property | Description |
|----------|-------------|
| Display | Balance / Quota (platform items only) |
| Metric | Tokens / Cache% / Cost$ / Requests (today stats items only) |
| Label | Custom display name (leave empty for default) |
| Decimals | Number of decimal places for values |
| Line Mode | Single (label + value on same line) / Two-line (label top, value bottom) |
| Alignment | Left / Center / Right |
| Value Row Alignment | Independent alignment for the second line in two-line mode |
| Color | Follow system / Preset (red/green/orange) / Custom |

### Color Warning

When choosing a custom color, AiDog warns if the color may not be clearly visible under certain menu bar themes.

## Drag to Reorder

Display items support drag-and-drop reordering. Disabled items do not appear in the preview.

## Auto-start on Boot

Configure AiDog to auto-start with the system in **"Settings" → "Proxy Configuration"**.

# تكوين الشريط

يعمل AiDog في شريط النظام، عارضًا معلومات الحالة الأساسية في شريط القوائم/شريط المهام. اضبط عناصر العرض والأنماط في **«الإعدادات» → «شريط النظام»**.

## سلوك الشريط

بعد إغلاق النافذة الرئيسية، يواصل AiDog العمل في شريط النظام لإبقاء خدمة البروكسي متاحة. انقر بزر الفأرة الأيمن على أيقونة الشريط لإظهار/إخفاء النافذة الرئيسية، أو بدء/إيقاف البروكسي، أو إنهاء التطبيق.

## المعاينة المباشرة

يُظهر أعلى صفحة التكوين معاينة محاكاة لشريط القوائم. كل التغييرات تنعكس فورًا. تُظهر المعاينة أيضًا استخدام السطر الحالي (حد أقصى سطرين) وتحذّر عند التجاوز.

## أنواع عناصر العرض

انقر «إضافة عنصر عرض» لإضافة أي من هذه الأنواع:

### رصيد/حصة المنصة

اختر منصة مُفعّلة لعرض رصيدها أو حصة Coding Plan:

| وضع العرض | الوصف |
|--------------|-------------|
| الرصيد | رصيد الحساب (مثل `$12.50`) |
| الحصة | النسبة المتبقية لـ Coding Plan (مثل `85%`) |

### إحصاءات اليوم

اعرض بيانات اليوم المُجمّعة، مع 4 مقاييس متاحة:

| المقياس | الوصف |
|--------|-------------|
| التوكنز | إجمالي استخدام التوكنز اليوم |
| Cache% | معدل إصابة الكاش اليوم |
| Cost$ | التكلفة التقديرية اليوم |
| الطلبات | إجمالي عدد طلبات اليوم |

### الفواصل

أدرج فواصل بين عناصر العرض. ستة أنماط مُسبقة (`|`، `·`، `—`، `/`، `»`، مسافة) بالإضافة إلى أحرف مخصصة.

## تخصيص العناصر

وسّع أي عنصر عرض لضبط هذه الخصائص:

| الخاصية | الوصف |
|----------|-------------|
| العرض | الرصيد / الحصة (عناصر المنصة فقط) |
| المقياس | التوكنز / Cache% / Cost$ / الطلبات (إحصاءات اليوم فقط) |
| التسمية | اسم عرض مخصص (فارغ للافتراضي) |
| المنازل العشرية | عدد المنازل العشرية للقيم |
| وضع السطر | مفرد (تسمية + قيمة في نفس السطر) / سطرين (تسمية أعلى، قيمة أسفل) |
| المحاذاة | يسار / وسط / يمين |
| محاذاة سطر القيمة | محاذاة مستقلة للسطر الثاني في وضع السطرين |
| اللون | اتباع النظام / مُسبق (أحمر/أخضر/برتقالي) / مخصص |

### تحذير اللون

عند اختيار لون مخصص، يحذّر AiDog إن كان اللون قد لا يكون واضحًا تحت بعض سمات شريط القوائم.

## السحب لإعادة الترتيب

تدعم عناصر العرض إعادة الترتيب بالسحب والإفلات. العناصر المُعطّلة لا تظهر في المعاينة.

## البدء التلقائي عند الإقلاع

اضبط AiDog للبدء تلقائيًا مع النظام في **«الإعدادات» → «تكوين البروكسي»**.

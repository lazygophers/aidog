# السمات

يوفر AiDog عدة سمات مدمجة، كل منها بأوضاع Light و Dark.

## السمات المدمجة

| السمة | النمط |
|-------|-------|
| Liquid Glass | زجاجية، شفافة |
| Nord | درجات زرقاء قطبية، هادئة |
| Dracula | سمة داكنة، لوحة ألوان شائعة |
| Catppuccin | ألوان دافئة، مريحة للعين |
| Solarized | لوحة ألوان علمية كلاسيكية |

## تبديل السمات

اختر سمة في **"الإعدادات"** ← "السمات".

### وضع Light/Dark

كل سمة تدعم وضعي Light و Dark، تتبع إعدادات النظام أو التبديل اليدوي.

## بنية السمات

السمات مطبقة عبر نظام متغيرات CSS:
- كل سمة توفر مجموعتين من متغيرات CSS (light + dark)
- ملفات المتغيرات في دليل `src/themes/`
- تبديل السمة يستبدل مجموعة متغيرات CSS

### سمات مخصصة

لإنشاء سمة مخصصة:

1. نسخ ملف سمة موجودة كقالب
2. تعديل قيم متغيرات CSS
3. تسجيل السمة الجديدة في `src/themes/index.ts`
4. إعادة بناء التطبيق

# الثيمات

يوفر AiDog عدة ثيمات مدمجة، لكل منها وضعي فاتح وداكن.

## الثيمات المدمجة

| الثيم | النمط |
|-------|-------|
| Liquid Glass | زجاجية، شبه شفاف |
| Nord | درجات زرقاء قطبية، هادئة |
| Dracula | ثيم داكن، لوحة شائعة |
| Catppuccin | ألوان دافئة، مريحة للعين |
| Solarized | لوحة علمية كلاسيكية |

## تبديل الثيمات

اختر ثيمًا في منتقي الثيمات بصفحة **«الإعدادات»**.

### وضع فاتح/داكن

كل ثيم يدعم الوضعين الفاتح والداكن. يتبع إعدادات النظام تلقائيًا أو يُبدّل يدويًا.

## ميزات الثيم

- **زوايا مدورة** — نظام border-radius متدرّج (sm / md / lg / xl)
- **زجاج مصنف** — ألواح شبه شفافة بخلفيات ضبابية
- **انتقالات سلسة** — تبديل الثيم بحركات easing
- **تباين عالٍ** — يضمن قراءة جيدة في الوضعين الفاتح والداكن

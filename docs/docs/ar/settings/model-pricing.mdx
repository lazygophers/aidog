# أسعار النماذج

أدر كل بيانات أسعار النماذج في تبويب **«الإعدادات» → «أسعار النماذج»**. تُستخدم بيانات الأسعار لتقدير تكاليف الطلبات.

## مزامنة الأسعار

مزامنة بنقرة واحدة لأحدث الأسعار من مصادر التسعير:

- انقر «زامن الآن» لجلب أحدث الأسعار
- نتيجة المزامنة: المُضافة، المُحدّثة، الفاشلة
- فعّل «المزامنة التلقائية» للجلب في فترات مُهيّأة

فترات المزامنة التلقائية: 1 ساعة / 6 ساعات / 12 ساعة / 24 ساعة / 7 أيام

## سعر الرجوع

للنماذج غير المُدرجة في الجدول، يُستخدم سعر رجوع:

- **رجوع الإدخال** — افتراضي $3.0 / مليون توكن
- **رجوع الإخراج** — افتراضي $3.0 / مليون توكن

قابل للتعديل في قسم المزامنة.

## جدول الأسعار

يُظهر الجدول هذه الأعمدة:

| العمود | الوصف |
|--------|-------------|
| النموذج | اسم النموذج |
| المصدر | LiteLLM (مُزامن) / يدوي |
| المنصة الافتراضية | المنصة الافتراضية لهذا النموذج |
| الإدخال $/M | سعر توكن الإدخال (لكل مليون) |
| الإخراج $/M | سعر توكن الإخراج |
| قراءة الكاش $/M | سعر قراءة توكن الكاش |

بحث باسم النموذج، تصفية حسب المصدر، حجم صفحة قابل للتعديل وقفز للصفحة.

## إدارة الأسعار

- حذف سجل سعر (زر الحذف في نهاية الصف)
- تسري تغييرات الأسعار فورًا

## تقدير التكلفة

بعد كل طلب، يقوم AiDog تلقائيًا بـ:

1. استخراج استخدام التوكنز (إدخال / إخراج / كاش) من استجابة المنبع
2. البحث عن السعر الفردي في جدول الأسعار
3. حساب التكلفة التقديرية

ترتيب الرجوع للأسعار: سعر مخصص → سعر مُزامن → سعر رجوع

:::tip{title="ملاحظة"}
التكاليف التقديرية للاسترشاد فقط. الفوترة الفعلية تعتمد على فوترة المنصة.
:::

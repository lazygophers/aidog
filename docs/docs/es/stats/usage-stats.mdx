# Estadísticas de Uso

La página **«Estadísticas»** proporciona un resumen de uso, análisis de tendencias y rankings por dimensión para todas las peticiones de proxy.

## Rango de tiempo

Tres presets de tiempo disponibles arriba:

| Preset | Rango |
|--------|-------|
| Hoy | Desde medianoche de hoy |
| Últimos 7 días | Pasados 7 días |
| Últimos 30 días | Pasados 30 días |

Todas las métricas y gráficos se actualizan automáticamente al seleccionar un preset.

## Tarjetas de resumen

Ocho métricas clave se muestran arriba, cada una con comparación de período:

| Métrica | Descripción |
|--------|-------------|
| Peticiones totales | Número total de peticiones en el período |
| Tasa de éxito | Porcentaje de peticiones exitosas (codificado por color) |
| Tokens de entrada | Total tokens de entrada |
| Tokens de salida | Total tokens de salida |
| Tokens de caché | Total tokens de acierto de caché |
| Tasa de caché | Tokens de caché como porcentaje de tokens de entrada |
| Latencia media | Tiempo medio de respuesta en milisegundos |
| Coste estimado | Coste estimado según tabla de precios |

Flechas de comparación: verde arriba = cambio positivo (p. ej. mayor tasa de éxito), rojo arriba = cambio negativo (p. ej. mayor coste).

## Filtros

Junto a los presets de tiempo, refina datos con estos filtros:

- **Granularidad**: Diario / Por hora
- **Agrupar por**: Plataforma / Modelo / Grupo
- **Grupo**: filtrar por nombre de grupo (buscable)
- **Modelo**: filtrar por nombre de modelo (buscable)
- **Tipo de plataforma**: filtrar por tipo de plataforma (buscable)

Los datos se recargan automáticamente al cambiar los filtros.

## Gráfico de tendencia

Un gráfico de barras bajo las tarjetas de resumen muestra las tendencias del volumen de peticiones:

- El eje X agrupa por la granularidad seleccionada (día/hora)
- La altura de la barra representa el número de peticiones
- El degradado de color refleja el ratio de errores (más rojo = más fallos)
- Hover para tooltip: peticiones, éxitos, errores, latencia media, coste estimado

## Tabla de ranking por dimensión

Bajo el gráfico, una tabla muestra el ranking por la dimensión actual (plataforma/modelo/grupo):

| Columna | Descripción |
|--------|-------------|
| Nombre | Nombre de plataforma / modelo / grupo |
| Peticiones | Total de peticiones para esta dimensión |
| Éxito | Número de peticiones exitosas (codificado por color para salud) |
| Entrada | Número de tokens de entrada |
| Salida | Número de tokens de salida |
| Caché | Número de tokens de caché |
| Latencia media | Tiempo medio de respuesta |
| Coste estimado | Coste estimado (codificado por color para nivel de gasto) |

### Orden y paginación

- Haz clic en cualquier cabecera de columna para ordenar (clic de nuevo para alternar ascendente/descendente)
- 50 filas por página con controles de paginación abajo

## Actualización en tiempo real

Las estadísticas se actualizan automáticamente a medida que llegan nuevas peticiones — sin acción manual.

:::tip{title="Nota"}
Los costes estimados se calculan a partir de la tabla de precios y son solo de referencia. El cobro real se basa en la facturación de la plataforma.
:::

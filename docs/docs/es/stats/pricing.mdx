# Precios y Facturación

AiDog estima automáticamente el coste de cada petición según la tabla de precios de modelos. Gestiona todos los datos de precios en **«Ajustes» → «Precios de Modelos»**.

## Estimación de coste

Tras cada petición, AiDog automáticamente:

1. Extrae el uso de tokens (entrada / salida / caché) de la respuesta upstream
2. Busca el precio unitario en la tabla de precios
3. Calcula el coste estimado

Orden de fallback de precios: Precio personalizado → Precio sincronizado → Precio de fallback

:::tip{title="Nota"}
Los costes estimados son solo de referencia. El cobro real se basa en la facturación de la plataforma.
:::

## Página de gestión de precios

### Sincronización de precios

AiDog soporta la sincronización con un clic de los últimos datos de precios:

- Pulsa «Sincronizar ahora» para obtener los últimos precios
- El resultado muestra: añadidos, actualizados, fallidos
- Activa «Auto-sync» para obtener a un intervalo configurado

Intervalos de auto-sync: 1 hora / 6 horas / 12 horas / 24 horas / 7 días

### Precio de fallback

Para modelos no listados, se usa un precio de fallback:

- **Fallback de entrada** — Por defecto $3.0 / millón de tokens
- **Fallback de salida** — Por defecto $3.0 / millón de tokens

Ajustable en la sección de sincronización de la página «Precios de Modelos».

### Explorar y buscar

La tabla muestra estas columnas:

| Columna | Descripción |
|--------|-------------|
| Modelo | Nombre del modelo |
| Fuente | LiteLLM (sincronizado) / Manual |
| Plataforma por defecto | Plataforma por defecto para este modelo |
| Entrada $/M | Precio token entrada (por millón) |
| Salida $/M | Precio token salida |
| Lectura caché $/M | Precio lectura token caché |

- Búsqueda por nombre de modelo
- Filtro por fuente (LiteLLM / Manual)
- Paginación con tamaño ajustable y salto de página

### Gestionar precios

- **Eliminar** — Quitar un registro de precio (botón eliminar al final de la fila)
- Los cambios de precio surten efecto inmediatamente para peticiones posteriores

## Ver costes

Visualiza datos de coste en la página **«Estadísticas»**:

- «Coste estimado» en las tarjetas de resumen (con comparación de período)
- Columna «Coste estimado» en la tabla de ranking por dimensión
- Detalles de coste en los tooltips del gráfico de tendencia

# Precios y facturación

AiDog estima automáticamente los costos de las solicitudes basándose en tablas de precios de modelos.

## Estimación de costos

Después de cada solicitud, AiDog:
1. Extrae el uso de tokens de la respuesta upstream
2. Busca el precio unitario del modelo en la tabla de precios
3. Calcula `Costo = Tokens Prompt × Precio entrada + Tokens Completion × Precio salida`

## Fuentes de precios

AiDog incluye datos de precios integrados y soporta:
- **Precios integrados** — Precios oficiales para modelos principales
- **Precios personalizados** — Establecer precios manualmente para modelos específicos
- **Sincronización de precios** — Obtener automáticamente los últimos precios de las plataformas

## Campo de costo estimado

El campo `est_cost` de cada entrada de registro registra el costo estimado:
- Usa la cadena de respaldo `resolve_price` para garantizar que siempre haya un precio disponible
- Orden de respaldo: Precios personalizados → Precios integrados → Precio predeterminado
- El frontend usa `SUM(est_cost)` para agregar costos diarios/mensuales

:::tip{title="Nota"}
Los costos estimados son solo de referencia. Los costos reales se basan en la facturación de la plataforma.
:::

## Ver costos

En la página **"Estadísticas"**, pestaña "Precios":
- Ver precios de cada modelo
- Editar precios personalizados
- Activar sincronización de precios

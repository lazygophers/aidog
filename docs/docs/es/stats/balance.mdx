# Consulta de saldo

AiDog puede consultar saldos de cuenta y cuotas de Coding Plan de las plataformas soportadas.

## Plataformas soportadas

| Plataforma | Contenido de consulta |
|----------|---------------|
| OpenAI | Saldo de cuenta |
| DeepSeek | Saldo de cuenta |
| StepFun | Saldo de cuenta |
| SiliconFlow | Saldo de cuenta |
| OpenRouter | Saldo de cuenta / uso |
| Novita | Saldo de cuenta |
| Kimi | Saldo de cuenta |
| GLM (Zhipu) | Saldo de cuenta |
| MiniMax | Saldo de cuenta |

## Ver saldos

La información de saldo se muestra en:

1. **Tarjetas de plataforma** — Saldo en tiempo real por plataforma
2. **Tarjetas de grupo** — Saldo agregado de las plataformas del grupo
3. **Bandeja del sistema** — Visualización del estado del saldo

## Actualización

Los datos de saldo se actualizan automáticamente según un horario. También puedes hacer clic manualmente en el botón de actualizar.

## Coding Plan

Algunas plataformas proporcionan información de cuota de Coding Plan, que AiDog consulta y muestra junto con los datos de saldo.

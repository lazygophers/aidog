# Saldo y Cuotas

AiDog rastrea saldos de cuenta y cuotas de coding plan entre plataformas, dándote visibilidad total de tus gastos.

## Consulta automática de saldo

AiDog consulta automáticamente saldos para estas plataformas:

| Plataforma | Contenido consultado |
|----------|---------------|
| OpenAI | Saldo de cuenta |
| DeepSeek | Saldo de cuenta |
| StepFun | Saldo de cuenta |
| SiliconFlow | Saldo de cuenta |
| OpenRouter | Saldo / uso |
| Novita | Saldo de cuenta |
| Kimi (Moonshot) | Saldo de cuenta |
| GLM (Zhipu) | Saldo de cuenta |
| MiniMax | Saldo de cuenta |

Los datos de saldo se actualizan automáticamente según un calendario. También puedes pulsar el botón de actualización manualmente.

## Cuotas de Coding Plan

Algunas plataformas ofrecen coding plans, como OpenAI Plus/Team/Enterprise, Zhipu Coding Plan, Kimi Code Plan, etc.

AiDog consulta y muestra información de cuota:

- Nombres de los niveles
- Tasa de utilización actual
- Porcentaje restante

Los presets de plataforma marcados con «Coding Plan» configuran consultas de cuota automáticamente.

## Consulta de saldo NewAPI

Para plataformas que usan el sistema NewAPI (p. ej. agregadores de API autoalojados), AiDog soporta configuración separada de consulta de saldo:

- **URL de consulta** — Separada de la URL de API
- **Token de consulta** — Obtenido del panel (diferente de la API Key)
- **User ID** — Opcional

Rellena estos detalles en la sección «Configuración de consulta de saldo» de la plataforma.

## Presupuesto manual

Para plataformas que no soportan consulta automática de saldo, AiDog proporciona límites de presupuesto manual.

### Configuración

En la sección «Presupuesto manual» de la plataforma, pulsa «Añadir límite»:

| Tipo | Descripción |
|------|-------------|
| Total | Límite global de gasto |
| Ventana móvil | Límite en una ventana temporal móvil |
| Ventana fija | Límite que se reinicia según un calendario fijo |
| Diario | Límite de gasto diario |

### Unidades de presupuesto

- **USD** — Medido por coste
- **Tokens** — Medido por número de tokens

### Duración de ventana

La ventana móvil y fija soportan varias granularidades: minuto, hora, día, semana, mes.

### Cuando el presupuesto se agota

Cuando se alcanza un límite, AiDog deja de reenviar peticiones a esa plataforma (devuelve error 402) y activa automáticamente failover a otras plataformas del grupo. El reenvío se reanuda al reiniciarse la ventana.

## Dónde ver los saldos

La información de saldo se muestra en varios lugares:

- **Tarjetas de plataforma** — Saldo en tiempo real y utilización de cuota por plataforma
- **Tarjetas de grupo** — Saldo agregado de las plataformas del grupo
- **Bandeja del sistema** — Visualización configurable del estado de saldo

## Página de Estadísticas

En la página **«Estadísticas»**, visualiza datos por plataforma, grupo o modelo:

- Número de peticiones de hoy y tasa de éxito
- Uso de tokens (entrada / salida / caché)
- Coste estimado
- Gráficos de tendencia de uso (diario / semanal / mensual)

# Balance Query

AiDog can query account balances and Coding Plan quotas for supported platforms.

## Auto-Query Platforms

| Platform | Query Content |
|----------|---------------|
| OpenAI | Account balance |
| DeepSeek | Account balance |
| StepFun | Account balance |
| SiliconFlow | Account balance |
| OpenRouter | Account balance / usage |
| Novita | Account balance |
| Kimi | Account balance |
| GLM (Zhipu) | Account balance |
| MiniMax | Account balance |

## Viewing Balances

Balance information is displayed in:

1. **Platform cards** — Real-time balance per platform
2. **Group cards** — Aggregated balance across group platforms
3. **System tray** — Balance status display

## Refreshing

Balance data refreshes automatically on a schedule. You can also manually click the refresh button.

## Coding Plan

Some platforms provide Coding Plan quota information (e.g. OpenAI Plus/Team/Enterprise plans), which AiDog queries and displays alongside balance data.

## Manual Budget

For platforms that don't support automatic balance queries, AiDog provides a manual budget feature:

### Configuration

Set up a manual budget in the platform configuration. Supported types:

| Type | Description |
|------|-------------|
| Total | Set an overall spending limit |
| Rolling Window | Spending limit within a rolling time window |
| Fixed Period | Spending limit that resets on a fixed schedule |
| Daily | Daily spending limit |

### Budget Units

- **USD** — Measured by cost
- **Tokens** — Measured by token count

### Window Duration

Rolling window and fixed period support multiple time units: minute, hour, day, week, month.

Budget consumption is automatically tracked by AiDog based on proxied requests. When the limit is reached, the platform is marked as out of balance.

# Crear y gestionar grupos

Los grupos son el núcleo de enrutamiento de AiDog — organiza las plataformas por propósito y controla el enrutamiento con reglas.

## Crear un grupo

1. Ve a la página **«Grupos»**
2. Haz clic en **«+ Añadir grupo»**
3. Introduce nombre y ruta
4. Selecciona plataformas
5. Elige modo de enrutamiento

:::tip{title="Grupo auto"}
Al añadir una plataforma, AiDog crea automáticamente un grupo correspondiente. También puedes crear grupos manualmente.
:::

## Modos de enrutamiento

### Failover

Modo por defecto. Las plataformas se usan en orden. Si la actual falla, prueba la siguiente automáticamente. Ideal para configuraciones primario/backup.

```
Petición → Plataforma A (primaria) → Éxito → Devolver
                ↓ (falla)
         Plataforma B (backup) → Éxito → Devolver
```

### Load Balance

Las peticiones se distribuyen por peso. Ideal para dividir tráfico multiplataforma.

| Plataforma | Peso | Tráfico esperado |
|----------|--------|-----------------|
| OpenAI | 70 | ~70% |
| DeepSeek | 30 | ~30% |

## Ajustes de timeout

Cada grupo puede definir timeouts independientes, anulando los globales:

- **Timeout de petición** — Tiempo máximo para respuesta upstream
- **Timeout de conexión** — Tiempo máximo para establecer conexión

Vacío para usar timeouts globales.

## Gestión de plataformas

En un grupo:

- **Arrastrar para reordenar** — Ajusta prioridad (determina orden primario/backup en failover)
- **Activar/Desactivar** — Saltar temporalmente una plataforma
- **Definir pesos** — Ratio de peticiones en load balance
- **Mapeo de modelo** — Mapeo a nivel de grupo, prioridad sobre plataforma

## Comandos de inicio en un clic

Cada tarjeta de grupo ofrece dos botones de copia:

- **Claude** — Copia el comando de inicio de Claude Code para este grupo, con ruta del archivo de ajustes
- **Codex** — Copia el comando de inicio de Codex, con el perfil

Pega el comando en tu terminal para usar el enrutamiento del grupo.

## Estado de salud

Las tarjetas de grupo muestran indicadores para sus plataformas:

- 🟢 Todos con éxito — Verde
- 🟡 Algunos fallos — Amarillo
- 🔴 Todos fallidos — Rojo

## Editar y eliminar

- Haz clic en una tarjeta para entrar en modo edición. «Guardar» tras cambios
- Eliminar un grupo no elimina sus plataformas — solo quita la asociación
- Arrastra las tarjetas para reordenar

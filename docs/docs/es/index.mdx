## Tres pasos para empezar

1. **Añadir una plataforma** —— introduce la clave API y el endpoint de una plataforma AI
2. **Establecer la dirección proxy** —— apunta tu cliente al proxy local de AiDog
3. **Empezar a usarlo** —— las peticiones se enrutan, miden y registran automáticamente

→ [Guía de instalación](/es/getting-started/installation) · [Inicio rápido](/es/getting-started/quick-start)

## Stack técnico

| Capa | Tecnología |

| Frontend | React 19 + TypeScript |
| Backend | Rust + Axum |
| Base de datos | SQLite |
| Build | Vite + Yarn |

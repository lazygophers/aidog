## Primeros pasos en tres pasos

1. **Agregar una plataforma** —— introduce la clave de API y el punto de conexión de una plataforma de IA
2. **Configurar la dirección del proxy** —— apunta tu cliente al proxy local de AiDog
3. **Comenzar a usar** —— las solicitudes se enrutan, miden y registran automáticamente

→ [Guía de instalación](/es/getting-started/installation) · [Inicio rápido](/es/getting-started/quick-start)

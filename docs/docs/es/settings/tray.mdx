# Configuración de Bandeja

AiDog se ejecuta en la bandeja del sistema, mostrando información clave de estado en la barra de menús/barra de tareas. Configura los ítems de visualización y estilos en **«Ajustes» → «Bandeja del Sistema»**.

## Comportamiento de la bandeja

Tras cerrar la ventana principal, AiDog sigue ejecutándose en la bandeja del sistema para mantener el servicio proxy disponible. Clic derecho en el icono de la bandeja para mostrar/ocultar la ventana principal, iniciar/detener el proxy o salir de la app.

## Vista previa en vivo

La parte superior de la página de configuración muestra una vista previa simulada de la barra de menús. Todos los cambios se reflejan al instante. La vista previa también muestra el uso de línea actual (máx. 2 líneas) y avisa si se excede.

## Tipos de ítem de visualización

Pulsa «Añadir ítem de visualización» para añadir cualquiera de estos tipos de contenido:

### Saldo/Cuota de plataforma

Selecciona una plataforma habilitada para mostrar su saldo o cuota de Coding Plan:

| Modo de visualización | Descripción |
|--------------|-------------|
| Saldo | Saldo de cuenta (p. ej. `$12.50`) |
| Cuota | Porcentaje restante de Coding Plan (p. ej. `85%`) |

### Estadísticas de hoy

Muestra los datos agregados de hoy, con 4 métricas disponibles:

| Métrica | Descripción |
|--------|-------------|
| Tokens | Uso total de tokens de hoy |
| Caché% | Tasa de acierto de caché de hoy |
| Coste$ | Coste estimado de hoy |
| Peticiones | Número total de peticiones de hoy |

### Separadores

Inserta separadores entre ítems de visualización. Seis estilos preset (`|`, `·`, `—`, `/`, `»`, espacio) más caracteres personalizados.

## Personalización de ítems

Expande cualquier ítem de visualización para ajustar estas propiedades:

| Propiedad | Descripción |
|----------|-------------|
| Visualización | Saldo / Cuota (solo ítems de plataforma) |
| Métrica | Tokens / Caché% / Coste$ / Peticiones (solo estadísticas de hoy) |
| Etiqueta | Nombre de visualización personalizado (vacío para defecto) |
| Decimales | Número de decimales para valores |
| Modo de línea | Única (etiqueta + valor misma línea) / Dos líneas (etiqueta arriba, valor abajo) |
| Alineación | Izquierda / Centro / Derecha |
| Alineación de fila de valor | Alineación independiente para la segunda línea en modo dos líneas |
| Color | Seguir sistema / Preset (rojo/verde/naranja) / Personalizado |

### Advertencia de color

Al elegir un color personalizado, AiDog avisa si el color puede no ser claramente visible bajo ciertos temas de barra de menús.

## Arrastrar para reordenar

Los ítems de visualización soportan reordenamiento por arrastrar y soltar. Los ítems deshabilitados no aparecen en la vista previa.

## Auto-inicio al arrancar

Configura AiDog para auto-iniciar con el sistema en **«Ajustes» → «Configuración de Proxy»**.

# Ajustes de la App

Configura el comportamiento global de AiDog en la página **«Ajustes»**. Contiene 5 pestañas: Proxy, Claude Code, Codex, Precios de Modelos y Bandeja del Sistema.

![AiDog — Ajustes](/screenshots/settings.png)

## Proxy

### Ajustes de Proxy

| Ajuste | Por defecto | Descripción |
|---------|---------|-------------|
| Puerto proxy | `9876` | Puerto de escucha del proxy HTTP |
| Timeout de petición | `300s` | Timeout de petición upstream |
| Auto-inicio | Activado | Iniciar el proxy automáticamente al lanzar AiDog |

### Ajustes de Logs

Controla la granularidad y retención del logging. Consulta [Ajustes de logs](/es/logs/log-settings).

### Gestión de Datos

- Exportar config de plataforma
- Exportar config de grupo
- Exportar estadísticas
- Limpiar logs expirados
- Reiniciar estadísticas

## Claude Code

Edita los ajustes de Claude Code incluyendo modelo, permisos, sandbox, plugins y más.

- **Importar desde Claude Code** — Lee la configuración existente hacia AiDog
- **Exportar a Claude Code** — Escribe la dirección proxy en el archivo de config de Claude Code
- **Sincronizar ajustes de grupo** — Genera archivos de config independientes para cada grupo

Consulta [Integración con Codex](/es/proxy/codex-integration).

## Codex

Edita los ajustes de Codex directamente en AiDog, sin localizar manualmente los archivos de config.

Consulta [Integración con Codex](/es/proxy/codex-integration).

## Precios de Modelos

Explora, busca y gestiona todos los datos de precios de modelos. Consulta [Precios y Facturación](/es/stats/pricing).

## Bandeja del Sistema

Configura el contenido y formato de visualización de la bandeja. Consulta [Configuración de Bandeja](/es/settings/tray).

## Seguridad de Datos

Todos los datos se almacenan localmente. Nunca salen de tu máquina ni se suben a ningún servidor.

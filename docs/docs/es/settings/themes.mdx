# Temas

AiDog proporciona varios temas integrados, cada uno con modos Claro y Oscuro.

## Temas integrados

| Tema | Estilo |
|-------|-------|
| Liquid Glass | Glassmorfismo, translúcido |
| Nord | Tonos azules árticos, calmado |
| Dracula | Tema oscuro, paleta popular |
| Catppuccin | Colores cálidos, descansado para la vista |
| Solarized | Paleta científica clásica |

## Cambiar de tema

Selecciona un tema en el selector de la página **«Ajustes»**.

### Modo Claro/Oscuro

Cada tema soporta los modos Claro y Oscuro. Sigue automáticamente los ajustes del sistema o se cambia manualmente.

## Características del tema

- **Esquinas redondeadas** — Sistema de border-radius en capas (sm / md / lg / xl)
- **Cristal esmerilado** — Paneles semitransparentes con fondos desenfocados
- **Transiciones suaves** — Cambio de tema con animaciones de easing
- **Alto contraste** — Garantiza buena legibilidad en modos claro y oscuro

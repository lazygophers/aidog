# Claude Code

Configura la integración de AiDog con Claude Code en la pestaña **«Ajustes» → «Claude Code»**.

## Edición de configuración

Edita todos los ajustes de Claude Code directamente en AiDog:

- **Modelo** — Especifica el modelo que usa Claude Code
- **Permisos** — Controles de lectura/escritura de archivos, acceso a red
- **Sandbox** — Configuración del sandbox de ejecución de código
- **Plugins** — Gestión de plugins de Claude Code
- **Hooks** — Hooks de ciclo de vida (PreToolUse / PostToolUse, etc.)
- **StatusLine** — Configuración de visualización de la barra de estado

Todos los cambios se guardan inmediatamente en `~/.claude.json`.

## Importar desde Claude Code

Pulsa «Importar desde Claude Code» para leer el archivo de configuración de Claude Code existente y cargarlo en el editor. Ideal para migrar configuraciones existentes durante la configuración inicial.

## Exportar a Claude Code

Pulsa «Exportar a Claude Code» para escribir la dirección proxy en el archivo de configuración de Claude Code:

```json
{
  "apiBaseUrl": "http://127.0.0.1:9876"
}
```

Tras la exportación, todas las peticiones de Claude Code se proxificarán a través de AiDog.

## Sincronizar ajustes de grupo

Pulsa «Sincronizar ajustes de grupo» para generar archivos de configuración independientes de Claude Code para cada grupo:

- Guardados en `~/.aidog/settings.<group-name>.json`
- Cada archivo de configuración usa la ruta proxy del grupo correspondiente
- Usar junto con el botón de copiar «Claude» en las tarjetas de grupo

### Uso

1. Pulsa el botón «Claude» en una tarjeta de grupo para copiar el comando de inicio
2. Pega en tu terminal:

```bash
claude --brief --dangerously-skip-permissions \
  --settings ~/.aidog/settings.<group-name>.json
```

## Configuración manual

Alternativamente, configura mediante variables de entorno:

```bash
export ANTHROPIC_BASE_URL="http://127.0.0.1:9876"
export ANTHROPIC_API_KEY="any-value"
```

## Notas

- Claude Code usa el protocolo Anthropic nativo
- Asegúrate de que la plataforma destino soporta el protocolo Anthropic o tiene conversión de protocolo configurada
- AiDog proporciona una ruta dedicada para Claude Code
- Añade mapeo de modelos en ajustes de plataforma o grupo si es necesario

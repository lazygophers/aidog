# Uso del proxy

AiDog ejecuta un servidor proxy HTTP local que recibe y reenvía peticiones a la API de IA.

## Dirección del proxy

Dirección por defecto:

```
http://127.0.0.1:9876
```

El puerto se cambia en **«Ajustes → Proxy»**.

## Auto-inicio

Activa «Auto-iniciar proxy». Cuando está activo, el servicio proxy arranca automáticamente al lanzar AiDog.

## Uso básico

Apunta cualquier cliente de IA que soporte direcciones de API personalizadas a AiDog:

| Tipo de cliente | Configuración |
|-------------|---------------|
| OpenAI SDK | `base_url` a la dirección del proxy |
| curl | Pide la dirección del proxy directamente |
| Apps web | Endpoint de API al proxy |
| Asistentes de código | URL base de la API |

### Python (OpenAI SDK)

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:9876/proxy/v1",
    api_key="any-value"  # AiDog usa tu clave real configurada
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)
```

### Node.js

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'http://127.0.0.1:9876/proxy/v1',
  apiKey: 'any-value',
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello!' }],
});
```

### curl

```bash
curl http://127.0.0.1:9876/proxy/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer any-value" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

## Streaming

AiDog soporta completamente streaming SSE (Server-Sent Events). Las peticiones de streaming del cliente se reenvían transparentemente a la plataforma upstream.

## Timeout

Las peticiones del proxy tienen protección de timeout:

- Timeout por defecto: 300 segundos
- Configurable globalmente o por grupo
- Devuelve 504 Gateway Timeout al expirar

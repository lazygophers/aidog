# Plataformas e Interfaces Soportadas

AiDog soporta una amplia gama de plataformas AI y tipos de interfaz, manejando automáticamente la conversión de protocolo entre diferentes plataformas.

## Plataformas soportadas

AiDog incluye 50+ presets de plataforma integrados que cubren los principales servicios AI:

| Categoría | Ejemplos |
|----------|----------|
| Internacionales | OpenAI, Anthropic, Google Gemini, Mistral |
| Chinas | DeepSeek, Qwen, Zhipu GLM, Kimi, MiniMax |
| Agregadores | OpenRouter, SiliconFlow, Novita |

Selecciona un preset de plataforma y toda la configuración se autorellena — sin necesidad de elegir manualmente un tipo de interfaz.

## Detección automática de interfaz

AiDog detecta automáticamente el tipo de interfaz de las peticiones del cliente — sin configuración manual:

- **Interfaz de chat** — Las peticiones de chat AI más comunes
- **Interfaz de asistente de codificación** — Usada por herramientas como Codex, Claude Code
- **Interfaz de completado de texto** — Peticiones de completado de texto heredadas

Esto significa que puedes servir diferentes tipos de clientes a través de la misma dirección proxy.

## Conversión automática de protocolo

Si tu cliente envía peticiones en un formato pero la plataforma destino usa otro, AiDog convierte automáticamente. Por ejemplo:

- El cliente envía formato OpenAI → Automáticamente convertido a formato Anthropic para Claude
- El asistente de codificación usa su propia interfaz → Automáticamente convertido al formato de la plataforma

## Plataforma mock

AiDog proporciona una plataforma mock que simula respuestas AI sin enviar peticiones reales. Puedes personalizar el contenido de las respuestas, códigos de estado, retrasos y más.

## In drei Schritten startklar

1. **Plattform hinzufügen** — trage den API-Key und Endpunkt einer KI-Plattform ein
2. **Proxy-Adresse einstellen** — richte deinen Client auf den lokalen AiDog-Proxy aus
3. **Loslegen** — Anfragen werden automatisch geroutet, gemessen und protokolliert

→ [Installationsanleitung](/de/getting-started/installation) · [Schnellstart](/de/getting-started/quick-start)

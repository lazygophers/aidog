## Drei Schritte zum Start

1. **Plattform hinzufügen** —— API-Key und Endpoint einer AI-Plattform eingeben
2. **Proxy-Adresse festlegen** —— Client auf den AiDog-Local-Proxy zeigen
3. **Nutzen starten** —— Anfragen werden automatisch geroutet, gemessen und geloggt

→ [Installationsanleitung](/de/getting-started/installation) · [Schnellstart](/de/getting-started/quick-start)

# AiDog

AiDog ist eine plattformübergreifende Desktop-Anwendung (macOS / Windows / Linux), die als **KI-API-Gateway-Proxy** fungiert und Ihnen hilft, API-Schlüssel mehrerer KI-Plattformen zu verwalten, Anfragen intelligent zu routen und Nutzung sowie Kosten zu verfolgen.

## Kernfunktionen

- **Multi-Plattform-Aggregation** — API-Verwaltung für OpenAI, Anthropic, Gemini, DeepSeek, Qwen und 50+ Plattformen
- **Intelligentes Routing** — Automatische Plattformauswahl über Gruppen und Modellzuordnungsregeln
- **Nutzungsanalysen** — Echtzeit-Token-Tracking, Kostenschätzung, Guthabenabfrage
- **Protokollierungssystem** — 3-stufige Aufzeichnungsgranularität, konfigurierbare Aufbewahrungsrichtlinien
- **Codex-Integration** — Native Unterstützung für OpenAI Codex / Claude Code und andere Programmierassistenten
- **Multi-Protokoll** — OpenAI Chat/Completions/Responses, Anthropic, Gemini native Protokolle
- **Theme-Anpassung** — Liquid Glass, Nord, Dracula und mehr
- **Systemleiste** — Persistente Anzeige von Guthaben und Status

## Schnellstart

```
Herunterladen → Starten → Erste Plattform hinzufügen → Proxy-Adresse konfigurieren → Loslegen
```

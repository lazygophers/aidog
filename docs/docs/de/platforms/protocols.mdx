# Unterstützte Plattformen & Schnittstellen

AiDog unterstützt eine breite Palette von AI-Plattformen und Schnittstellentypen und übernimmt automatisch die Protokollkonvertierung zwischen verschiedenen Plattformen.

## Unterstützte Plattformen

AiDog enthält 50+ eingebaute Plattform-Presets, die wichtige AI-Dienste abdecken:

| Kategorie | Beispiele |
|----------|----------|
| International | OpenAI, Anthropic, Google Gemini, Mistral |
| Chinesisch | DeepSeek, Qwen, Zhipu GLM, Kimi, MiniMax |
| Aggregatoren | OpenRouter, SiliconFlow, Novita |

Ein Plattform-Preset auswählen und die gesamte Konfiguration wird automatisch ausgefüllt — kein manuelles Auswählen des Schnittstellentyps nötig.

## Automatische Schnittstellenerkennung

AiDog erkennt den Schnittstellentyp automatisch aus Client-Anfragen — keine manuelle Konfiguration nötig:

- **Chat-Schnittstelle** — Die häufigsten AI-Chat-Anfragen
- **Coding-Assistant-Schnittstelle** — Genutzt von Tools wie Codex, Claude Code
- **Text-Completion-Schnittstelle** — Legacy-Text-Completion-Anfragen

Das bedeutet, Sie können verschiedene Client-Typen über dieselbe Proxy-Adresse bedienen.

## Automatische Protokollkonvertierung

Wenn Ihr Client Anfragen in einem Format sendet, die Zielplattform aber ein anderes nutzt, konvertiert AiDog automatisch. Zum Beispiel:

- Client sendet OpenAI-Format → Automatisch ins Anthropic-Format für Claude konvertiert
- Coding-Assistant nutzt eigene Schnittstelle → Automatisch ins Plattform-Format konvertiert

## Mock-Plattform

AiDog bietet eine Mock-Plattform, die AI-Antworten simuliert, ohne echte Anfragen zu senden. Sie können Antwortinhalt, Statuscodes, Verzögerungen u. v. m. anpassen.

# Guthaben & Kontingente

AiDog verfolgt Kontoguthaben und Coding-Plan-Kontingente plattformübergreifend und gibt Ihnen volle Ausgaben-Transparenz.

## Automatische Guthabenabfrage

AiDog fragt automatisch Guthaben für diese Plattformen ab:

| Plattform | Abgefragt |
|----------|---------------|
| OpenAI | Kontoguthaben |
| DeepSeek | Kontoguthaben |
| StepFun | Kontoguthaben |
| SiliconFlow | Kontoguthaben |
| OpenRouter | Guthaben / Nutzung |
| Novita | Kontoguthaben |
| Kimi (Moonshot) | Kontoguthaben |
| GLM (Zhipu) | Kontoguthaben |
| MiniMax | Kontoguthaben |

Guthabensdaten werden planmäßig automatisch aktualisiert. Sie können auch manuell auf „Aktualisieren" klicken.

## Coding-Plan-Kontingente

Einige Plattformen bieten Coding-Pläne an, z. B. OpenAI Plus/Team/Enterprise, Zhipu Coding Plan, Kimi Code Plan usw.

AiDog fragt und zeigt Kontingent-Informationen:

- Stufennamen
- Aktuelle Auslastung
- Verbleibender Prozentsatz

Plattform-Presets mit „Coding Plan"-Markierung konfigurieren Kontingentabfragen automatisch.

## NewAPI-Guthabenabfrage

Für Plattformen mit dem NewAPI-System (z. B. selbstgehostete API-Aggregatoren) unterstützt AiDog eine separate Guthabenabfrage-Konfiguration:

- **Abfrage-URL** — Separat von der API-URL
- **Abfrage-Token** — Aus dem Dashboard (anders als API-Key)
- **User-ID** — Optional

Tragen Sie diese Angaben im Abschnitt „Guthabenabfrage-Konfiguration" der Plattform ein.

## Manuelles Budget

Für Plattformen ohne automatische Guthabenabfrage bietet AiDog manuelle Budgetlimits.

### Einrichtung

Im Abschnitt „Manuelles Budget" der Plattform auf „Limit hinzufügen" klicken:

| Typ | Beschreibung |
|------|-------------|
| Gesamt | Gesamte Ausgabengrenze |
| Rollierendes Fenster | Ausgabengrenze im rollierenden Zeitfenster |
| Festes Fenster | Ausgabengrenze mit festem Zurücksetzungszeitplan |
| Täglich | Tägliche Ausgabengrenze |

### Budget-Einheiten

- **USD** — Nach Kosten gemessen
- **Tokens** — Nach Token-Anzahl gemessen

### Fensterdauer

Rollierendes und festes Fenster unterstützen mehrere Zeitgranularitäten: Minute, Stunde, Tag, Woche, Monat.

### Wenn das Budget erschöpft ist

Wenn ein Limit erreicht ist, stoppt AiDog die Weiterleitung an diese Plattform (gibt 402-Fehler) und löst automatisch Failover auf andere Plattformen der Gruppe aus. Die Weiterleitung wird beim Zurücksetzen des Fensters fortgesetzt.

## Wo Guthaben sichtbar sind

Guthaben-Infos werden an mehreren Stellen angezeigt:

- **Plattform-Karten** — Echtzeitguthaben und Kontingentnutzung pro Plattform
- **Gruppen-Karten** — Aggregiertes Guthaben der Gruppen-Plattformen
- **System-Tray** — Konfigurierbare Guthabenstatus-Anzeige

## Statistik-Seite

Auf der **„Statistiken"**-Seite Daten nach Plattform, Gruppe oder Modell anzeigen:

- Heutige Anfragezahl und Erfolgsquote
- Token-Nutzung (Eingabe / Ausgabe / Cache)
- Geschätzte Kosten
- Nutzungstrend-Diagramme (täglich / wöchentlich / monatlich)

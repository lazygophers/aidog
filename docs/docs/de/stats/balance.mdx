# Guthabenabfrage

AiDog kann Kontoguthaben und Coding Plan-Kontingente unterstützter Plattformen abfragen.

## Unterstützte Plattformen

| Plattform | Abfrageinhalt |
|----------|---------------|
| OpenAI | Kontoguthaben |
| DeepSeek | Kontoguthaben |
| StepFun | Kontoguthaben |
| SiliconFlow | Kontoguthaben |
| OpenRouter | Kontoguthaben / Nutzung |
| Novita | Kontoguthaben |
| Kimi | Kontoguthaben |
| GLM (Zhipu) | Kontoguthaben |
| MiniMax | Kontoguthaben |

## Guthaben einsehen

Guthabeninformationen werden angezeigt in:

1. **Plattformkarten** — Echtzeit-Guthaben pro Plattform
2. **Gruppenkarten** — Aggregiertes Guthaben der Gruppenplattformen
3. **Systemleiste** — Guthabenstatus-Anzeige

## Aktualisierung

Guthabendaten werden automatisch nach Zeitplan aktualisiert. Sie können auch manuell auf die Aktualisierungsschaltfläche klicken.

## Coding Plan

Einige Plattformen stellen Coding Plan-Kontingentinformationen bereit, die AiDog zusätzlich zu den Guthabendaten abfragt und anzeigt.

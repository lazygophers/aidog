# Pricing & Billing

AiDog automatically estimates the cost of each request based on the model pricing table. Manage all pricing data in **"Settings" → "Model Pricing"** tab.

## Cost Estimation

After each request, AiDog automatically:

1. Extracts token usage (input / output / cache) from the upstream response
2. Looks up the unit price in the pricing table
3. Calculates the estimated cost

Pricing fallback order: Custom pricing → Synced pricing → Fallback price

:::tip{title="Note"}
Estimated costs are for reference only. Actual charges are based on platform billing.
:::

## Pricing Management Page

### Price Sync

AiDog supports one-click sync of the latest model pricing data:

- Click "Sync Now" to fetch the latest prices
- Sync result shows: added count, updated count, failed count
- Enable "Auto Sync" to automatically fetch at a configured interval

Auto sync intervals: 1 hour / 6 hours / 12 hours / 24 hours / 7 days

### Fallback Price

For models not listed in the pricing table, a fallback price is used:

- **Input fallback** — Default $3.0 / million tokens
- **Output fallback** — Default $3.0 / million tokens

Adjustable in the sync settings section of the "Model Pricing" page.

### Browse & Search

The price table shows these columns:

| Column | Description |
|--------|-------------|
| Model | Model name |
| Source | LiteLLM (synced) / Manual |
| Default Platform | Default platform for this model |
| Input $/M | Input token price (per million tokens) |
| Output $/M | Output token price |
| Cache Read $/M | Cache token read price |

- Search by model name
- Filter by source (LiteLLM / Manual)
- Paginated with adjustable page size and jump-to-page

### Manage Pricing

- **Delete** — Remove a pricing record (click the delete button at end of row)
- Price changes take effect immediately for subsequent requests

## View Costs

View cost-related data on the **"Statistics"** page:

- "Estimated Cost" in the overview cards (with period comparison)
- "Estimated Cost" column in the dimension ranking table
- Cost details in trend chart tooltips

# Nutzungsstatistiken

Die **„Statistiken"**-Seite bietet Nutzungsüberblick, Trendanalyse und Dimensions-Rankings für alle Proxy-Anfragen.

## Zeitraum

Drei Zeit-Presets oben verfügbar:

| Preset | Bereich |
|--------|-------|
| Heute | Seit Mitternacht heute |
| Letzte 7 Tage | Vergangene 7 Tage |
| Letzte 30 Tage | Vergangene 30 Tage |

Alle Metriken und Diagramme aktualisieren sich automatisch bei Auswahl eines Presets.

## Übersichtskarten

Acht Kernmetriken werden oben angezeigt, jeweils mit Periodenvergleich:

| Metrik | Beschreibung |
|--------|-------------|
| Anfragen gesamt | Gesamtanzahl Anfragen im Zeitraum |
| Erfolgsquote | Anteil erfolgreicher Anfragen (farbcodiert) |
| Eingabe-Tokens | Gesamt Eingabe-Tokens |
| Ausgabe-Tokens | Gesamt Ausgabe-Tokens |
| Cache-Tokens | Gesamt Cache-Treffer-Tokens |
| Cache-Rate | Cache-Tokens als Anteil der Eingabe-Tokens |
| Ø Latenz | Durchschnittliche Antwortzeit in Millisekunden |
| Geschätzte Kosten | Geschätzte Kosten laut Preistabelle |

Vergleichspfeile: grün hoch = positive Veränderung (z. B. höhere Erfolgsquote), rot hoch = negative Veränderung (z. B. höhere Kosten).

## Filter

Neben den Zeit-Presets Daten mit diesen Filtern verfeinern:

- **Granularität**: Täglich / Stündlich
- **Gruppieren nach**: Plattform / Modell / Gruppe
- **Gruppe**: nach Gruppenname filtern (durchsuchbar)
- **Modell**: nach Modellname filtern (durchsuchbar)
- **Plattformtyp**: nach Plattformtyp filtern (durchsuchbar)

Daten laden automatisch neu bei Filteränderung.

## Trenddiagramm

Ein Balkendiagramm unter den Übersichtskarten zeigt Anfragevolumen-Trends:

- X-Achse gruppiert nach gewählter Granularität (Tag/Stunde)
- Balkenhöhe repräsentiert Anfragezahl
- Farbverlauf spiegelt Fehlerquote (mehr Rot = mehr Fehler)
- Hover für Tooltip: Anfragen, Erfolge, Fehler, Ø Latenz, geschätzte Kosten

## Dimensions-Ranking-Tabelle

Unter dem Diagramm eine Tabelle mit Ranking nach aktueller Dimension (Plattform/Modell/Gruppe):

| Spalte | Beschreibung |
|--------|-------------|
| Name | Plattform- / Modell- / Gruppenname |
| Anfragen | Gesamtanfragen für diese Dimension |
| Erfolg | Anzahl erfolgreicher Anfragen (farbcodiert für Gesundheit) |
| Eingabe | Eingabe-Token-Anzahl |
| Ausgabe | Ausgabe-Token-Anzahl |
| Cache | Cache-Token-Anzahl |
| Ø Latenz | Durchschnittliche Antwortzeit |
| Geschätzte Kosten | Geschätzte Kosten (farbcodiert für Ausgabenniveau) |

### Sortierung & Paginierung

- Auf einen Spaltenkopf klicken zum Sortieren (erneut klicken für auf-/absteigend)
- 50 Zeilen pro Seite mit Paginierung unten

## Echtzeit-Aktualisierung

Statistiken aktualisieren sich automatisch bei neuen Anfragen — kein manueller Eingriff nötig.

:::tip{title="Hinweis"}
Geschätzte Kosten werden aus der Preistabelle berechnet und dienen nur als Referenz. Tatsätzliche Abrechnung basiert auf der Plattformabrechnung.
:::

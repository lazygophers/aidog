# App-Einstellungen

Konfigurieren Sie AiDogs globales Verhalten auf der **„Einstellungen"**-Seite. Sie enthält 5 Tabs: Proxy, Claude Code, Codex, Modellpreise und System-Tray.

![AiDog — Einstellungen](/screenshots/settings.png)

## Proxy

### Proxy-Einstellungen

| Einstellung | Standard | Beschreibung |
|---------|---------|-------------|
| Proxy-Port | `9876` | HTTP-Proxy-Listen-Port |
| Anfrage-Timeout | `300s` | Upstream-Anfrage-Timeout |
| Autostart | An | Proxy automatisch starten, wenn AiDog startet |

### Log-Einstellungen

Granularität und Aufbewahrung steuern. Siehe [Log-Einstellungen](/de/logs/log-settings).

### Datenverwaltung

- Plattform-Config exportieren
- Gruppen-Config exportieren
- Statistiken exportieren
- Abgelaufene Logs bereinigen
- Statistiken zurücksetzen

## Claude Code

Claude-Code-Einstellungen inklusive Modell, Berechtigungen, Sandbox, Plugins u. v. m. bearbeiten.

- **Aus Claude Code importieren** — Bestehende Konfiguration in AiDog einlesen
- **Nach Claude Code exportieren** — Proxy-Adresse in Claude-Code-Config-Datei schreiben
- **Gruppeneinstellungen synchronisieren** — Unabhängige Config-Dateien pro Gruppe erzeugen

Siehe [Codex-Integration](/de/proxy/codex-integration).

## Codex

Codex-Einstellungen direkt in AiDog bearbeiten, ohne Config-Dateien manuell zu suchen.

Siehe [Codex-Integration](/de/proxy/codex-integration).

## Modellpreise

Alle Modellpreisdaten durchsuchen, suchen und verwalten. Siehe [Preise & Abrechnung](/de/stats/pricing).

## System-Tray

Tray-Anzeigeinhalt und -format konfigurieren. Siehe [Tray-Konfiguration](/de/settings/tray).

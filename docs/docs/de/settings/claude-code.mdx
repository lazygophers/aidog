# Claude Code

Konfigurieren Sie die AiDog-Integration mit Claude Code im Tab **„Einstellungen" → „Claude Code"**.

## Konfigurationsbearbeitung

Alle Claude-Code-Einstellungen direkt in AiDog bearbeiten:

- **Modell** — Gibt das von Claude Code genutzte Modell an
- **Berechtigungen** — Datei-Lese-/Schreib-, Netzwerkzugriff-Steuerung
- **Sandbox** — Code-Ausführungs-Sandbox-Konfiguration
- **Plugins** — Claude-Code-Plugin-Verwaltung
- **Hooks** — Lifecycle-Hooks (PreToolUse / PostToolUse usw.)
- **StatusLine** — Konfiguration der Statusleisten-Anzeige

Alle Änderungen werden sofort in `~/.claude.json` gespeichert.

## Aus Claude Code importieren

Auf „Aus Claude Code importieren" klicken, um die bestehende Claude-Code-Konfigurationsdatei zu lesen und in den Editor zu laden. Ideal, um bestehende Configs bei der Ersteinrichtung zu migrieren.

## Nach Claude Code exportieren

Auf „Nach Claude Code exportieren" klicken, um die Proxy-Adresse in Claude-Codes Konfigurationsdatei zu schreiben:

```json
{
  "apiBaseUrl": "http://127.0.0.1:9876"
}
```

Nach dem Export werden alle Claude-Code-Anfragen über AiDog proxied.

## Gruppeneinstellungen synchronisieren

Auf „Gruppeneinstellungen synchronisieren" klicken, um unabhängige Claude-Code-Konfigurationsdateien für jede Gruppe zu erzeugen:

- Gespeichert unter `~/.aidog/settings.<gruppen-name>.json`
- Jede Config-Datei nutzt den Proxy-Pfad der entsprechenden Gruppe
- Zusammen mit der „Claude"-Kopier-Schaltfläche auf Gruppenkarten verwenden

### Verwendung

1. Auf die „Claude"-Schaltfläche einer Gruppenkarte klicken, um den Startbefehl zu kopieren
2. Im Terminal einfügen:

```bash
claude --brief --dangerously-skip-permissions \
  --settings ~/.aidog/settings.<gruppen-name>.json
```

## Manuelle Konfiguration

Alternativ über Umgebungsvariablen konfigurieren:

```bash
export ANTHROPIC_BASE_URL="http://127.0.0.1:9876"
export ANTHROPIC_API_KEY="any-value"
```

## Hinweise

- Claude Code nutzt das native Anthropic-Protokoll
- Stellen Sie sicher, dass die Zielplattform das Anthropic-Protokoll unterstützt oder eine Protokollkonvertierung konfiguriert ist
- AiDog stellt einen dedizierten Pfad für Claude Code bereit
- Falls nötig, Model-Mapping in Plattform- oder Gruppeneinstellungen hinzufügen

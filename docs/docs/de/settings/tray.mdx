# Tray-Konfiguration

AiDog läuft im System-Tray und zeigt wichtige Statusinformationen in der Menüleiste/Taskleiste. Anzeige-Items und Stile unter **„Einstellungen" → „System-Tray"** konfigurieren.

## Tray-Verhalten

Nach dem Schließen des Hauptfensters läuft AiDog im System-Tray weiter, um den Proxy-Dienst verfügbar zu halten. Rechtsklick auf das Tray-Icon, um das Hauptfenster ein-/auszublenden, den Proxy zu starten/stoppen oder die App zu beenden.

## Live-Vorschau

Oben auf der Konfigurationsseite wird eine simulierte Menüleisten-Vorschau gezeigt. Alle Änderungen werden sofort reflektiert. Die Vorschau zeigt auch die aktuelle Zeilennutzung (max. 2 Zeilen) und warnt bei Überschreitung.

## Anzeige-Item-Typen

Auf „Anzeige-Item hinzufügen" klicken, um einen dieser Inhaltstypen hinzuzufügen:

### Plattform-Guthaben/Kontingent

Eine aktivierte Plattform auswählen, um Guthaben oder Coding-Plan-Kontingent zu zeigen:

| Anzeigemodus | Beschreibung |
|--------------|-------------|
| Guthaben | Kontoguthaben (z. B. `$12.50`) |
| Kontingent | Verbleibender Coding-Plan-Prozentsatz (z. B. `85%`) |

### Heutige Statistiken

Heutige aggregierte Daten anzeigen, mit 4 verfügbaren Metriken:

| Metrik | Beschreibung |
|--------|-------------|
| Tokens | Heutige Gesamt-Token-Nutzung |
| Cache% | Heutige Cache-Trefferrate |
| Cost$ | Heutige geschätzte Kosten |
| Requests | Heutige Gesamt-Anfragezahl |

### Trennzeichen

Trennzeichen zwischen Anzeige-Items einfügen. Sechs Preset-Stile (`|`, `·`, `—`, `/`, `»`, Leerzeichen) plus eigene Zeichen.

## Item-Anpassung

Jedes Anzeige-Item aufklappen, um diese Eigenschaften anzupassen:

| Eigenschaft | Beschreibung |
|----------|-------------|
| Anzeige | Guthaben / Kontingent (nur Plattform-Items) |
| Metrik | Tokens / Cache% / Cost$ / Requests (nur heutige Statistiken) |
| Label | Eigener Anzeigename (leer für Standard) |
| Dezimalen | Anzahl Dezimalstellen für Werte |
| Zeilenmodus | Einzeilig (Label + Wert gleiche Zeile) / Zweizeilig (Label oben, Wert unten) |
| Ausrichtung | Links / Mitte / Rechts |
| Wertzeilen-Ausrichtung | Unabhängige Ausrichtung für die zweite Zeile im Zweizeilenmodus |
| Farbe | System folgen / Preset (rot/grün/orange) / Benutzerdefiniert |

### Farbwarnung

Bei Wahl einer benutzerdefinierten Farbe warnt AiDog, wenn die Farbe unter bestimmten Menüleisten-Themen möglicherweise schlecht sichtbar ist.

## Drag zum Sortieren

Anzeige-Items unterstützen Drag-and-Drop-Sortierung. Deaktivierte Items erscheinen nicht in der Vorschau.

## Autostart beim Boot

AiDog unter **„Einstellungen" → „Proxy-Konfiguration"** so konfigurieren, dass es mit dem System startet.

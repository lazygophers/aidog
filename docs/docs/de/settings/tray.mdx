# Systemleisten-Konfiguration

AiDog unterstützt die Persistenz in der Systemleiste, läuft im Hintergrund und zeigt wichtige Statusinformationen an.

## Systemleiste

Das Schließen des Hauptfensters hält AiDog in der Systemleiste am Laufen und erhält die Proxy-Verfügbarkeit.

### Leisteninformationen

Das Leisten-Symbol kann anzeigen:
- Proxy-Dienststatus (läuft/gestoppt)
| Kontoguthaben
| Tagesnutzungsübersicht

### Leistenmenü

Rechtsklick auf das Leisten-Symbol zum:
- Hauptfenster anzeigen/verbergen
- Proxy starten/stoppen
- Anwendung beenden

## Leistenanpassung

Einstellungen konfigurieren unter **„Einstellungen"** → „Systemleiste":

### Anzeigeinhalt

Wählen Sie, welche Informationen in der Leiste angezeigt werden:
- Guthabeninformationen
- Heutige Anfrageanzahl
- Heutige Token-Anzahl
- Heutige geschätzte Kosten

### Statuszeilenformat

Statuszeilen-Anzeigeformat mit benutzerdefinierten Trennzeichen und Anzeigereihenfolge konfigurieren.

## Autostart

AiDog so konfigurieren, dass es automatisch mit dem System startet (minimiert in die Systemleiste).

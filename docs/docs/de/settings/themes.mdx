# Themes

AiDog bietet mehrere integrierte Themes, jedes mit Light- und Dark-Modus.

## Integrierte Themes

| Theme | Stil |
|-------|-------|
| Liquid Glass | Glasmorphismus, Transparenz |
| Nord | Arktische Blautöne, ruhig |
| Dracula | Dunkles Theme, beliebte Palette |
| Catppuccin | Warme Farben, augenfreundlich |
| Solarized | Klassische wissenschaftliche Palette |

## Theme wechseln

Wählen Sie ein Theme unter **„Einstellungen"** → „Themes".

### Light/Dark-Modus

Jedes Theme unterstützt Light- und Dark-Modi, folgt den Systemeinstellungen oder kann manuell umgeschaltet werden.

## Theme-Architektur

Themes werden über ein CSS-Variablen-System implementiert:
- Jedes Theme bietet Light + Dark CSS-Variablen-Sets
- Variablen-Dateien befinden sich im Verzeichnis `src/themes/`
| Theme-Wechsel ersetzt das CSS-Variablen-Set

### Benutzerdefinierte Themes

So erstellen Sie ein benutzerdefiniertes Theme:

1. Eine bestehende Theme-Datei als Vorlage kopieren
2. CSS-Variablenwerte ändern
3. Neues Theme in `src/themes/index.ts` registrieren
4. Anwendung neu erstellen

# Themes

AiDog bietet mehrere integrierte Themes, jeweils mit Hell- und Dunkelmodus.

## Integrierte Themes

| Theme | Stil |
|-------|-------|
| Liquid Glass | Glassmorphism, transparent |
| Nord | Arktische Blautöne, ruhig |
| Dracula | Dunkles Theme, beliebte Palette |
| Catppuccin | Warme Farben, augenschonend |
| Solarized | Klassische wissenschaftliche Palette |

## Theme wechseln

Wählen Sie ein Theme im Theme-Selektor der **„Einstellungen"**-Seite.

### Hell-/Dunkelmodus

Jedes Theme unterstützt Hell- und Dunkelmodus. Folgt automatisch den Systemeinstellungen oder manuell umschaltbar.

## Theme-Funktionen

- **Abgerundete Ecken** — Gestaffeltes Border-Radius-System (sm / md / lg / xl)
- **Mattglas** — Halbtransparente Panels mit unscharfem Hintergrund
- **Sanfte Übergänge** — Theme-Wechsel mit Easing-Animationen
- **Hoher Kontrast** — Gewährleistet gute Lesbarkeit in Hell und Dunkel

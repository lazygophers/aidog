# Gruppen erstellen & verwalten

Gruppen sind AiDogs Routing-Kern — organisieren Sie Plattformen nach Zweck und steuern Sie das Routing per Regeln.

## Gruppe erstellen

1. Gehen Sie zur **„Gruppen"**-Seite
2. Klicken Sie **„+ Gruppe hinzufügen"**
3. Name und Pfad eingeben
4. Plattformen auswählen
5. Routing-Modus wählen

:::tip{title="Auto-Gruppe"}
Beim Hinzufügen einer Plattform erzeugt AiDog automatisch eine entsprechende Gruppe. Weitere Gruppen können manuell erstellt werden.
:::

## Routing-Modi

### Failover

Standardmodus. Plattformen werden der Reihe nach genutzt. Bei Fehler wird automatisch die nächste versucht. Ideal für Primär/Backup-Setups.

```
Anfrage → Plattform A (primär) → Erfolg → Rückgabe
                ↓ (fehlgeschlagen)
         Plattform B (Backup) → Erfolg → Rückgabe
```

### Load Balance

Anfragen werden gewichtet verteilt. Ideal für Multi-Plattform-Traffic-Splitting.

| Plattform | Gewicht | Erwarteter Traffic |
|----------|--------|-----------------|
| OpenAI | 70 | ~70% |
| DeepSeek | 30 | ~30% |

## Timeout-Einstellungen

Jede Gruppe kann unabhängig Timeouts setzen und globale Einstellungen überschreiben:

- **Anfrage-Timeout** — Max. Wartezeit auf Upstream-Antwort
- **Verbindungs-Timeout** — Max. Zeit zum Verbindungsaufbau

Leer lassen für globale Timeouts.

## Plattformverwaltung

In einer Gruppe:

- **Drag & Drop** — Priorität anpassen (bestimmt Primär/Backup-Reihenfolge im Failover)
- **Aktivieren/Deaktivieren** — Plattform temporär überspringen
- **Gewichte setzen** — Anfrage-Verhältnis für Load Balance
- **Modell-Mapping** — Gruppen-Mapping, Vorrang vor Plattform

## Ein-Klick-Startbefehle

Jede Gruppenkarte bietet zwei Ein-Klick-Kopierbuttons:

- **Claude** — Kopiert den Claude-Code-Startbefehl für diese Gruppe inkl. Einstellungsdateipfad
- **Codex** — Kopiert den Codex-Startbefehl inkl. Profil

Im Terminal einfügen, um das Gruppen-Routing sofort zu nutzen.

## Gesundheitsstatus

Gruppenkarten zeigen Statusindikatoren für verknüpfte Plattformen:

- 🟢 Alle erfolgreich — Grün
- 🟡 Teilweise fehlgeschlagen — Gelb
- 🔴 Alle fehlgeschlagen — Rot

## Bearbeiten & löschen

- Karte klicken für Edit-Modus; nach Änderungen «Speichern»
- Löschen entfernt nur die Verknüpfung, nicht die Plattformen
- Karten per Drag in der Liste umsortieren

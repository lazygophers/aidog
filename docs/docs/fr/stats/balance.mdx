# Consultation du solde

AiDog peut consulter les soldes de compte et les quotas Coding Plan des plateformes supportées.

## Plateformes supportées

| Plateforme | Contenu de la requête |
|----------|---------------|
| OpenAI | Solde du compte |
| DeepSeek | Solde du compte |
| StepFun | Solde du compte |
| SiliconFlow | Solde du compte |
| OpenRouter | Solde du compte / utilisation |
| Novita | Solde du compte |
| Kimi | Solde du compte |
| GLM (Zhipu) | Solde du compte |
| MiniMax | Solde du compte |

## Consultation des soldes

Les informations de solde sont affichées dans :

1. **Cartes de plateforme** — Solde en temps réel par plateforme
2. **Cartes de groupe** — Solde agrégé des plateformes du groupe
3. **Barre d'état système** — Affichage du statut du solde

## Actualisation

Les données de solde s'actualisent automatiquement selon un planning. Vous pouvez aussi cliquer manuellement sur le bouton d'actualisation.

## Coding Plan

Certaines plateformes fournissent des informations de quota Coding Plan, qu'AiDog consulte et affiche alongside les données de solde.

# Solde & Quotas

AiDog suit les soldes de compte et quotas de coding plan entre plateformes, vous donnant une visibilité complète de vos dépenses.

## Requête automatique du solde

AiDog interroge automatiquement les soldes pour ces plateformes :

| Plateforme | Contenu interrogé |
|----------|---------------|
| OpenAI | Solde du compte |
| DeepSeek | Solde du compte |
| StepFun | Solde du compte |
| SiliconFlow | Solde du compte |
| OpenRouter | Solde / utilisation |
| Novita | Solde du compte |
| Kimi (Moonshot) | Solde du compte |
| GLM (Zhipu) | Solde du compte |
| MiniMax | Solde du compte |

Le solde se rafraîchit automatiquement selon une planification. Vous pouvez aussi cliquer manuellement sur le bouton de rafraîchissement.

## Quotas Coding Plan

Certaines plateformes offrent des coding plans, comme OpenAI Plus/Team/Enterprise, Zhipu Coding Plan, Kimi Code Plan, etc.

AiDog interroge et affiche les informations de quota :

- Noms des paliers
- Taux d'utilisation courant
- Pourcentage restant

Les presets de plateforme marqués « Coding Plan » configurent automatiquement les requêtes de quota.

## Requête de solde NewAPI

Pour les plateformes utilisant le système NewAPI (ex. agrégateurs API auto-hébergés), AiDog supporte une configuration séparée de requête de solde :

- **URL de requête** — Séparée de l'URL API
- **Token de requête** — Obtenu depuis le tableau de bord (différent de la clé API)
- **User ID** — Optionnel

Renseignez ces détails dans la section « Configuration de requête de solde » de la plateforme.

## Budget manuel

Pour les plateformes ne supportant pas la requête automatique de solde, AiDog fournit des limites de budget manuel.

### Configuration

Dans la section « Budget manuel » de la plateforme, cliquez « Ajouter une limite » :

| Type | Description |
|------|-------------|
| Total | Limite de dépense globale |
| Fenêtre glissante | Limite sur une fenêtre temporelle glissante |
| Fenêtre fixe | Limite réinitialisée selon un calendrier fixe |
| Quotidienne | Limite de dépense journalière |

### Unités de budget

- **USD** — Mesuré par coût
- **Tokens** — Mesuré par nombre de tokens

### Durée de fenêtre

Fenêtre glissante et fixe supportent plusieurs granularités temporelles : minute, heure, jour, semaine, mois.

### Quand le budget est épuisé

Quand une limite est atteinte, AiDog arrête de transférer les requêtes à cette plateforme (renvoie erreur 402) et déclenche automatiquement le failover vers d'autres plateformes du groupe. Le transfert reprend quand la fenêtre se réinitialise.

## Où voir les soldes

Le solde s'affiche à plusieurs endroits :

- **Cartes de plateforme** — Solde en temps réel et utilisation de quota par plateforme
- **Cartes de groupe** — Solde agrégé des plateformes du groupe
- **Barre des tâches système** — Affichage configurable du statut de solde

## Page Statistiques

Sur la page **« Statistiques »**, consultez les données par plateforme, groupe ou modèle :

- Nombre de requêtes et taux de succès du jour
- Utilisation de tokens (entrée / sortie / cache)
- Coût estimé
- Graphiques de tendance (journalier / hebdomadaire / mensuel)

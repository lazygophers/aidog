# Tarifs & Facturation

AiDog estime automatiquement le coût de chaque requête selon la table de tarifs des modèles. Gérez toutes les données tarifaires dans l'onglet **« Réglages » → « Tarifs des modèles »**.

## Estimation du coût

Après chaque requête, AiDog automatiquement :

1. Extrait l'utilisation de tokens (entrée / sortie / cache) de la réponse amont
2. Recherche le prix unitaire dans la table tarifaire
3. Calcule le coût estimé

Ordre de repli tarifaire : Tarif personnalisé → Tarif synchronisé → Prix de repli

:::tip{title="Note"}
Les coûts estimés sont fournis à titre indicatif. La facturation réelle se base sur la facturation de la plateforme.
:::

## Page de gestion des tarifs

### Synchronisation des prix

AiDog supporte la synchronisation en un clic des dernières données tarifaires :

- Cliquez « Synchroniser maintenant » pour récupérer les derniers prix
- Le résultat affiche : ajoutés, mis à jour, échoués
- Activez « Synchronisation auto » pour récupérer à intervalle configuré

Intervalles auto : 1 heure / 6 heures / 12 heures / 24 heures / 7 jours

### Prix de repli

Pour les modèles absents de la table, un prix de repli s'applique :

- **Repli entrée** — Défaut $3.0 / million de tokens
- **Repli sortie** — Défaut $3.0 / million de tokens

Ajustable dans la section synchronisation de la page « Tarifs des modèles ».

### Parcourir & Rechercher

La table affiche ces colonnes :

| Colonne | Description |
|--------|-------------|
| Modèle | Nom du modèle |
| Source | LiteLLM (synchronisé) / Manuel |
| Plateforme par défaut | Plateforme par défaut pour ce modèle |
| Entrée $/M | Prix token entrée (par million) |
| Sortie $/M | Prix token sortie |
| Lecture cache $/M | Prix lecture token cache |

- Recherche par nom de modèle
- Filtre par source (LiteLLM / Manuel)
- Pagination avec taille de page ajustable et saut de page

### Gérer les tarifs

- **Supprimer** — Retirer un enregistrement (bouton en fin de ligne)
- Les changements de prix prennent effet immédiatement pour les requêtes suivantes

## Voir les coûts

Consultez les données de coût sur la page **« Statistiques »** :

- « Coût estimé » dans les cartes d'aperçu (avec comparaison de période)
- Colonne « Coût estimé » dans le tableau de classement par dimension
- Détails de coût dans les infobulles du graphique de tendance

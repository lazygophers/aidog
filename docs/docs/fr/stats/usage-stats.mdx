# Usage Statistics

AiDog automatically tracks usage data for all proxied requests.

## Statistics Page

Go to the **"Statistics"** page for a usage overview.

### Today's Stats

- Total requests
- Success/failure rate
- Total token usage (Prompt + Completion)
- Estimated cost

### Trends

View by day/week/month:
- Request count trends
- Token usage trends
- Cost trends

### Group Statistics

Group card statistics are aggregated from their associated platforms.

## Dimensions

View statistics by:
- **By platform** — Per-platform usage
- **By group** — Per-group usage
- **By model** — Per-model usage
- **By time** — Daily/weekly/monthly aggregation

## Data Source

- Token usage: Extracted directly from upstream API responses
- Estimated cost: Calculated automatically based on model pricing tables
- Request count: Aggregated from proxy log records

:::tip{title="Note"}
Estimated costs are based on built-in pricing tables and are for reference only. Actual costs are based on platform billing.
:::

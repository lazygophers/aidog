# Créer & gérer les groupes

Les groupes sont le cœur du routage d'AiDog — organisez les plateformes par usage et contrôlez le routage avec des règles.

## Créer un groupe

1. Allez sur la page **« Groupes »**
2. Cliquez sur **« + Ajouter un groupe »**
3. Saisissez un nom et un chemin
4. Sélectionnez les plateformes à inclure
5. Choisissez un mode de routage

:::tip{title="Groupe auto"}
À l'ajout d'une plateforme, AiDog crée automatiquement un groupe correspondant. Vous pouvez aussi créer des groupes manuellement.
:::

## Modes de routage

### Failover

Mode par défaut. Les plateformes sont utilisées dans l'ordre. En cas d'échec, la suivante est essayée automatiquement. Idéal pour les configurations primaire/secours.

```
Requête → Plateforme A (primaire) → Succès → Retour
                ↓ (échec)
         Plateforme B (secours) → Succès → Retour
```

### Load Balance

Les requêtes sont réparties entre plateformes selon un poids. Idéal pour répartir le trafic multi-plateformes.

| Plateforme | Poids | Trafic attendu |
|----------|--------|-----------------|
| OpenAI | 70 | ~70% |
| DeepSeek | 30 | ~30% |

## Réglages de timeout

Chaque groupe peut définir indépendamment ses timeouts, surchargeant les réglages globaux :

- **Timeout de requête** — Temps max d'attente de réponse amont
- **Timeout de connexion** — Temps max pour établir la connexion

Laissez vide pour utiliser les timeouts globaux.

## Gestion des plateformes

Dans un groupe, vous pouvez :

- **Glisser pour réordonner** — Ajuster la priorité (détermine l'ordre primaire/secours en mode failover)
- **Activer/Désactiver** — Sauter temporairement une plateforme
- **Définir les poids** — Ratio de requêtes en mode load balance
- **Mapping de modèle** — Mapping au niveau groupe, prioritaire sur la plateforme

## Commandes de lancement en un clic

Chaque carte de groupe fournit deux boutons de copie en un clic :

- **Claude** — Copie la commande de lancement Claude Code pour ce groupe, avec le chemin du fichier de réglages
- **Codex** — Copie la commande de lancement Codex pour ce groupe, avec le profil

Collez la commande dans votre terminal pour utiliser immédiatement le routage du groupe.

## État de santé

Les cartes de groupe affichent des indicateurs d'état pour leurs plateformes associées :

- 🟢 Tous succès — Vert
- 🟡 Quelques échecs — Jaune
- 🔴 Tous échoués — Rouge

## Édition & suppression

- Cliquez une carte de groupe pour entrer en mode édition. Cliquez « Enregistrer » après modifications
- Supprimer un groupe ne supprime pas ses plateformes — seule l'association est retirée
- Glissez les cartes de groupe dans la liste pour réordonner

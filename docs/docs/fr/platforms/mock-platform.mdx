# Plateforme Mock

Les plateformes mock n'envoient pas de vraies requêtes IA — elles renvoient des réponses préconfigurées. Idéal pour le développement, le débogage, le test des règles de routage et la vérification des intégrations client.

## Créer une plateforme mock

1. Allez sur la page **« Plateformes »**
2. Cliquez sur le bouton **« + »**
3. Sélectionnez **« Mock »** dans la liste des types de plateforme
4. Aucune API Key ou Base URL requise
5. Enregistrer

Après sauvegarde, AiDog crée automatiquement un groupe correspondant. Pointez votre client sur le chemin de ce groupe pour utiliser le mock.

## Configuration de réponse

Personnalisez les paramètres de réponse :

| Paramètre | Description | Défaut |
|-----------|-------------|---------|
| Texte de réponse | Texte de réponse simulé | `Hello from mock` |
| Code de statut | Code de statut HTTP | `200` |
| Délai de réponse | Délai simulé en millisecondes | `0` |
| Tokens d'entrée | Nombre simulé de tokens d'entrée | `10` |
| Tokens de sortie | Nombre simulé de tokens de sortie | `20` |
| Tokens de cache | Tokens de cache simulés | `0` |
| Raison de fin | Raison de fin simulée (ex. stop) | `stop` |
| Streaming | Retourner en flux (suivre la requête, ou forcer on/off) | Suivre la requête |

## Simulation d'erreurs

Supporte la simulation de divers scénarios d'erreur :

| Mode d'erreur | Description |
|------------|-------------|
| Aucune erreur | Renvoie une réponse normale |
| Erreur HTTP | Renvoie un code d'erreur HTTP spécifié |
| Rate Limit (429) | Simule une limitation de débit |
| Timeout | Simule un timeout de requête |

Utile pour tester la gestion d'erreurs du client ou le comportement de basculement d'AiDog.

## Cas d'usage

- **Développement** — Tester l'intégration client sans vraies clés API
- **Tests de routage** — Vérifier les règles de correspondance de groupe et de mapping
- **Test de basculement** — Simuler des erreurs pour observer la commutation automatique
- **Test de performance** — Configurer des délais pour tester le comportement sous réponses lentes

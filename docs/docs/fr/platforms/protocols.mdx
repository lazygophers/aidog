# Plateformes & interfaces supportées

AiDog supporte un large éventail de plateformes AI et types d'interface, gérant automatiquement la conversion de protocole entre plateformes.

## Plateformes supportées

AiDog inclut 50+ presets de plateforme intégrés couvrant les principaux services AI :

| Catégorie | Exemples |
|----------|----------|
| Internationales | OpenAI, Anthropic, Google Gemini, Mistral |
| Chinoises | DeepSeek, Qwen, Zhipu GLM, Kimi, MiniMax |
| Agrégateurs | OpenRouter, SiliconFlow, Novita |

Sélectionner un preset de plateforme et toute la configuration est auto-remplie — pas besoin de choisir manuellement un type d'interface.

## Détection automatique d'interface

AiDog détecte automatiquement le type d'interface depuis les requêtes client — aucune configuration manuelle nécessaire :

- **Interface chat** — Les requêtes de chat AI les plus courantes
- **Interface assistant de codage** — Utilisée par des outils comme Codex, Claude Code
- **Interface de complétion de texte** — Requêtes de complétion de texte héritées

Cela signifie que vous pouvez servir différents types de clients via la même adresse proxy.

## Conversion automatique de protocole

Si votre client envoie des requêtes dans un format mais que la plateforme cible en utilise un autre, AiDog convertit automatiquement. Par exemple :

- Le client envoie le format OpenAI → Automatiquement converti au format Anthropic pour Claude
- L'assistant de codage utilise sa propre interface → Automatiquement converti au format de la plateforme

## Plateforme mock

AiDog fournit une plateforme mock qui simule des réponses AI sans envoyer de vraies requêtes. Vous pouvez personnaliser le contenu des réponses, codes de statut, délais, et plus.

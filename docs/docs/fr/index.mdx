## Prise en main en trois étapes

1. **Ajouter une plateforme** — saisissez la clé d'API et le point de terminaison d'une plateforme d'IA
2. **Définir l'adresse du proxy** — pointez votre client vers le proxy local d'AiDog
3. **Commencer à utiliser** — les requêtes sont automatiquement routées, mesurées et enregistrées

→ [Guide d'installation](/fr/getting-started/installation) · [Démarrage rapide](/fr/getting-started/quick-start)

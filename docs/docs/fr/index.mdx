# AiDog

AiDog est une application de bureau multiplateforme (macOS / Windows / Linux) qui agit comme un **proxy de passerelle API IA**, vous aidant à gérer les clés API de plusieurs plateformes IA, à router intelligemment les requêtes et à suivre l'utilisation et les coûts.

## Fonctionnalités principales

- **Agrégation multiplateforme** — Gérez les API d'OpenAI, Anthropic, Gemini, DeepSeek, Qwen et plus de 50 plateformes
- **Routage intelligent** — Sélection automatique des plateformes via groupes et règles de mappage de modèles
- **Analytique d'utilisation** — Suivi en temps réel des tokens, estimation des coûts, consultation des soldes
- **Système de journaux** — 3 niveaux de granularité d'enregistrement, politiques de rétention configurables
- **Intégration Codex** — Support natif d'OpenAI Codex / Claude Code et autres assistants de codage
- **Multi-protocole** — Protocoles natifs OpenAI Chat/Completions/Responses, Anthropic, Gemini
- **Personnalisation des thèmes** — Liquid Glass, Nord, Dracula et plus
- **Barre d'état système** — Affichage persistant du solde et du statut

## Démarrage rapide

```
Télécharger → Lancer → Ajouter la première plateforme → Configurer l'adresse proxy → Commencer
```

→ [Guide d'installation](/getting-started/installation)

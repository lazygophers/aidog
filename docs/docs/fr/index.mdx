## Trois étapes pour démarrer

1. **Ajouter une plateforme** —— saisissez la clé API et l'endpoint d'une plateforme AI
2. **Définir l'adresse proxy** —— pointez votre client vers le proxy local AiDog
3. **Commencer à l'utiliser** —— les requêtes sont automatiquement routées, mesurées et journalisées

→ [Guide d'installation](/fr/getting-started/installation) · [Démarrage rapide](/fr/getting-started/quick-start)

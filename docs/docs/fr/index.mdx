# AiDog

AiDog is a cross-platform desktop app (macOS / Windows / Linux) that acts as an **AI API gateway proxy**, helping you manage API keys for multiple AI platforms, route requests intelligently, and track usage and costs.

## Key Features

- **Multi-Platform Aggregation** — Manage APIs for 50+ platforms including OpenAI, Anthropic, Gemini, DeepSeek, Qwen, and more
- **Smart Routing** — Automatically select the best platform by group and model mapping rules, with failover and load balancing
- **Usage Analytics** — Track token usage in real time, estimate costs, query balances and Coding Plan quotas
- **Logging System** — Three-level recording granularity, configurable retention policies, full request chain tracing
- **Model Testing** — One-click connectivity testing with quick, batch, and custom modes
- **Codex Integration** — Built-in Codex config editor, one-click Claude Code config export
- **Multi-Protocol Support** — OpenAI Chat/Completions/Responses, Anthropic, Gemini native protocols with automatic inbound/outbound conversion
- **Theme Customization** — Liquid Glass, Nord, Dracula, and more themes, each with light/dark mode
- **System Tray** — Persistent tray display of balance, usage, and status with customizable content
- **Pricing Management** — Built-in model pricing table, auto-sync, custom pricing support

## Quick Start

Download the latest release → Launch → Add your first platform → Set proxy address → Start using

## App Overview

The AiDog main interface contains these pages:

| Page | Function |
|------|----------|
| **Platforms** | Manage all AI platform connections, endpoints, models, and balances |
| **Groups** | Organize platforms by use case, configure routing strategies and model mapping |
| **Statistics** | Usage stats, cost estimation, balance query, pricing management |
| **Logs** | View complete request chain details for every proxied request |
| **Settings** | Proxy config, Claude Code settings, Codex settings, model pricing, system tray |

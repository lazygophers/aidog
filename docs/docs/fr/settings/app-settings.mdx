# Réglages de l'app

Configurez le comportement global d'AiDog dans la page **« Réglages »**. Elle contient 5 onglets : Proxy, Claude Code, Codex, Tarifs des modèles, et Barre des tâches.

![AiDog — Réglages](/screenshots/settings.png)

## Proxy

### Réglages proxy

| Réglage | Défaut | Description |
|---------|---------|-------------|
| Port proxy | `9876` | Port d'écoute du proxy HTTP |
| Délai requête | `300s` | Délai requête amont |
| Démarrage auto | On | Démarrer automatiquement le proxy au lancement d'AiDog |

### Réglages des logs

Contrôlez la granularité et rétention des logs. Voir [Réglages des logs](/fr/logs/log-settings).

### Gestion des données

- Exporter la config plateforme
- Exporter la config groupe
- Exporter les statistiques
- Nettoyer les logs expirés
- Réinitialiser les statistiques

## Claude Code

Éditez les réglages Claude Code incluant modèle, permissions, sandbox, plugins, et plus.

- **Importer depuis Claude Code** — Lit la configuration existante dans AiDog
- **Exporter vers Claude Code** — Écrit l'adresse proxy dans le fichier de config Claude Code
- **Synchroniser les réglages de groupe** — Génère des fichiers de config indépendants pour chaque groupe

Voir [Intégration Codex](/fr/proxy/codex-integration).

## Codex

Éditez les réglages Codex directement dans AiDog, sans chercher manuellement les fichiers de config.

Voir [Intégration Codex](/fr/proxy/codex-integration).

## Tarifs des modèles

Parcourez, recherchez et gérez toutes les données tarifaires des modèles. Voir [Tarifs & Facturation](/fr/stats/pricing).

Configurez le contenu et le format d'affichage de la barre des tâches. Voir [Configuration de la barre des tâches](/fr/settings/tray).

## Sécurité des données

Toutes les données sont stockées localement. Elles ne quittent jamais votre machine et ne sont jamais téléversées sur un serveur.

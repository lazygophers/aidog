# Claude Code

Configurez l'intégration AiDog avec Claude Code dans l'onglet **« Réglages » → « Claude Code »**.

## Édition de la configuration

Éditez tous les réglages Claude Code directement dans AiDog :

- **Modèle** — Spécifie le modèle utilisé par Claude Code
- **Permissions** — Contrôles lecture/écriture fichier, accès réseau
- **Sandbox** — Configuration du bac à sable d'exécution de code
- **Plugins** — Gestion des plugins Claude Code
- **Hooks** — Hooks de cycle de vie (PreToolUse / PostToolUse, etc.)
- **StatusLine** — Configuration de l'affichage de la barre de statut

Tous les changements sont enregistrés immédiatement dans `~/.claude.json`.

## Importer depuis Claude Code

Cliquez « Importer depuis Claude Code » pour lire le fichier de configuration existant et le charger dans l'éditeur. Idéal pour migrer des configurations existantes lors de la configuration initiale.

## Exporter vers Claude Code

Cliquez « Exporter vers Claude Code » pour écrire l'adresse proxy dans le fichier de configuration Claude Code :

```json
{
  "apiBaseUrl": "http://127.0.0.1:9876"
}
```

Après export, toutes les requêtes Claude Code seront proxifiées via AiDog.

## Synchroniser les réglages de groupe

Cliquez « Synchroniser les réglages de groupe » pour générer des fichiers de configuration Claude Code indépendants pour chaque groupe :

- Enregistrés dans `~/.aidog/settings.<group-name>.json`
- Chaque fichier utilise le chemin proxy du groupe correspondant
- À utiliser avec le bouton copier « Claude » sur les cartes de groupe

### Utilisation

1. Cliquez le bouton « Claude » sur une carte de groupe pour copier la commande de lancement
2. Collez dans votre terminal :

```bash
claude --brief --dangerously-skip-permissions \
  --settings ~/.aidog/settings.<group-name>.json
```

## Configuration manuelle

Alternativement, configurez via variables d'environnement :

```bash
export ANTHROPIC_BASE_URL="http://127.0.0.1:9876"
export ANTHROPIC_API_KEY="any-value"
```

## Notes

- Claude Code utilise le protocole Anthropic natif
- Assurez-vous que la plateforme cible supporte le protocole Anthropic ou a une conversion configurée
- AiDog fournit un chemin dédié pour Claude Code
- Ajoutez du model mapping dans les réglages plateforme ou groupe si nécessaire

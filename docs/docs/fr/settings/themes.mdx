# Thèmes

AiDog fournit plusieurs thèmes intégrés, chacun avec des modes Clair et Sombre.

## Thèmes intégrés

| Thème | Style |
|-------|-------|
| Liquid Glass | Glassmorphism, translucide |
| Nord | Tons bleus arctiques, calme |
| Dracula | Thème sombre, palette populaire |
| Catppuccin | Couleurs chaudes, reposant pour les yeux |
| Solarized | Palette scientifique classique |

## Changer de thème

Sélectionnez un thème dans le sélecteur de la page **« Réglages »**.

### Mode Clair/Sombre

Chaque thème supporte les modes Clair et Sombre. Suit automatiquement les réglages système ou bascule manuellement.

## Fonctionnalités des thèmes

- **Coins arrondis** — Système de border-radius en couches (sm / md / lg / xl)
- **Verre dépoli** — Panneaux semi-transparents avec arrière-plans floutés
- **Transitions fluides** — Changement de thème avec animations d'easing
- **Contraste élevé** — Assure une bonne lisibilité en modes clair et sombre

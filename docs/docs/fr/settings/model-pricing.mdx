# Tarifs des modèles

Gérez toutes les données tarifaires des modèles dans l'onglet **« Réglages » → « Tarifs des modèles »**. Les données tarifaires servent à estimer les coûts des requêtes.

## Synchronisation des prix

Synchronisation en un clic des derniers prix depuis les sources tarifaires :

- Cliquez « Synchroniser maintenant » pour récupérer les derniers prix
- Résultat de sync : ajoutés, mis à jour, échoués
- Activez « Synchronisation auto » pour récupérer à intervalles configurés

Intervalles auto : 1 heure / 6 heures / 12 heures / 24 heures / 7 jours

## Prix de repli

Pour les modèles absents de la table, un prix de repli s'applique :

- **Repli entrée** — Défaut $3.0 / million de tokens
- **Repli sortie** — Défaut $3.0 / million de tokens

Ajustable dans la section synchronisation.

## Table des prix

La table affiche ces colonnes :

| Colonne | Description |
|--------|-------------|
| Modèle | Nom du modèle |
| Source | LiteLLM (synchronisé) / Manuel |
| Plateforme par défaut | Plateforme par défaut pour ce modèle |
| Entrée $/M | Prix token entrée (par million) |
| Sortie $/M | Prix token sortie |
| Lecture cache $/M | Prix lecture token cache |

Recherche par nom de modèle, filtre par source, taille de page ajustable et saut de page.

## Gérer les tarifs

- Supprimer un enregistrement tarifaire (bouton supprimer en fin de ligne)
- Les changements de prix prennent effet immédiatement

## Estimation du coût

Après chaque requête, AiDog automatiquement :

1. Extrait l'utilisation de tokens (entrée / sortie / cache) de la réponse amont
2. Recherche le prix unitaire dans la table tarifaire
3. Calcule le coût estimé

Ordre de repli tarifaire : Tarif personnalisé → Tarif synchronisé → Prix de repli

:::tip{title="Note"}
Les coûts estimés sont fournis à titre indicatif. La facturation réelle se base sur la facturation de la plateforme.
:::

Voir [Tarifs & Facturation](/fr/stats/pricing).

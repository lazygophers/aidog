# Tray Configuration

AiDog supports system tray persistence, running in the background and displaying key status information.

## System Tray

Closing the main window keeps AiDog running in the system tray, maintaining proxy availability.

### Tray Information

The tray icon can display:
- Proxy service status (running/stopped)
- Account balance
- Today's usage overview

### Tray Menu

Right-click the tray icon to:
- Show/hide main window
- Start/stop proxy
- Quit the application

## Tray Customization

Configure tray settings in **"Settings"** → "Tray":

### Display Content

Choose which information to show in the tray:
- Balance information
- Today's request count
- Today's token count
- Today's estimated cost

### Status Line Format

Configure status line display format with custom separators and display order.

## Auto-Start

Configure AiDog to automatically start with the system (minimized to tray).

# Configuration de la barre des tâches

AiDog s'exécute dans la barre des tâches système, affichant les informations clés dans la barre des menus. Configurez les éléments d'affichage et styles dans **« Réglages » → « Barre des tâches système »**.

## Comportement de la barre

Après fermeture de la fenêtre principale, AiDog continue de tourner dans la barre des tâches pour garder le service proxy disponible. Clic droit sur l'icône de la barre pour afficher/masquer la fenêtre principale, démarrer/arrêter le proxy, ou quitter l'app.

## Aperçu en direct

Le haut de la page de configuration montre un aperçu simulé de la barre des menus. Tous les changements sont reflétés instantanément. L'aperçu montre aussi l'utilisation de ligne courante (max 2 lignes) et avertit si dépassé.

## Types d'éléments d'affichage

Cliquez « Ajouter un élément d'affichage » pour ajouter un de ces types :

### Solde/Quota de plateforme

Sélectionnez une plateforme activée pour afficher son solde ou quota Coding Plan :

| Mode d'affichage | Description |
|--------------|-------------|
| Solde | Solde du compte (ex. `$12.50`) |
| Quota | Pourcentage restant Coding Plan (ex. `85%`) |

### Statistiques du jour

Affiche les données agrégées du jour, avec 4 métriques disponibles :

| Métrique | Description |
|--------|-------------|
| Tokens | Utilisation totale de tokens du jour |
| Cache% | Taux de cache hit du jour |
| Cost$ | Coût estimé du jour |
| Requests | Nombre total de requêtes du jour |

### Séparateurs

Insérez des séparateurs entre éléments. Six styles prédéfinis (`|`, `·`, `—`, `/`, `»`, espace) plus caractères personnalisés.

## Personnalisation des éléments

Dépliez un élément d'affichage pour ajuster ces propriétés :

| Propriété | Description |
|----------|-------------|
| Affichage | Solde / Quota (éléments plateforme uniquement) |
| Métrique | Tokens / Cache% / Cost$ / Requests (statistiques du jour uniquement) |
| Étiquette | Nom d'affichage personnalisé (vide pour défaut) |
| Décimales | Nombre de décimales pour les valeurs |
| Mode ligne | Simple (étiquette + valeur même ligne) / Deux lignes (étiquette haut, valeur bas) |
| Alignement | Gauche / Centre / Droite |
| Alignement ligne valeur | Alignement indépendant pour la seconde ligne en mode deux lignes |
| Couleur | Suivre système / Préréglé (rouge/vert/orange) / Personnalisé |

### Avertissement couleur

En choisissant une couleur personnalisée, AiDog avertit si la couleur peut ne pas être clairement visible sous certains thèmes de barre des menus.

## Glisser pour réordonner

Les éléments d'affichage supportent le réordonnancement par glisser-déposer. Les éléments désactivés n'apparaissent pas dans l'aperçu.

## Démarrage auto au boot

Configurez AiDog pour démarrer avec le système dans **« Réglages » → « Configuration proxy »**.

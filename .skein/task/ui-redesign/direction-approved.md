# 设计方向选定 (huashu-design gate)

## 展示
- 三方向真实视觉预览: `.skein/task/ui-redesign/previews.png`（6 组件 × 3 方向 × light/dark）
- 方向: A 静谧玻璃 / B 柔感微光 / C 锐利扁平

## 用户选择原话
> "a的深色+b的亮色，同时保留a的发光边框、扁平化"

## 落地契约（三个 subagent/执行的唯一共同输入）
- **Light 模式 = 方向 B 柔感微光**：淡紫蓝渐变底 + 双层柔阴影(收敛) + 大圆角(radius-lg ~14) + tinted 近白卡面 + 胶囊工具栏。
- **Dark 模式 = 方向 A 静谧玻璃 (底色改用 C)**：背景用 C 的中性近黑 #0a0a0c（非 A 的 navy）+ 中性深灰卡面 #161619/#1c1c20 + 金 #ffd98a 主色 + 柔和暗阴影 + 独立软圆图标按钮。（用户后续: 黑暗模式背景色使用 C）
- **保留「发光边框」**：`.glass` hover 的蓝金流光 conic 描边(微光流沙签名效) + 亮发丝边缘，两者都留。
- **扁平化 (覆盖修饰)**：主按钮去金属多段渐变(改近纯色/单柔渐变)、去外发光；去 pillow inset 高光过量；阴影整体更轻。
- **金蓝身份不变**：light 蓝主/金辅，dark 金主/蓝辅。

## 覆盖组件（用户抱怨点）
1. 卡片阴影 — 降噪、柔化、随主题色。
2. 主按钮 — 扁平化去 3D/金属/发光。
3. 幽灵按钮 — 统一 tinted。
4. 下拉菜单 — 暗色适配（popover token 双模）。
5. 工具栏图标按钮 — light 胶囊 / dark 独立软圆，统一。
6. 开关 toggle + tooltip — 重做协调。
